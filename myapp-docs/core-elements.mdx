The usage of BrightSign’s products can be broken down into the following core elements:

# Content Management

Managing the content that players will display. This may also include content authoring.

# BSN.Cloud

BrightSign’s platform for player setup and management as well as optional cloud-based content management.

# Player Setup (Provisioning)

Setting up a player for use, an essential part of which is establishing the player’s **Publishing Mode** which defines how content will be published / delivered to the player.

# Player Management

The ongoing managing and monitoring of players.
