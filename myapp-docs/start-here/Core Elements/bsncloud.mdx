BSN.Cloud is BrightSign’s cloud-based platform for player and content management.

# 5-Year Warranty

<Image src="https://archbee-image-uploads.s3.amazonaws.com/0prk6SO2dy9LdU77jB8D1/1tdmnhZVEJXlcnaa92_3C_image.png" size="70" width="750" height="421" position="center" showCaption="false"></Image>

New players purchased from Jan 01, 2025 onwards come with an industry-leading 5-year warranty. Connecting the player to BSN.Cloud (either the free tier or the paid tier as described below) activates the 5-year warranty. Players not connected to BSN.Cloud have a 2-year warranty. More details can be found :Link[here]{href="https://www.brightsign.biz/brightsign-players/warranty/" newTab="true" hasDisabledNofollow="false"}.

# BSN.Cloud Tiers

<VerticalSplit layout="right">
  <VerticalSplitItem>
    BSN.Cloud consists of two tiers:

    - The *free* **bsn.Control** offers useful tools for player setup and management.
    - The subscription-based **bsn.Content** includes everything bsn.Control offers plus content hosting and management.

    bsn.Control can be thought of as a subset of bsn.Content, both of which are powered by the BSN.Cloud platform.
  </VerticalSplitItem>

    ![](https://archbee-image-uploads.s3.amazonaws.com/0prk6SO2dy9LdU77jB8D1/lEZF0MRwuAKs1mD6la1Fd_image.png)
  </VerticalSplitItem>
</VerticalSplit>

## bsn.Control

<VerticalSplit layout="right">
  <VerticalSplitItem>
    The *free* bsn.Control offers useful tools for player setup and ongoing player management, all of which can be accessed using either the BrightAuthor\:connected desktop app or a web browser.

    ### control.bsn.cloud for Partners

    For users who prefer not to use the BrightAuthor\:connected desktop application, bsn.Control’s functionality can also be accessed via the *free* and *unbranded* web interface at control.bsn.cloud. *Partners in particular are highly recommended to take advantage of this option when integrating with BrightSign.*
  </VerticalSplitItem>

  <VerticalSplitItem>


    ![](https://archbee-image-uploads.s3.amazonaws.com/0prk6SO2dy9LdU77jB8D1/a5VVAMHPYrDNNphHWyaZp_image.png)
  </VerticalSplitItem>
</VerticalSplit>

<hint type="info">
  Note that since control.bsn.cloud is a web interface, players must be connected to the internet in order to be accessible via this interface.
</hint>

## bsn.Content

<VerticalSplit layout="right">
  <VerticalSplitItem>
    The subscription-based bsn.Content consists of all the features of bsn.Control plus the hosting of presentations and media assets along with tools to manage that content. Subscribers can use the BrightAuthor\:connected desktop app or a web browser to create, publish, and schedule presentations via our cloud platform.
