A rich set of tools is offered by bsn.Cloud (both the free bsn.Control and paid bsn.Content tiers), for users to manage players on an ongoing basis. These tools can be accessed via BrightAuthor\:connected or a web browser.

Player management functions include:

- Monitoring player status / activity / health.
- Modifying player settings / properties.
- Viewing screenshots of what the player is displaying.
- Troubleshooting (accessing the Diagnostic Web Server, viewing logs, enabling debugging, etc.).
- Installing OS updates.
