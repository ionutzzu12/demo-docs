The Ivanti Service Management Portfolio UI V3 has been modernized from Classic UI V1 to improve usability, navigation, and efficiency. While the core functions remain the same, UI V3 introduces configurable layouts, a streamlined workspace structure, and enhanced filtering and search capabilities. The modern interface reduces clicks, supports customization through the Modern Page Designer, and provides a more intuitive experience for daily operations.

The following table outlines the key differences between the Classic UI V1 and Ivanti Service Management Portfolio UI V3:

| Field                              | Description                                | Values                                                                                         |
| ---------------------------------- | ------------------------------------------ | ---------------------------------------------------------------------------------------------- |
| Row selection                      | Clicking anywhere in a row selected it.    | Each row must be selected using a **checkbox**.                                                |
| Delete action                      | Always visible.                            | Enabled only after selecting rows.                                                             |
| Pin records                        | Only available from toolbar.               | Pin it action available per row (adds to **Watchlist**).                                       |
| Saved searches vs. Filter searches | Managed separately                         | Combined into **Saved filters**,                                                               |
| Advanced filter                    | Opened from a separate search dialog.      | Opens in **Saved Filters** editor.                                                             |
| Saved filter publishing            | Only role based publishing.                | **Role** and **team** based publishing.                                                        |
| Group by and Column chooser        | Located along side the column in the grid. | Accessed from **Settings** icon next to **Actions**.                                           |
| Navigation                         | Not available.                             | **Collapsible** left pane, **More Workspaces** option.                                         |
| Theme toggle                       | Not available.                             | Use the theme icon in the top-right corner to toggle between **Light mode** and **Dark mode**. |
| Multiple view types                | Only Data Grid.                            | Data Grid, Kanban View, and Gallery View.                                                      |
| View switching                     | Not available.                             | Switch between Grid, Kanban, and Gallery from workspace toolbar.                               |
| Kanban board                       | Not supported.                             | Group incidents into lanes, drag-and-drop between statuses.                                    |
| Gallery View                       | Not supported.                             | Mobile-friendly card layout with templates (Single Row, Card Compact, etc.).                   |
| Layout configuration               | Fixed layout; limited customization.       | Fully configurable layouts using Modern Page Designer.                                         |
