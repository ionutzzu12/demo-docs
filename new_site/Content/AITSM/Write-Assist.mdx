The **Write Assist** feature provides AI-powered support for enhancing and generating content. It helps you to prepare quality content with proper clarity, grammar, and context. The feature is applicable to HTML-based text fields and email text fields.

The feature is accessible through **AI Configuration Hub** and includes two sub-features:

- Field-Level Write Assist
- Email Assist

## Enable Write Assist

**To enable the feature, do the following:**

1. Log in to \{\{Primary.Product}} as an \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
2. Go to the Configuration console, scroll to **AI Configuration Hub** and select it.
3. Set **Write Assist** to **Active** using the toggle button.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/Yab5Ui6LnOzkIvys_URdi_write-assist-toggle.png" position="flex-start" size="40" indent="1" initialPath="../Resources/Images/Agentic-AI-Images/write-assist-toggle.png" githubPath="Content/Resources/Images/Agentic-AI-Images/write-assist-toggle.png" showCaption="false"></Image>

4. Select the **Configure** option beside the toggle button.
5. Enable AI Write Assist using the following features:
   - **Field-Level Write Assist**: Enhances content in HTML-based texts fields.

<Paragraph indent="2">
  To enable the feature, refer :Link[Field-Level Write Assist]{label="Field-Level Write Assist" overridedLabel="Field-Level Write Assist" spaceId docId="g1d9P8mpFACKiQ7-0KZXG" version="v2" docAnchorId="#Field-Le" loadingMethod="dynamic" newTab="false" href="Write-Assist.mdx"}.
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  **Email Assist**: Generates AI-assisted draft content in email text fields.
</Paragraph>

<Paragraph indent="2">
  To enable the feature, refer :Link[Email Assist]{label="Email Assist" overridedLabel="Email Assist" spaceId docId="g1d9P8mpFACKiQ7-0KZXG" version="v2" docAnchorId="#Email" loadingMethod="dynamic" newTab="false" href="Write-Assist.mdx"}.
</Paragraph>

6. Click **Save**.

### Field-Level Write Assist

This feature provide assistance to enhance content in HTML-based text fields such as Description, Resolution, and others.

When enabled, it adds the **AI Text Suggestion** button to the HTML field's control properties. Users can click this button to improve the content they have entered in the text field.

1. **Enable Field-Level Write Assist**:

Set **Field-Level Write Assist** to **Active** using the toggle button.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/nKFEsDkP0Hdch3p7f9VO7_write-assist-field-level-toggle.png" position="flex-start" size="40" initialPath="../Resources/Images/Agentic-AI-Images/write-assist-field-level-toggle.png" githubPath="Content/Resources/Images/Agentic-AI-Images/write-assist-field-level-toggle.png" showCaption="false"></Image>

2. **Enhance a content using Field-Level Write Assist**:

<Paragraph indent="1">
  a. From the **Home** page, open any Workspace. For example, Incident Workspace.
</Paragraph>

<Paragraph indent="1">
  b. Double-click to open any specific record from the list.
</Paragraph>

<Paragraph indent="1">
  c. Enter you inputs in an HTML-based fields such as Description, Resolution, and others.
</Paragraph>

<Paragraph indent="1">
  d. Click the right-arrow icon and select the **AI Text Suggestion** option.<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/v8dAA2YB_14_ipvjtwrFl_write-assist-field-level.png" position="flex-start" size="40" initialPath="../Resources/Images/Agentic-AI-Images/write-assist-field-level.png" githubPath="Content/Resources/Images/Agentic-AI-Images/write-assist-field-level.png" showCaption="false"></Image>A new windows appears, displaying an initial text suggestion in a professional format (default).
</Paragraph>

<Paragraph indent="1">
  e. To further enhance the text, click the **Refine** dropdown and select the appropriate options from the list:<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/XhKIDbi1B7D4MNz9n3l0l_write-assist-field-level-refine.png" position="flex-start" size="40" initialPath="../Resources/Images/Agentic-AI-Images/write-assist-field-level-refine.png" githubPath="Content/Resources/Images/Agentic-AI-Images/write-assist-field-level-refine.png" showCaption="false"></Image>* **Change tone to**: Professional, Empathetic, Casual, and Neutral as per your preference.
  * **Summarize**: Brief main points of the current information.
  * **Make shorter**: Concise long text into a shorter version.
  * **Make longer**: Elaborate the content in detailed manner.
</Paragraph>

<Paragraph indent="1">
  f. Click **Accept** to overwrite the field with enhanced text.Alternatively, click **Copy** to copy the text to your clipboard, and then paste it manually into the text box.
</Paragraph>

### Email Assist

This feature is an AI-powered tool designed to help users quickly generate email drafts.

When enabled, it adds the **Email Assist** button to the email form. Users can click this button to generate an email draft automatically.

To enable, configure, and use the feature, do the following:

1. **Enable Email Assist**:

<Paragraph indent="1">
  a. Set **Email Assist** to **Active** using the toggle button.<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/3k8OtY9sirj6ipmyqgdu6_write-assist-email-assist.png" position="flex-start" size="40" initialPath="../Resources/Images/Agentic-AI-Images/write-assist-email-assist.png" githubPath="Content/Resources/Images/Agentic-AI-Images/write-assist-email-assist.png" showCaption="false"></Image>
</Paragraph>

<Paragraph indent="1">
  b. From the first dropdown, select the business object where you want to enable the Email Assist feature.
</Paragraph>

<Paragraph indent="1">
  c. From the **Select fields for AI Write Assist** dropdown, select the fields that will give context to the AI. This helps AI understand the email draft it has to generate.It is a mandatory field. Only the fields configured here will be considered for generating the email draft.
</Paragraph>

<Paragraph indent="1">
  d. Click the Add <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/n7jVffB7coo5bs3AuNfzI_add-icon.png" size="10" isUploading="false" initialPath="../Resources/Images/Add-icon.png" githubPath="Content/Resources/Images/Add-icon.png" width="17" height="16" showCaption="false"></Image> button to configure additional fields if needed.
</Paragraph>

<hint type="info" indent="1">
  Once Email Assist is enabled, ensure that the **Email Assist** button is available in the email form. If it is missing, add it to the respective form. For more information, refer :Link[Enable Email Assist button:]{label="Enable Email Assist button:" overridedLabel="Enable Email Assist button:" spaceId docId="g1d9P8mpFACKiQ7-0KZXG" version="v2" docAnchorId="#Enable_Email_Assist_button" loadingMethod="dynamic" newTab="false" href="Write-Assist.mdx"}.
</hint>

2. **Enable Email Assist button**:

<Paragraph indent="1">
  a. Go to the Configuration console, and open the **Journal.Email** business object.
</Paragraph>

<Paragraph indent="1">
  b. Under **Forms**, open the **Journal.Email** form.
</Paragraph>

<Paragraph indent="1">
  c. Expand the **Other** section, and locate the **Write with AI** button.<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/YIvRGUxanPLiNVELG6ntU_email-assist-button.png" position="flex-start" size="40" initialPath="../Resources/Images/Agentic-AI-Images/email-assist-button.png" githubPath="Content/Resources/Images/Agentic-AI-Images/email-assist-button.png" showCaption="false"></Image>
</Paragraph>

<Paragraph indent="1">
  d. Drag the button and drop it to the desired location on the form.
</Paragraph>

<Paragraph indent="1">
  e. Under **Control Properties**, rename the label name to Email Assist for user clarity.<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/Yu9jy0giGl4JX7weMDn4O_email-assist-label-edit.png" position="flex-start" size="40" initialPath="../Resources/Images/Agentic-AI-Images/email-assist-label-edit.png" githubPath="Content/Resources/Images/Agentic-AI-Images/email-assist-label-edit.png" showCaption="false"></Image>Once changed, the button name changes to **Email Assist**.
</Paragraph>

<Paragraph indent="1">
  f. Click **Save** to save the form.
</Paragraph>

<hint type="info" indent="1">
  If the business object has a journal configured, the system includes it automatically. If not, it skips the journal.
</hint>

<hint type="info" indent="1">
  For upgraded customers, the **Email Assist** button needs to be configured manually as given above.
  This is a one-time configuration and you don't need to repeat it every time.
</hint>

<hint type="info" indent="1">
  For out-of-the-box customers, the **Email Assist** button is available by default.
</hint>

3. **Enhance the content using Email Assist**:

<Paragraph indent="1">
  a. From the **Home** page, open any Workspace. For example, Incident Workspace.
</Paragraph>

<Paragraph indent="1">
  b. Double-click to open any specific record from the list.
</Paragraph>

<Paragraph indent="1">
  c. Under **Journal**, click the **Add** dropdown arrow and select **Email** to create a new email.<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/SGKHadTHmcLzXhn9T6XOj_write-assist-email-assist-compose-email.png" position="flex-start" size="40" initialPath="../Resources/Images/Agentic-AI-Images/write-assist-email-assist-compose-email.png" githubPath="Content/Resources/Images/Agentic-AI-Images/write-assist-email-assist-compose-email.png" showCaption="false"></Image>A new window appears where you can compose the email.
</Paragraph>

<Paragraph indent="1">
  d. Click **Email Assist**.An incident overview is generated based on the record overview, configured fields, and the email conversation history in the Journal.
</Paragraph>

<Paragraph indent="1">
  e. Optionally, you can enter additional instructions in the text box to enhance the email draft. For example: "Generate an email in a polite manner."
</Paragraph>

<Paragraph indent="1">
  f. Click **Generate Draft** to generate the email draft.
</Paragraph>

<Paragraph indent="1">
  g. Click **Copy to clipboard** to copy the draft, and then paste it manually into the text box.<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/Iuy1qLxOL05LkhGlZo7ub_write-assist-email-assist-message.png" position="flex-start" size="40" initialPath="../Resources/Images/Agentic-AI-Images/write-assist-email-assist-message.png" githubPath="Content/Resources/Images/Agentic-AI-Images/write-assist-email-assist-message.png" showCaption="false"></Image>
</Paragraph>
