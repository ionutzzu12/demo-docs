**Role**: \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.

**Platform**: SaaS only.

**Minimum Version**: \{\{Primary.Company}} \{\{Primary.Product}} 2024.2.

**Feature Eligibility**: Ivanti's \{\{FRS\_VARS.AITSM}} capabilities are available to customers who have subscribed to Premium or Enterprise Premium license. To know your eligibility, reach out to your Ivanti Account Manager.

With the Neurons AI feature, you can effortlessly create dashboards and widgets using intuitive text prompt AI. With AI Assist, administrators can now generate widgets and customize dashboards by simply entering a prompt, eliminating the need for manual setup or dragging elements.

This innovative AI widget and dashboard functionality empowers users to create widgets tailored to their needs, enhancing productivity and user experience. Whether you're looking to customize widgets or streamline the widget creation process, Neurons AI makes it seamless with its smart AI based NLP prompt capabilities.

For example, enter "show all incidents grouped by status", a widget is displayed with incidents grouped by status.

<hint type="success">
  When entering prompts, remember it should contain the name of the Business Object and at least one field from the same Business Object. Only then the widget generation is possible. If the prompt does not meet this criteria, widget generation fails and the message "*I'm here to assist with data queries. Please provide a query related to our data.*" is displayed instead.
</hint>

## Enabling Neurons AI

**Prerequisite**

- Enable the Dashboard V2 feature. For more information, refer to [Dashboard V2](../Configure/Dashboards/Dashboard-V2.mdx).

**Steps to enable Neurons AI**

1. Log in to \{\{FRS\_VARS.ProductName}} as an \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
2. Open the Configuration console.
3. Select **AI Configuration Hub** and set the **Neurons AI** option to **Active**.

## Using Neurons AI

1. Log in to \{\{FRS\_VARS.ProductName}}.

<Paragraph indent="1">
  The **Home** page is displayed. If Dashboard V2 is set as the Home page for Admin role, all the configured widgets and dashboards for Admin role is displayed.
</Paragraph>

2. Click the **Neurons AI** button.

<Paragraph indent="1">
  The **Neurons AI** panel is displayed on the right side.
</Paragraph>

3. You can either select an option from the predefined options or enter your prompt in the text box.

<hint type="info" indent="1">
  Ensure the prompt is related only to \{\{FRS\_VARS.ITSM}} data, if not the application displays the response "*I'm here to assist with data queries. Please provide a query related to our data.*". The prompt should contain the name of a Business Object and a field of the same Business Object.
</hint>

  *Predefined prompts*

<Paragraph indent="1">
  The application displays the widget in the most suitable graph, such as a pie chart, bar chart or a tabular list.
</Paragraph>

5. Click on the widget to go to the relevant workspace.

<Paragraph indent="1">
  The widgets are clickable. You will be taken to the workspace with the same query as the widget.
</Paragraph>

6. Click thumbs up or thumbs down at the end of the page and enter your comments. Sharing your feedback helps us improve the capability.

<hint type="info">
  The scope of widget creation using Neurons AI is limited to Business Object (BO). For any BO created in the application, you can generate a widget using Neurons AI as long as you specify the BO name and at least one field name.
</hint>

## Other actions

You can perform the following actions for a generated widget:

- **Change the widget type** - click :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/Q2yjIEvxLup4XzO9Lq3hH_widget-type-icon18x15.png" alt caption size="10" width="18" height="15"} to change the widget type to Pie Chart, Line Chart, Vertical Bar Chart, or Horizontal Bar Chart.
- **Refresh Data** - click the **Refresh** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/Li8JC3fX7zH8wvLIS_jG1_refresh-icon.png" alt caption size="10" width="15" height="16"} to refresh the data.
- **Save the widget as an image** - click the **Export to Image** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/0ULp023D-AMD-4KJoCcox_export-to-image.png" alt caption size="10" width="17" height="16"} to save the widget as an image. It can then be used in presentations and other purposes.
- **Toggle Legend** - click the **Toggle Legend** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/LPAbZruFYKmIp8noixzuW_toggle-legend-icon.png" alt caption size="10" width="15" height="16"}, to highlight the legend.

<Paragraph indent="1">
  *Widget with Toggle Legend selected*
</Paragraph>

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/FVxP_ljmkz39U8KVtXjuu_toggle-legend.png" position="flex-start" size="40" indent="1" initialPath="../Resources/Images/Agentic-AI-Images/Toggle-Legend.png" githubPath="Content/Resources/Images/Agentic-AI-Images/Toggle-Legend.png" showCaption="false"></Image>

- **Maximize** - click the **Maximize** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/TqijutZO1fzuux-UccO0V_maximize-icon.png" alt caption size="10" width="17" height="16"} to open the image in full screen.
- **Add to Dashboard** - click **Add to Dashboard** to add the widget to the current dashboard.

<hint type="info" indent="1">
  The widget is added at the last in the Dashboard. And, once the report is added to the dashboard, the **Add to Dashboard** option is disabled.
</hint>

- **Create New Dashboard** - click **Create New Dashboard** to add the widget to a new dashboard.

## Sample prompts

Here are some sample prompts you can refer to effectively use the Neurons AI feature.

| Use Case                                                                                           | Prompt                                                                                                   | Business Object | Field                     |
| -------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------- | --------------- | ------------------------- |
| To view all logged Change records that are categorized as Major.                                   | Show me all changes where status is 'Logged' and typeofchange is 'Major'.                                | Change          | Logged and Major          |
| To view all Change records grouped by priority assigned in descending order of the priority value. | I want to see all change records grouped by priority, ordered in descending order of the grouping field. | Change          | Priority                  |
| To view top 5 change by category.                                                                  | I want to see the top five change records grouped by category.                                           | Change          | Category                  |
| To view all change records grouped by the month of close date and time, and impact.                | Show me all change records grouped by the month of closeddatetime and impact.                            | Change          | Closeddatetime and Impact |
| To view all incidents of priority below 3.                                                         | Now, show me all changes whose priority is less than 3.                                                  | Incident        | Priority                  |
| To view all incidents created between a certain date.                                              | Now, show me all incidents created between 1st January and 1st June 2011.                                | Incident        | CreatedDateTime           |
