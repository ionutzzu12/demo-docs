<font color="#000000"></font>**Feature Availability**: This feature requires an update to \{\{Primary.Company}}'s database service. Customers entitled to AI SKU can enable this feature once \{\{Primary.Company}} completes the database service upgrade in their respective region.

**Prerequisites**: Reach out to [Ivanti Support](https://www.ivanti.com/support) to enable the feature flag in ConfigDB.

The Expression Assist feature enables AI-based suggestions to generate ISM expressions from natural language. With this feature enabled, users can enter their queries in natural language in the expression editor and convert them into valid expressions by leveraging the inbuilt functions for basic expressions.

## Enable Expression Assist

1. From the \{\{FRS\_VARS.Configuration\_console}}, select **AI Configuration Hub** > **AI & ML**.
2. Turn on the toggle for **Expression Assist** feature.
3. When it is enabled, the **Expression Assist** button appears on every expression editor.

## Use Expression Assist feature

1. Open any Expression editor and enter the query in natural language.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/cFyoFFdEN-zKnrGgkKUeT_expression-assist-natural-language.png" position="flex-start" size="40" indent="1" initialPath="../Resources/Images/Agentic-AI-Images/expression-assist-natural-language.png" githubPath="Content/Resources/Images/Agentic-AI-Images/expression-assist-natural-language.png" showCaption="false"></Image>

2. Click the **Expression Assist** button, and the AI converts the query into a valid expression.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/wLL2fYTIU8NTfsLe2DMUa_expression-assist-convertion.png" position="flex-start" size="40" indent="1" initialPath="../Resources/Images/Agentic-AI-Images/expression-assist-convertion.png" githubPath="Content/Resources/Images/Agentic-AI-Images/expression-assist-convertion.png" showCaption="false"></Image>

3. To update the query again, click the **Edit Query** button.

<hint type="info">

</hint>

- Once the expression is generated, a message appears that explains the AI conversion.
- The Expression Assist feature is available on both the Graphical and Classic user interfaces.

<hint type="danger">
  The system displays an error message if the query is either out-of-context, incomplete, or is an expression. To resolve the issue, rephrase the query using the **Edit Query** button.
</hint>
