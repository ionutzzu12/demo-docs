You can create or modify the alert categories that appear on the **Alert** page. You can either add categories directly into the database, or create a workspace that enables users to add alert categories.

The **Alert Category** workspace is not provided in the out-of-the-box version of \{\{FRS\_VARS.ProductName}}. If you want to use it, you must create it by following the instructions below.

1. Log in to the \{\{FRS\_VARS.Neurons\_ITSM\_-\_Service\_Desk\_Console}} console.
2. Open the **AlertCategory** business object.
3. Create a new layout called **AlertCategory**. This creates the workspace.
4. Open the **Role** workspace, and add the **AlertCategory** workspace that you just created to the top-level tab of the administrator and any other role for which to provide category privileges. This reveals the workspace to the roles you chose.

</Paragraph>

5. Refresh the browser screen, then open the **AlertCategory** (the one that you just created) workspace. The list of current categories appear.

<Paragraph indent="1">
  *AlertCategory Workspace*
</Paragraph>

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/QTB7tmVp84rRCMe-OlcqC_alertcategoryworkspace.png" alt="The AlertCategory workspace" position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/alertcategory_workspace.png" githubPath="Content/Resources/Images/help-images/alertcategory_workspace.png" showCaption="false"></Image>

6. Click **New Alert Category**, then enter a category into the field.
7. Click **Save**.
8. Click **List View** to ensure that your category is added to the list.

<Paragraph indent="1">
  When users create a new alert, they can choose the new alert category.
</Paragraph>
