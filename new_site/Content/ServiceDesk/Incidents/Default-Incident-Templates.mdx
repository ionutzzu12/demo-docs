To view the list of incident templates:

1. Log in to the \{\{FRS\_VARS.Neurons\_ITSM\_-\_Service\_Desk\_Console}} console.
2. Open the **Incident** workspace.
3. Open an incident.
4. From the toolbar, click **Template**.

<Paragraph indent="1">
  The following templates can be used to create incidents:
</Paragraph>

<ExpandableHeading indent="1">
  ## Account Lockout - Domain

  | Field    | Description              |
  | -------- | ------------------------ |
  | Status   | Active                   |
  | Service  | Desktop Service          |
  | Category | Account Lockout          |
  | Urgency  | Medium                   |
  | Impact   | Medium                   |
  | Summary  | Cannot log in to Windows |
