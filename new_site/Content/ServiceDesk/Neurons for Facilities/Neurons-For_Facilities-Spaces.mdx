Facilities spaces can be added to a floor. You can add a space to an existing floor, or create a new floor to which you can add a space.

Refer to [Facilities Location, Building, Floor, Room, and Cubicles](../FacilitiesManagement/Facilities-Location.mdx) for further information on creating locations, buildings, floors and rooms.

**To add a Facilities Space:**

<hint type="info">
  The **Location**, **Building** and **Floor** fields for the new facilities space are mandatory fields. If a location, building and floor are not already created for the space, you can create them from these fields.
</hint>

1. Ensure that you have an existing location, building and floor you can add the space too.
2. Open the **Facilities Space** workspace, or select the **Spaces** tab in the **Location** workspace.
3. Click **New Facilities Space**.
4. Enter the required information into the fields.

<hint type="info">
  Mandatory fields are marked with an asterisk.
</hint>
