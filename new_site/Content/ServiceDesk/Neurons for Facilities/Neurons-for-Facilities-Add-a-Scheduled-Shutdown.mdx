In \{\{FRS\_VARS.Company}} \{\{FRS\_VARS.Neurons\_for\_Facilities}} you can add a scheduled shutdown and apply the shutdown to a location, building, floor, room, cubicle or space.

When you add a scheduled shutdown at location level, the location and all facilities associated to the location are disabled for bookings. For example, if you have building associated to the location, no bookings for the building or it's associated floors, rooms, etc, can be made. Any existing bookings are automatically canceled.

The same also applies when you add a scheduled shutdown at building and floor level, bookings are disabled for all of the building's or floor's associated facilities, and any existing bookings are automatically canceled.

## Add a Scheduled Shutdown

<hint type="info">
  When a scheduled shutdown is applied the **Name** field on the location, building, floor, room, cubicle or space record is changed to a read-only field.
</hint>

1. Log in as either a Facilities Manager or Facilities Administrator role.
2. Open the workspace for the facility you want to apply the shutdown to:
   - Facilities Location
   - Facilities Building
   - Facilities Floor
   - Facilities Room
   - Facilities Cubicle
   - Facilities Space.
3. Click **Schedule Shutdown**.
   The **Schedule Shutdown** dialog opens.
4. Enter the required **Start Date**, **End Date**, and a name and description for the shutdown.
5. Click **OK**.

The scheduled shutdown entry is added under the **Scheduled Shutdown** tab in the relevant facilities workspace.

You can view the shutdown details, edit or cancel the shutdown under the **Scheduled Shutdown** tab.

## Edit a Scheduled Shutdown

1. Log in as either a Facilities Manager or Facilities Administrator role.
2. Open the workspace for the facility the scheduled shutdown applies to.
3. Select the **Scheduled Shutdown** tab.
4. Double-click the scheduled shutdown in the list.
   The **Edit Facility Shutdown** dialog opens.
5. Make the required edit.
6. Click **Save**.

## Cancel a Scheduled Shutdown

1. Logon as either a Facilities Manager or Facilities Administrator role.
2. Open the workspace for the facility the scheduled shutdown applies to.

<hint type="info">
  You can only cancel a shutdown from the facility workspace where the shutdown record originated from.
</hint>

3. Select the **Scheduled Shutdown** tab.
4. Select the scheduled shutdown in the list.
5. Click **Cancel Shutdown**.
