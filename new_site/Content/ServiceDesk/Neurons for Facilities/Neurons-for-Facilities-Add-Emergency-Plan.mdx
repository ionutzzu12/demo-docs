Create an emergency plan and associate it to a building.

## Prerequisites

To create an emergency plan, you must be logged in as one of the following roles:

- Facilities Manager.
- Facilities Administrator.
- Facilities Analyst.

You must have an existing location and building object, or create them, to which you associate the emergency plan.

## To Create an Emergency Plan:

1. Open the **Emergency Plan** workspace.
2. Click **New Emergency Plan**.
3. Enter a brief summary in the **Summary** field.
4. Select a location and building in the **Location** and **Building** drop-down lists.

<hint type="info">
  You can save the emergency plan record at this point, then add a description, review date and assign it later. The **Summary**, **Location** and **Building** fields are mandatory.
</hint>

5. Add a description in the **Description** text box.
6. Enter or select a date for review in the **Review Date** field if required.

<hint type="info">
  The emergency plan must be assigned to an owner, this can be done later if required. You must save the plan if you have made any further edits to it at this point.
</hint>

7. Select **Assigned** from the **Status** drop-down list.
   This step marks the **Owner** field as a mandatory field.
8. Select a team from the **Team** drop-down list.
   The employee you intend to assign as the owner must be a member of the team you select.
9. Select the owner from the **Owner** drop-down list.
10. Click **Save**.
    The emergency plan is created.

### Add Details to the Emergency Plan

You can add further details to the emergency plan by selecting the tabs. By default the following tabs are displayed:

- **Emergency Roles**: Link or unlink employees with an emergency role, or view employees already linked to the plan.
- **Assets**: Link or unlink assets, or view assets already linked to the plan.
- **Checklists**: Create, link or unlink a plan checklist, or view existing checklists.
- **Approvals**: Lists plan approval details. This list is populated when the emergency plan is approved. You also add approvers from this tab.

You can also attach relevant documents or paste further information.

  <Paragraph indent="1">

<ExpandableHeading>
  ## To link an asset to the emergency plan:

  1. Select the **Assets** tab.
  2. Click **Link**.
     The **Assets** search list opens.
  3. Select the required asset from the list.

  <hint type="info">
    If the asset you want to add does not appear in the list, select **All** in the **Favorite Searches** drop-down list. If the asset is still not listed, open the **Facilities Assets** workspace and ensure it is listed there. You may need to create the asset if it is not present in the **Facilities Assets** workspace.
  </hint>

  4. Click **Select**.

  <Paragraph indent="1">
    The selected asset is added under the **Assets** tab in the emergency plan.
  </Paragraph>

  <Paragraph indent="1">
    You can add further assets if required, to do this repeat the above steps.
  </Paragraph>

  <Paragraph indent="1">
    To unlink an asset, select it in the list, and then click **Unlink**.
  </Paragraph>
</ExpandableHeading>

<ExpandableHeading>
  ## To attach or paste files to the emergency plan:

  You can attach any file type to the plan. Copied graphic files, copied text or captured screen shots can be pasted from the clipboard.

  To attach a file:

  1. Click **Attach**.
  2. Navigate to the folder containing the file, and then select it.
  3. Click **Open**.
  4. Click **Save**.

  To paste content from the clipboard:

  1. Click **Paste**.
     The clipboard dialog opens.
  2. Enter a name for the attachment in the **Attachment name** field.
  3. Select the **Clipboard preview** area, then press CTRL + V,
     The clipboard content appears in the preview.
  4. Click **OK**.

  You can also add an **Attachments** tab to the plan. Under this tab you can attach files or add a URL link to a website.

  To view the **Attachments** tab:

  1. In the emergency plan record, click the **Restore Hidden Tabs** button on the right side of the **Tabs** section.
     The list of hidden tabs is displayed.
  2. Select **Attachments**.
</ExpandableHeading>

## Send the Emergency Plan for Approval

Once an emergency plan has been assigned to an owner it can be approved or denied.

Before you send the emergency plan for approval, you should check that you have added all required information, and assigned any emergency roles, assets, checklists, etc.

To send the emergency plan for approval:

1. Select the **Status** field drop-down list, and then select **Pending Approval**.
2. Click **Save.**

After you select **Pending Approval**, the emergency plan fields are locked and cannot be edited further at this point until approved, after which the plan can be further edited if required by changing the **Status** field from **Active**, or **Denied**, to **Review**.

The **Approval Vote Tracking** tab appears, this is where approver(s) can cast their vote to approve or deny the emergency plan. An email is sent to everyone that is assigned as an approver. Approvers can also vote to approve or deny from the email message.

### Approve or Deny the Emergency Plan

To approve or deny and emergency plan:

1. Select the **Approval Vote Tracking** tab.
2. Select **Approve** to approve the plan, or select **Deny** to deny the plan.

If the emergency plan is approved, the **Status** field changes to **Active**. When the status is **Active**, you can not edit the emergency plan. To enable editing, change the **Status** field to **Review**.

If the emergency plan is denied, the **Status** field changes to **Denied**,

If you want to edit and resubmit the emergency plan after it has been denied:

1. Select the **Status** drop-down field, and then select **Review**.
2. Make the required edits to the plan.
3. Select the **Status** drop-down field, and then select **Pending Approval**.
4. Click **Save**.

When you select **Pending Approval**, the plan is resubmitted for approval.

## Deprecate an Emergency Plan

Deprecate an emergency plan that is no longer required for any reason.

<hint type="info">
  A deprecated emergency plan is no longer editable but remains in the emergency plan workspace list, it cannot be resurrected.
</hint>

To deprecate an emergency plan:

1. Select the **Status** drop-down field, and then select **Deprecated**.
2. Click **Save**.
