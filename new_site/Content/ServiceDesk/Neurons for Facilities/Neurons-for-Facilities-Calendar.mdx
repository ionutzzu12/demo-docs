\{\{FRS\_VARS.Company}} \{\{FRS\_VARS.Neurons\_for\_Facilities}} calendars display information on facility preventative maintenance schedules, shutdowns, and reservation information for rooms, spaces and cubicles.

Three calendars are available, these are:

- **Facilities Calendar**: This calendar displays preventative maintenance schedules for a location.
- **Facility Shutdown Calendar**: This calendar displays shutdowns that are scheduled for a location, building, floor, conference room, cubicle or space.
- **Room Reservation Calendar**: This calendar displays reservations made for a room, space or cubicle.

<ExpandableHeading>
  ## Using the Facilities Calendar

  **To view the Facilities Calendar and display preventative maintenance schedules:**

  1. Log in as either a Facilities Manager, Facilities Administrator or Facilities Analyst role.
  2. Select the **Facilities Calendar** workspace.
  3. Select the preventative maintenance search type you want to view from the **Saved Searches** drop-down list.

  <hint type="info">
    If you require a new search, you can create one, refer to Using Saved Search. Click the **Saved Searches** link to open the **Saved Searches** list for **Room Reservations**, and then click **New Search** or the **New Saved Search** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/mkLzCNnMihNrR2UjgHBtP_newsavedsearchicon.png" alt="New Saved Seach Icon" caption size="10" width="18" height="15"}.
  </hint>

  4. Select the calendar view you require from the **View** drop-down list (Day, Week, Month, Only Weekday).
  5. To select the calendar period, click the **Back**:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/uCJH6vba-PHJkU9DbrHOp_calendar-back-arrow.png" alt="back arrow" caption size="10" width="14" height="16"} or **Forward**:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/UhDoYZaCZtPqPzzqZOHgc_calendar-forward-arrow.png" alt="forward arrow" caption size="10" width="14" height="16"} arrows, or select the month/date field (top left) to display the date selector.

  <hint type="info">
    Legend colors assigned to a facility displayed on the calendar can be shown by clicking the **Legend** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/D2KxDWNbOMQWETEh0G_BW_calendar-legend-icon.png" alt="calendar legend-icon" caption size="10" width="15" height="16"}.
  </hint>
</ExpandableHeading>
