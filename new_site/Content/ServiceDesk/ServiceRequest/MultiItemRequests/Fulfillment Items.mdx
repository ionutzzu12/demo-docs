A fulfillment item is the runtime instance of a fulfillment item package. It contains a list of tasks for delivering the fulfillment item and one or more workflows for task execution.

When a Service Catalog user selects a fulfillment item package in a service request form, a fulfillment item is created for that instance of the request. After the request is fulfilled, the fulfillment item status is set to completed.
