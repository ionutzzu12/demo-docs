An end-user request that is not a failure (incident) is a request for service (service request). Service requests are usually initiated on the Self-Service Portal.

You can create service requests for a requester or provide details on an existing request.

Service owners create Service Catalog request items.

## Viewing Service Requests

1. Log in to the \{\{FRS\_VARS.Neurons\_ITSM\_-\_Service\_Desk\_Console}} console.
2. Open the **Service Request** workspace. A list of service requests appears.
3. Double-click a service request record to see details.

  Refer to for more information on searching for an existing service request.
