The Bulk Update option allows you to update multiple records for a set of search results in one go.

<hint type="info">
  Only the first 100 records is updated on performing Bulk Update to minimize system/environment crash in case of large data.
</hint>

1. Log in to \{\{FRS\_VARS.ProductName}} as an Asset Administrator.
2. Open the **Hardware Assets** workspace. Only Asset Administrators can access the **Hardware Assets** workspace.
3. The records are displayed as per the saved search.
4. Click **Bulk Update**.
5. Enter information into the fields:
6. **Field&#x20;**- Select the field that you want to update.
   - **Replace All Values** - Select the check box if you want all values of the selected field to be replaced.
   - **Value&#x20;**- This field is disabled if the **Replace All Value** is selected. Select the value that you want replaced.
   - **Replace with Value** - Enter the value that you want the field to be replaced with.
   - **Find results in selected rows** - Select this check box if you want to find and replaced only for selected records.
   - **Find all results in current saved search** - Select this check box if you want to find and replace records based on the current Saved Search.
7. Click **Replace**. The Bulk Update is done.
