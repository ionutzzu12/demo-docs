The Security Operations team can create and assign tasks to external teams, that is, to teams other than Security. These tasks can also be created from the Security Incidents and Security Requests workspaces.

## Working with External Tasks

1. Open the **External Tasks** workspace.

<Paragraph indent="1">
  A list of all tasks assigned to external teams is displayed.
</Paragraph>

2. Double-click a task to view its details.
3. Click **New External Task** to create a new task.
4. Enter information into the field:
   - **Summary**, **Details**, **Service**, and **Vendor**.
5. Click **Save**.
