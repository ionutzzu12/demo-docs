Report Managers do the following:

- Create report templates.
- Use the report templates to create reports.
- Make those reports available to various other roles.

The default workspaces for the Report Manager are:

- Change
- Configuration Items
- Incident
- Problem
- Release
- Reports
- Report Templates
- Task
