\{\{FRS\_VARS.Neurons\_ITSM\_-\_Discovery\_Manager}}s install the gateway software component for asset tracking purposes and conduct audits of the network software and configuration for compliance tracking purposes.

The default workspaces for the \{\{FRS\_VARS.Neurons\_ITSM\_-\_Discovery\_Manager}} are as follows:

- Change Calendar
- Configuration Item
- Gateway
- Report
- Software Inventory
- and many more

A configuration workspace that consolidates other workspaces logs and tools is also available. Refer to Discovery Management Workspace for information about how to access it.
