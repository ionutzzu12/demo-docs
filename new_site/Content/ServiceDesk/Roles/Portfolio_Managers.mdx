Portfolio Managers do the following:

- Create and manage project portfolios consisting of multiple projects.
- Monitor progress of projects within portfolios including heath, schedules and budgets.
- View associated Issues and risks opened for portfolios.

The default workspaces for the Portfolio Manager are:

- Portfolio
- Alert
- Analytic Metrics
- Announcement
- Change
- FAQ
- Incident
- Issue
- Knowledge
- Problem
- Project
- Project Calendar
- Report
- Risk
- Search
- Service Request
- Social Board
