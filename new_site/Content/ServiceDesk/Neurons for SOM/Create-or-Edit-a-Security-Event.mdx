Manually create a Security Event to track and manage security-specific events.

**To create a Security Event:**

1. Log in as Administrator.
2. Open the **Event** workspace.

<hint type="info" indent="1">
  To locate the **Event** workspace, click **More** in the title bar. If the **Event** workspace does not appear in the **My Workpace** list, click the **Search objects** drop-down and select it from the list. You can also type "Event" into the field to locate it.
</hint>

3. From the **New Record** menu drop-down list, select **New Security Event**.

<Paragraph indent="1">
  A blank form opens.
</Paragraph>

4. Enter the information into the fields as required.

<Paragraph indent="1">
  a. The **Risk Score** field populates after you import RiskSense data.
</Paragraph>

<Paragraph indent="1">
  b. If this is an outage, select the **Is Outage** checkbox.
</Paragraph>

<Paragraph indent="1">
  c. Enter the start date time to begin timing the duration of the outage.
</Paragraph>

<Paragraph indent="2">
  The duration of the outage is calculated when the end date of the event is entered and the entry is saved. It's reported as **Service Available**.
</Paragraph>

5. To link an Incident or Change to the Event, select the **Details** tab.

<Paragraph indent="1">
  a. Select the **Search** icon, and then double-click the Incident or Change. You can also select the Incident or Change, and then select **Use Selected Incident/Change**.
</Paragraph>

<Paragraph indent="1">
  b. You can also link an Incident or Change to the Event if you select the **Incident&#x20;**&#x6F;r **Change&#x20;**&#x74;ab.
</Paragraph>

<Paragraph listStyleType="disc" indent="3">
  Select **Link**, and then double-click the Incident or Change. You can also select the **Incident&#x20;**&#x6F;r **Change**, and then select **Select**.
</Paragraph>

6. Select **Save**.

## Edit a Security Event

**To edit a Security Event:**

1. Open the **Event&#x20;**&#x77;orkspace.
2. Double-click an Event to open the details.
3. Change the information as needed.
4. Select **Save**.
