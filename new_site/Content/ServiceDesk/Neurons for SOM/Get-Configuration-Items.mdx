Configure the script for integration with RiskSense. This links any existing CIs to the event. If there's no existing CI, an Unmanaged Device is created.

**To import CIs:**

1. Open the Integration Tools workspace.

<Paragraph indent="1">
  Refer to [About Configuring Data Integration.](../../Configure/ImportExportData/About-Configuring-Data-Integration.mdx)
</Paragraph>

<Paragraph indent="1">
  a. For the integration type, use **Web Service**. Refer to [Working with Incoming Web Service Connections.](../../Configure/ImportExportData/Using-Web-Services.mdx)
</Paragraph>

<Paragraph indent="1">
  b. Edit the **RiskSense Integration - Get CIs**.
</Paragraph>

<Paragraph indent="1">
  c. On the **Integration Details** page, select **Next**.
</Paragraph>

<Paragraph indent="1">
  d. On the **Integration Script** page, edit the **API Call Settings** to add the RiskSense API URL and the API Key.
</Paragraph>

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/bp7fVL5sUlrThsnSY2F1d_infsemgetciurlscript.png" alt="Settings for the RiskSense API" position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/Neurons-SOM-Images/infsem_get_ci_url_script.png" githubPath="Content/Resources/Images/Neurons-SOM-Images/infsem_get_ci_url_script.png" showCaption="false"></Image>

<Paragraph indent="1">
  e. Select **Next**.
</Paragraph>

<Paragraph indent="1">
  f. In the **Schedule Settings** window, select **Next**.
</Paragraph>

<Paragraph indent="1">
  g. Create a schedule to import these events on a regular basis. Refer to Creating a Schedule in [About the Schedule Entry and Scheduled Jobs Workspaces.](../../Configure/Schedules/Schedule_Entry.mdx)
</Paragraph>

<Paragraph indent="1">
  h. Select **Yes** in the **Confirmation** window.
</Paragraph>

<Paragraph indent="1">
  i. On the **Review and Publish** page, select **Publish**.
</Paragraph>

<Paragraph indent="1">
  j. Select **OK** on the **Data Import confirmation** window.
</Paragraph>
