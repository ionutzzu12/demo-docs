When you follow up on a CI, you can create an Event directly from the CI for quick and easy tracking.

1. Open the CI.
2. Select the **Event** tab.

<Paragraph indent="1">
  a. Select the **New Record Menu** drop-down list, and then select **New Network Event** or **New Security Event**.
</Paragraph>

<Paragraph indent="1">
  b. Enter the information into the fields as required.
</Paragraph>

<Paragraph indent="1">
  c. Select **Save**.
</Paragraph>
