Security Operations dashboards and reports gives you a visual representation of your organization's security threats and vulnerabilities. This helps you take business decisions and initiatives to improve the security performance of your team.

For further information on using dashboards, refer to [Using Dashboards](<../Dashboard/About Dashboards.mdx>).

## List of Security Operations Dashboards

The dashboards available to you are based on your user role. The following dashboards are available for Security Operations:

<ExpandableHeading>
  ## Chief Information Security Officer Dashboards

  - All Active Security Incidents by Priority
  - All Active Security Incidents by Category
  - All Active Security Incidents by Team
  - All Active Security Incidents by Service
  - All Active VIP Security Incidents
  - All Active Breached Security Incidents
  - My Open Items
</ExpandableHeading>

<ExpandableHeading>
  ## Security Administrator Dashboards

  - My Active Security Incidents By Priority
  - My Active Security Incidents
  - My Active Security Incidents By Service
  - My Open Security Incidents
  - My Active Security Requests
  - My Team's Breached Security Incidents
</ExpandableHeading>

<ExpandableHeading>
  ## Security Analyst Dashboards

  - My Active Security Incidents by Priority
  - My Open Security Incidents
  - My Active Security Incidents by Service
  - My Open Security Tasks
  - My Active Security Requests
  - My Team's Breached Security Incidents
</ExpandableHeading>

<ExpandableHeading>
  ## Security Manager Dashboards

  - Active Security Events By Severity
  - Active Events With Outages
  - CIS With Active Security Incidents
  - Active Security Incidents By Status
  - Breached Security Incidents By Team
</ExpandableHeading>

## List of Security Operations Reports

To view security reports, open the **Security Reports** workspace, and then click the report entry you want to open.

The following reports are available for Security Operations:

- Open High Priority Security Incidents
- Open Security Incidents by Category
- Open Security Incidents by Team
- Security Incidents SLAs by Assigned User
- Security Incidents Opened last month by Category
- Security Incidents Opened this month by Category
- Security Incidents with breached SLA
