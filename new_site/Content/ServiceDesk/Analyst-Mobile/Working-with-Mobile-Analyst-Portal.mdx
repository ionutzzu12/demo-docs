The \{\{FRS\_VARS.Mobile\_Analyst\_portal}} is a web-based responsive user interface that you can access using the \{\{FRS\_VARS.Mobile\_Analyst}} role. As a \{\{FRS\_VARS.Mobile\_Analyst}} user, you can perform the following actions using this portal:

- Create, track, resolve, and complete Incidents.
- Create, edit, and fulfill Service Requests.
- Create and complete Tasks and assign Tasks to others.
- Add notes and attachments to Incidents and Tasks.

All of these can be done on-the-go using your mobile devices.

<hint type="info">
  When you login to \{\{Primary.Company}} \{\{FRS\_VARS.ProductName}} on your mobile devices including tablets, only mobile-based user roles are available. If you want to login to non-mobile based user roles, use your desktop or a laptop.
</hint>

## Logging in to the Mobile Analyst portal

1. On your mobile device, enter the login URL provided by your administrator in the Tenant URL field.

<Paragraph indent="1">
  or
</Paragraph>

2. Click the **Scan Tenant QR**, allow access to camera **While using the App**.
3. From the desktop application click the **QR Code** :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/FiZdjg7RmkebwyqkeO9lt_qr-code.png" alt caption size="10" width="16" height="15"} icon and scan the QR Code using the phone camera.

<Paragraph indent="1">
  The login page is displayed.
</Paragraph>

4. In the login page, enter your username and password and click **Login**.
5. Select the \{\{FRS\_VARS.Mobile\_Analyst}} user role and click **Submit**.

<Paragraph indent="1">
  The application displays the **Home** page.
</Paragraph>

The **Home&#x20;**&#x70;age is the default view of the \{\{FRS\_VARS.Mobile\_Analyst\_portal}}. You can navigate to different workspaces from the **Home** page based on the actions you want to perform.

*Home Page*

### Enabling Logout for the Mobile Analyst Role

Starting with Release 2026.2, the Log out option is disabled by default for the Mobile Analyst role to support improved configuration.

To enable log out option for the role, do the following:

1. Log in to \{\{FRS\_VARS.ProductName}} as an Administrator.
2. Open the Configuration console.
3. Select **Configure** > **Users and Permissions** > **Roles and Permissions**.
4. In the list of roles, select \{\{FRS\_VARS.Mobile\_Analyst}}.
5. Under the **Branding options**, select the **Logout Button** checkbox and click **Save**.
