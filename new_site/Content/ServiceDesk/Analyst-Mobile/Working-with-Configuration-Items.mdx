Mobile Analysts can view, edit a Configuration Item (CI), assign a user to a CI, and create Incidents for a CI from their mobile devices.

On the Configuration Item (CI) workspace page, you can filter and sort the CI by expanding the **Filter & Sort** drop-down. You can expand the **Filter By** option to select the suitable options that fit your search criteria, like Status, Priority, Resolve By, Owner, Scanned Period, and so on. Similarly, you can expand the **Sort By** option and select suitable options from the drop-down list. You can also contact your administrator to customize your filters and sorts.

On the Configuration Item (CI) page, you can select the Tile View icon to view all the CIs in a tile view format, where you can get additional information about the user. When you click the image circle under any particular CI, a user information card pops up and displays the username, designation, department, email address, and phone number.

<ExpandableHeading>
  ## Viewing and editing a CI

  1. From the **Home** page, tap the :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/LcJc1_zDXC-M5aZMY7xqg_menuicon.png" alt caption size="10" width="16" height="14"} icon and select CI.

  <Paragraph indent="1">
    The CI page displays all the available CI records.
  </Paragraph>

  2. You can search and sort the CI records.
     - To search - either type a keyword in the Search bar or tap the Saved Search drop-down arrow and select a Saved Search option.
     - To sort - tap the :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/xXCM2B_4qMEv289yb68lE_filtericonmobileanalyst.jpg" alt caption size="10" width="16" height="16"} icon and select the sort option.
  3. Select a record to view its details.
  4. Tap **Edit** to edit the CI's details.

  <Paragraph indent="1">
    The CI record is displayed in edit mode.
  </Paragraph>

  5. Modify the details as required.
  6. You can add journals and attachments to the CI. For information, refer to [Working with Journals and Attachments](Mobile-Analyst-Journals-and-Attachments.mdx).
  7. Tap **Save** once you are finished modifying.
</ExpandableHeading>

<ExpandableHeading>
  ## Assigning a user to a CI

  1. Open the CI record.
  2. Tap **Edit**. The CI is displayed in edit mode.
  3. Tap the **Assign User** field and select the user.

  <Paragraph indent="1">
    The assigned user's initial is displayed in the CI list view.
  </Paragraph>

  4. Tap on the user's initial to view the user profile.
</ExpandableHeading>

<ExpandableHeading>
  ## Creating an Incident for a CI

  1. Open the CI record.
  2. Tap **Edit**. The CI is displayed in edit mode.
  3. Tap **Create New Incident**.

  <Paragraph indent="1">
    A new incident is created for the CI.
  </Paragraph>

  4. To view the Incident, expand the **Incident** section of the CI record.
</ExpandableHeading>

<ExpandableHeading>
  ## Configure CI List View Settings for Access Remote Control in Mobile Analyst UI V3

  **Enable Remote Control Integration**

  1. Log in to **Ivanti Neurons for ITSM** as an administrator.
  2. Go to **Settings**.
  3. Under **Build**, select **Global Constants**.
  4. Create **Enable Remote Control Integration**.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/7lPAuiQROoJw9k07gbO2e_mobile-analyst-global-constant.png" position="flex-start" size="40" indent="2" initialPath="../../Resources/Images/ServiceDesk-Images/Mobile Analyst Global Constant.png" githubPath="Content/Resources/Images/ServiceDesk-Images/Mobile Analyst Global Constant.png" showCaption="false"></Image>

  5. Set the value to **True**.
  6. Select **Save**.

  **Navigate to the CI business object**

  1. Under **Settings** > **Build**, select **Business Objects**.
  2. In the **Search** field, search for **CI Workspace**.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/swk7mUmjiWKJImPQzaXjS_ci-workspace.png" position="flex-start" size="40" indent="2" initialPath="../../Resources/Images/ServiceDesk-Images/CI Workspace.png" githubPath="Content/Resources/Images/ServiceDesk-Images/CI Workspace.png" showCaption="false"></Image>

  3. Select **CI** **(Configuration Item)**.
  4. Select **Layouts**.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/FaCwzCqS54M-CAqQ1hE0L_layout-used-in-responsive-analyst.png" position="flex-start" size="40" indent="2" initialPath="../../Resources/Images/ServiceDesk-Images/Layout used in Responsive Analyst.png" githubPath="Content/Resources/Images/ServiceDesk-Images/Layout used in Responsive Analyst.png" showCaption="false"></Image>

  5. Open **AssetManagerMobile.Asset.Default**
     (This layout is used in Responsive Analyst / Mobile Analyst.)
  6. Under **Views in this layout**, select **GridView**.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/RKP42UgnoU-RTVdPUiYRx_grid-view.png" position="flex-start" size="40" indent="2" initialPath="../../Resources/Images/ServiceDesk-Images/Grid View.png" githubPath="Content/Resources/Images/ServiceDesk-Images/Grid View.png" showCaption="false"></Image>

  7. In the **List View Editor**, locate the **List** field.
  8. Next to **MobileAnalystAsset.Default**, select **Edit**.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/cdHiYAjQY0N_vdeqLYdfZ_edit-the-layout-list-view.png" position="flex-start" size="40" indent="2" initialPath="../../Resources/Images/ServiceDesk-Images/Edit the layout list view.png" githubPath="Content/Resources/Images/ServiceDesk-Images/Edit the layout list view.png" showCaption="false"></Image>

  9. In **List Properties**, locate the **Group By** option.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/cL9W0KxzjKCKQwArENP9Y_list-properties.png" position="flex-start" size="40" indent="2" initialPath="../../Resources/Images/ServiceDesk-Images/List Properties.png" githubPath="Content/Resources/Images/ServiceDesk-Images/List Properties.png" showCaption="false"></Image>

  10. Set **Group By** to **Not Set**.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/BoEqu7qmwZq_GE7yaLRRW_group-by-option.png" position="flex-start" size="40" indent="2" initialPath="../../Resources/Images/ServiceDesk-Images/Group by option.png" githubPath="Content/Resources/Images/ServiceDesk-Images/Group by option.png" showCaption="false"></Image>

  11. Select **Submit** to save the list.
  12. Select **Save** to save the layout.
  13. Open **Mobile Analyst (UI V3)**.
  14. Navigate to **Configuration Items (CI)**.
  15. Open a **CI** record.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/LqUoPeOL4z2s1vVlfoNGx_access-remote-control-button.png" position="flex-start" size="40" indent="2" initialPath="../../Resources/Images/ServiceDesk-Images/Access Remote Control button.png" githubPath="Content/Resources/Images/ServiceDesk-Images/Access Remote Control button.png" showCaption="false"></Image>

  The Access Remote Control button is visible at the top of the CI record and is available for use.
</ExpandableHeading>
