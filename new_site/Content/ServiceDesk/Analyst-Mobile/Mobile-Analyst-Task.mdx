Mobile Analysts can create, edit, view, manage, and complete tasks, including assigning tasks to others from their mobile devices.

Emails, notes and attachments can be added and deleted in a task. For more information, refer to [Working with Journal and Attachments](Mobile-Analyst-Journals-and-Attachments.mdx).

On the Task workspace page, you can filter and sort the tasks by expanding the **Filter & Sort** drop-down. You can expand the **Filter By** option to select the suitable options that fit your search criteria, like Status, Priority, Resolve By, Owner, Scanned Period, and so on. Similarly, you can expand the **Sort By** option and select suitable options from the drop-down list. You can also contact your administrator to customize your filters and sorts.

On the Task workspace page, you can select the Tile View icon to view all the tasks in a tile view format, where you can get additional information about the user. When you click the image circle under any particular Task, a user information card pops up and displays the username, designation, department, email address, and phone number.

<ExpandableHeading>
  ## Creating a Task

  A task can only be created for an incident; therefore, you need to open the incident you want to add a task to.

  1. From the **Home** page, open an incident.
  2. Tap **Edit**, **More** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/bhZC2RDjm9lZUowq_FLnW_moreiconmobileanalyst.jpg" alt caption size="10" width="7" height="16"} and then select **Add New Task**.
  3. Fill out the information in the **Create New task** page and tap **Save**.
</ExpandableHeading>

<ExpandableHeading>
  ## Viewing and Editing a Task

  1. From the **Home** page, do one of the following:
     - Select **My Tasks** to view only your tasks.
     - Select **View Tasks** to view your and your team's tasks.
  2. Select a task to view its details and tap **Edit**.
  3. Make the necessary changes and tap the **Save** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/7ceKpPx916FJRL4jIRI8T_saveiconmobileanalyst.jpg" alt caption size="10" width="15" height="14"}.

  <hint type="info">
    To open the incident associated with the task, tap the **More** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/bhZC2RDjm9lZUowq_FLnW_moreiconmobileanalyst.jpg" alt caption size="10" width="7" height="16"} and select **Open Parent Item**. This option is available only if the parent item of the task is incident.
  </hint>
</ExpandableHeading>

<ExpandableHeading>
  ## Completing a Task

  1. From the **Home** page, open a task and tap **Edit**.
  2. Tap the **More** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/bhZC2RDjm9lZUowq_FLnW_moreiconmobileanalyst.jpg" alt caption size="10" width="7" height="16"} and select **Complete**.
</ExpandableHeading>
