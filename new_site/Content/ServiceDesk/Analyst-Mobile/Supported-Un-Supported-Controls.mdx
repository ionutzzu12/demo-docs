Only selected controls are supported for viewing in the responsive Analyst UI. If the form uses controls that are not supported for responsive UI, the text - "*\<control name> is not supported*" is displayed for the control.

## Incident Controls

Following are the supported and un-supported controls for Incident Form:

<ExpandableHeading>
  ## Supported Controls

  - Check Box
  - Link Field
  - Object Matching Control
  - Read-only Text Field
  - Text Field
  - Text Field (dropdown)
  - Color Picker
  - Related Objects List
  - Response Target
  - Child Object Viewer
  - Text Area
  - HTML Editor
  - Password y
  - Radio Button
  - Address Button
  - CI Tree Viewer
  - UI Command Button
</ExpandableHeading>

<ExpandableHeading>
  ## Unsupported controls

  - Link Field with Condition
  - Email
  - Number
  - Date and Time Picker
  - Currency Control
  - Attachments
  - Cancel Button
  - Expression Text
  - Label
  - Image
  - Color Bar Indicator
  - Schedule
  - Social Share Button
  - Software Installation
  - Tree List
  - URL Button
</ExpandableHeading>

## Service Request Controls

Following are the supported and un-supported controls for Service Request Form:

<ExpandableHeading>
  ## Controls supported for responsive UI

  - Category heading
  - Text field (Alphanumeric type)
  - Date field
  - Text area
  - Drop-down selection
  - Upload attachment
  - Check box
  - Label
  - Image field
  - Pick list (Custom and Validation list)
  - Price control
  - Currency symbol
</ExpandableHeading>

<ExpandableHeading>
  ## Controls not supported for responsive UI

  - Row Aligner
  - Time field
  - Date/Time field
  - List view
  - Combo box
</ExpandableHeading>

## Request Offering Controls

Following are the supported and un-supported controls for Request Offering Form:

<ExpandableHeading>
  ## Supported Controls:

  - Check Box
  - Text field with alphanumeric
  - Text field with number
  - Text field with money
  - Time field
  - DateTime field
  - Date field
  - Text area
  - Category heading
  - Drop-down list
  - Image
  - Upload control
  - Label
</ExpandableHeading>

<ExpandableHeading>
  ## Un-Supported Controls:

  - Text field with phone number
  - Text field with SSN
  - Text field with email
  - Text field with mac address
  - Text field with URL
  - Text field with custom mask
  - List
</ExpandableHeading>
