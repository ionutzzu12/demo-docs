When creating a request offering, you assign it to a category. When users view the Service Catalog, they can browse services offered according to this category. You can create your own request offering categories to fit your organization.

1. Log in to the \{\{FRS\_VARS.Neurons\_ITSM\_-\_Service\_Desk\_Console}} console.
2. Open the **Request Offerings** workspace.
3. Click **New > Request Offering**. The **Define Request Offering** page opens.
4. Next to the **Category** field, click **Edit**. The **Categories** window opens.
5. Choose **Categories** or one of the default categories to create a sub-category, then click **Add**. The **Categories** dialog box is displayed.

6. Modify the name and enter a description.
7. Click **Save**, then click **Close**.

The category now appears on the category list.

## Translating a Request Offering Category

1. Log in to \{\{FRS\_VARS.ProductName}} as a \{\{FRS\_VARS.Neurons\_ITSM\_-\_Service\_Owner}}.
2. Open the **Request Offerings** workspace. The **Define Request Offering** page opens.
3. Next to the **Category** field, click **Edit**. The **Categories** window opens.
4. Select the item to translate.
5. From the **Language** field, choose a language.

<Paragraph indent="1">
  *Translate Request Offering Categories*
</Paragraph>

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/S-3b9DCrK99G1uu8v8Iia_categorytranslate.png" position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/category_translate.png" githubPath="Content/Resources/Images/help-images/category_translate.png" showCaption="false"></Image>

6. Enter a name in the **Translate Field "Name"** field and a description in the **Translate Field "Description"** field in the desired language.
7. Click **Save**, then click **Close**. When you view the form in the selected language, the translated values are displayed.

## Deleting a Request Offering Category

1. Log in to \{\{FRS\_VARS.ProductName}} as a \{\{FRS\_VARS.Neurons\_ITSM\_-\_Service\_Owner}}.
2. Open the **Request Offerings** workspace. The **Define Request Offering** page opens.
3. Next to the **Category** field, click **Edit**. The **Categories** window opens.
4. Select the category to remove, then click **Delete**.
5. Click **Yes** at the confirmation message. The category no longer appears on the list.
