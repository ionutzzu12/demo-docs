You can copy a request offering and use it as the basis for a new one.

1. Log in to \{\{FRS\_VARS.ProductName}} as a \{\{FRS\_VARS.Neurons\_ITSM\_-\_Service\_Owner}}.
2. Open the **Request Offerings** workspace.
3. Click the **Duplicate template** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/cuZ-GO9UT-wRGa0u6ocj8_duplicate-template-button.png" alt caption size="10" width="26" height="26"} next to the request offering to copy. The **Copy Template** window opens.
4. Enter a unique name for the new request offering, form offering, or external offering, then click **Copy**. The **Define Request Offering** page opens.
5. Continue to define the new request, form, or external offering. as described in and [Creating a Form Offering](Creating-a-Form-Offering.mdx).
