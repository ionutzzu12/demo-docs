\{\{FRS\_VARS.Company-short}} \{\{FRS\_VARS.Neurons\_for\_PPM}} provides the ability to duplicate a Project for reuse, creating a copy of associated Tasks, Budgets, Risks, and Project details. Use any existing Project as a template for new Projects.

**Creating a Project Template**

In order to create a Template to use for future Projects, create a Project with all the Tasks, Budgets, and other details you need for new Projects based on this Template. Refer to [Creating a New Project](../PPM/Creating-New-Project.mdx) for detailed help with creating a new Project.

<hint type="info">
  Deliverables are Project-specific and aren't reused, so these do not copy over to your new Project and should not be a part of your Project Template.
</hint>

**Creating a Project from a Template**

Log in as a Portfolio or Project Manager to create a new Project using the Template you created.

1. Open the Project Template you created.
2. Click **More** > **Project Duplication** > **Create New Project From Template**.
3. Enter information for the new Project in the pop-up window:

<Paragraph indent="1">
  a. **Project Name**: Enter the name of your new Project.
</Paragraph>

<Paragraph indent="1">
  b. **Project Owner**: Choose the employee responsible for the Project.
</Paragraph>

<Paragraph indent="1">
  c. **Start Date**: Click the calendar icon to choose a start date for the Project.
</Paragraph>

<Paragraph indent="1">
  d. **End Date**: Click the calendar icon to choose an end date for the Project.
</Paragraph>

<Paragraph indent="1">
  e. Click **OK** to save your inputs and create the new Project.
</Paragraph>

4. From the list of Projects in the **Project** workspace, click the name of your new Project. From here, you can begin:
   - [Working with PPM Deliverables](PPM_Deliverables.mdx)
   - [Working with PPM Resources](PPM_resourcing.mdx)
   - [Working with Tasks](../Tasks/Tasks.mdx)
