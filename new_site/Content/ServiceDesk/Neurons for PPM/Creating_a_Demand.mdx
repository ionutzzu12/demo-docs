PPM Demands begin as ideas. Anyone with \{\{FRS\_VARS.ProductName}} credentials can submit an idea.

**To create a Demand:**

1. Open the **Demand** workspace.
2. Click **New Demand**.
3. Enter overview information:

<hint type="info">
  Mandatory fields are marked with an asterisk.
</hint>

| Field                                                                                                                                                                                                                        | Description                                                                                                                                                         |
| ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Requested By**                                                                                                                                                                                                             | The person requesting the Demand.                                                                                                                                   |
| **Title** :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/xmOk3Y_AvRY-im08UU6p__mandatoryfieldasterisk.png" alt="Mandatory field" caption size="10" width="8" height="11"}       | Brief summary of the Demand.                                                                                                                                        |
| **Description** :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/xmOk3Y_AvRY-im08UU6p__mandatoryfieldasterisk.png" alt="Mandatory field" caption size="10" width="8" height="11"} | Description of the Demand.                                                                                                                                          |
| **Business Reason**                                                                                                                                                                                                          | Provide a Business Reason for the Demand.                                                                                                                           |
| **Status** :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/xmOk3Y_AvRY-im08UU6p__mandatoryfieldasterisk.png" alt="Mandatory field" caption size="10" width="8" height="11"}      | Status is set to **Idea** initially.                                                                                                                                |
| **Requested Start Date**                                                                                                                                                                                                     | Provide a start date.                                                                                                                                               |
| **Requested End Date**                                                                                                                                                                                                       | Provide an end date.                                                                                                                                                |
| **Team**                                                                                                                                                                                                                     | Select a team.                                                                                                                                                      |
| **Owner**                                                                                                                                                                                                                    | Select an owner.                                                                                                                                                    |
| **Demand Score**                                                                                                                                                                                                             | The **Demand Score** is generated once the Demand is in the **Evaluate** phase.                                                                                     |
| **Resource Estimated Effort**                                                                                                                                                                                                | Displays the **Estimated Effort (Hrs.)** you have entered in the **Resource Plan** tab/workspace provided that the Demand workspace is linked to the Resource Plan. |

4. Enter information into the **Details** tab.

| Field                    | Description                                                          |
| ------------------------ | -------------------------------------------------------------------- |
| **T-Shirt Size**         | Rough estimate of the Demand size.                                   |
| **Area**                 | Select a Focus Area.                                                 |
| **Estimated Total Cost** | Enter the Demand's estimated total cost.                             |
| **Estimated Revenue**    | Enter the Demand's estimated revenue.                                |
| **Portfolio**            | Associate the Demand with an existing Portfolio or create a new one. |
| **Type**                 | Select the Type of Demand.                                           |
| **Strategic Objective**  | Select a Strategic Objective or create a new one.                    |

5. If you selected a Portfolio in the **Details** tab, it appears in the **Portfolio** tab, along with the other Projects in the selected Portfolio.
6. When the Demand is ready to be evaluated by a Demand Manager, change the status from **Idea** to **Evaluate**.
