Configure the following features before using \{\{Primary.Company}} \{\{FRS\_VARS.Neurons\_for\_PPM}}:

## PPM Roles

PPM Roles are used in Resourcing Plans to filter resources for Task assignments. Set up PPM Roles in the \{\{FRS\_VARS.Configuration\_console}} using the Project Role object.

## Focus Areas

\{\{FRS\_VARS.Neurons\_for\_PPM}} includes the following Focus Areas:

- Customer Satisfaction
- Financial Growth
- Innovation/Product
- People/Culture
- Processes/Operations

Add additional areas in the \{\{FRS\_VARS.Configuration\_console}} using the PPM Project Area pick list.

## Strategic Objectives

Strategic Objectives can be associated with Portfolios, Programs, and Demands. Set up PPM Roles in the \{\{FRS\_VARS.Configuration\_console}} using the Strategic Objective object. A \{\{FRS\_VARS.Demand\_Manager}}, \{\{FRS\_VARS.Neurons\_ITSM\_-\_Project\_Manager}}, or \{\{FRS\_VARS.Neurons\_ITSM\_-\_Portfolio\_Manager}} can also create a new Strategic Objective while working on Portfolios.

Use the **Stakeholders&#x20;**&#x74;ab in a Strategic Objective record to add employees involved in the creation and management of an organization's Strategic Objectives.

## Executive Approval Committee

Demands with a budget of over $1 million must be approved by a designated executive approval group.

**To create an executive approval group:**

1. In the \{\{FRS\_VARS.Configuration\_console}}, select **Users and Permissions > Groups**.
2. Select the **Demand Executive Approvals** group.
3. Link employees you want to add to the group.
