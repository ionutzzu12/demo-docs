Preventive Maintenance allows you to create maintenance plans that can be used for facilities that require regular maintenance. You can use the default plans available or create new plans.

## To create a new preventive maintenance plan

1. Click **New**.
2. Select the plan type based on the time interval the maintenance is required.
3. Enter information into the fields:
   - **Name**, **Start Date**, **Start Time**, **End Date**, **End Time**.
   - **Recur Every**: The time interval the maintenance should recur.
   - **All Day Event**: Select the checkbox if the maintenance job takes the entire day. If you select this, the Start and End fields are auto-filled.
   - **No End Date**: Select this checkbox if there is no definite end date.
4. Click **Save**.
