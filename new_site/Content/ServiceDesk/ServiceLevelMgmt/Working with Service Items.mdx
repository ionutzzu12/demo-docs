### Viewing Services Associated with a Change

1. Log in to the application using the Administrator role
2. Open the **Change** workspace.
3. Open a record.
4. Click the **Service** tab. The system displays a list of linked services, if any.

### Adding a Service to a Change

1. Log in to the application using the Administrator role
2. Open the **Change** workspace.
3. Open a record.
4. Click the **Service** tab. The system displays a list of linked services, if any.
5. Click **Link**. The system displays a list of all services.
6. Highlight a service and click **Select**. The system displays the service on the list.
7. Click **Save**.

### Editing a Service Associated with a Change

1. Log in to the application using the Administrator role
2. Open the **Change** workspace.
3. Open a record.
4. Click the **Service** tab. The system displays a list of linked services, if any.
5. Open a service. The system displays the **Edit Service** dialog box.
6. Make changes as needed.
7. Click **Save**. The system closes the dialog box.
8. Click **Save**.

### Removing a Service from a Change

Removing a service item from the list does not delete the service from the database.

1. Log in to the application using the Administrator role
2. Open a workspace such as the **Change** workspace.
3. Open a record.
4. Click the **Service** tab. The system displays a list of linked services, if any.
5. Highlight the service to remove.
6. Click **Unlink**. The system no longer displays the service on the list.
7. Click **Save**.
