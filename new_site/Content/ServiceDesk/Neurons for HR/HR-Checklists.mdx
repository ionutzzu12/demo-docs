The HR Manager role has the ability to create checklists for other HR roles to use. For example, onboarding checklist, or benefit elections.

**To create a checklist:**

1. Select **More workspaces** > **HR Checklist** > **New HR Checklist**.
2. Enter up to 10 items into your checklist with or without due dates.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/-ktEs1xxPENsPab3AWcxi_checklist-with-content.png" alt="Enter up to 10 items in your checklist with or without due dates" position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/Nurons-HR-Images/Checklist-with-content.png" githubPath="Content/Resources/Images/Nurons-HR-Images/Checklist-with-content.png" showCaption="false"></Image>

3. Select **Label Only** if you don't want to show the **Done&#x20;**&#x6F;r **Skipped&#x20;**&#x62;uttons when the checklist is being used as part of an HR Case.
4. If there is a due date for your item, enter the number of days in **Due in Days** and select the **Display Due Date** checkbox.

**To add a checklist to an HR Case:**

1. Open an HR Case.
2. Select the **Checklist** tab.
3. Select **New HR Checklist Item**.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/uDAic3ZiR20633Ge54TBI_hr-new-checklist-item-2022-3.png" alt="To add an HR checklist to a case, select New HR Checklist Item." position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/Nurons-HR-Images/HR-New-Checklist-Item-2022-3.png" githubPath="Content/Resources/Images/Nurons-HR-Images/HR-New-Checklist-Item-2022-3.png" showCaption="false"></Image>

4. Select a checklist type from the **Checklist Name** drop-down list.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/pgR3FZLx_Mn8CUzBSOm5__hr-new-checklist-name-2022-3.png" alt="Select a checklist type from the Checklist Name drop-down list." position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/Nurons-HR-Images/HR-New-Checklist-Name-2022-3.png" githubPath="Content/Resources/Images/Nurons-HR-Images/HR-New-Checklist-Name-2022-3.png" showCaption="false"></Image>

5. Select **Save**.

**To use a checklist associated with an HR Case:**

1. Open an HR Case.
2. Select the **Checklist** tab.
3. Select your chosen checklist and then select **Go to**.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/9NrY_b5Tf5DcnrFpchqpV_hr-new-checklist-use-2022-3.png" alt="Select your chosen checklist and then select Go to." position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/Nurons-HR-Images/HR-New-Checklist-Use-2022-3.png" githubPath="Content/Resources/Images/Nurons-HR-Images/HR-New-Checklist-Use-2022-3.png" showCaption="false"></Image>

4. Select **Done&#x20;**&#x6F;r **Skipped&#x20;**&#x62;uttons for the tasks that have been addressed.
5. Scroll across to the far right to add any comments for tasks.
6. Select **Save**.
