You can create an HR Batch Case, link several employees to it, and then create the associated HR Cases for those linked employees. You can also process those records from the Batch Case - for example, you can resolve the HR Batch Case and any active employee HR Cases linked to it.

<ExpandableHeading>
  ## Creating an HR Batch Case

  1. Open the **HR Batch Case&#x20;**&#x77;orkspace.
  2. Select **New**.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/Ykhz3YBAcHUHNOVDdlrFK_hr-new-batchcase-2022-3.png" alt="Select HR Batch Case then New from the tab" position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/Nurons-HR-Images/HR-New-BatchCase-2022-3.png" githubPath="Content/Resources/Images/Nurons-HR-Images/HR-New-BatchCase-2022-3.png" showCaption="false"></Image>

  3. Complete all the fields in the **Overview** section and then select **Save**.
</ExpandableHeading>

<ExpandableHeading>
  ## Linking Employees to an HR Batch Case

  1. Select **Link&#x20;**&#x6F;n the **Employees&#x20;**&#x74;ab.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/qjPOOwnIxXdUBGeYeZxqZ_hr-new-batchcase-link-2022-3.png" alt="Select Link on the Employees tab to link employees to a case" position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/Nurons-HR-Images/HR-New-BatchCase-Link-2022-3.png" githubPath="Content/Resources/Images/Nurons-HR-Images/HR-New-BatchCase-Link-2022-3.png" showCaption="false"></Image>

  2. Search for the employees you want to link.
  3. Select the employees by clicking in each row.

  <hint type="info">
    **Tip:** You can use standar&#x64;**&#x20;Ctrl+Click** and **Shift+Click** to select more than one employee.
  </hint>

  4. Select **Select**.
  5. Select **Save**.
</ExpandableHeading>

<ExpandableHeading>
  ## Using the Create HR Cases Action

  <Paragraph indent="1">
    This action creates HR Cases for each of the employees linked to the HR Batch Case.
  </Paragraph>

  1. Ensure that the Batch Case itself is in status **Active**. This makes the action available for use.

  <hint type="info">
    **Tip:** Depending on the order in which you link employees and change the Batch Case to **Active**, you may need to do a refresh to activate the **Create HR Cases** action button.
  </hint>

  2. Select **Action Menu&#x20;**> **HR Cases** > **Create HR Cases**.
  3. Select **Yes** to continue to create an HR Case for each linked employee. If you select **No**, the action is stopped and you can change the list of linked employees if you need to.
  4. Select the **HR Cases** tab to see the new HR Cases appear.

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/uZd_5VDmoNe3QF0A5jvdr_hr-new-batchcase-hrcases-2022-3.png" alt="Select the HR Cases tab to see the new linked cases appear " position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/Nurons-HR-Images/HR-New-BatchCase-HRCases-2022-3.png" githubPath="Content/Resources/Images/Nurons-HR-Images/HR-New-BatchCase-HRCases-2022-3.png" showCaption="false"></Image>
</ExpandableHeading>

<ExpandableHeading>
  ## Resolving HR Cases in a Batch

  1. Open the HR Batch Case you want to work with.
  2. Select **Action Menu&#x20;**> **HR Cases** > **Resolve HR Batch Case and Active HR Cases**.
  3. Select **Yes** to continue.

  <Paragraph indent="1">
    All **Active&#x20;**&#x72;ecords change to status **Resolved**. The red mark on the HR Cases Status field indicates that a Save is needed.
  </Paragraph>

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/gQH8OpWB6fZPBG8IeLkc5_hr-new-batchcase-resolved-2022-3.png" alt="The Batch case and associated linked HR Cases change to Resolved when the action has been run" position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/Nurons-HR-Images/HR-New-BatchCase-Resolved-2022-3.png" githubPath="Content/Resources/Images/Nurons-HR-Images/HR-New-BatchCase-Resolved-2022-3.png" showCaption="false"></Image>

  4. Select **Save**. The red marks disappear.

  <hint type="info">
    **Note**: An HR Batch Case with linked HR Cases closes automatically after three business days of being in **Resolved** status.
  </hint>
</ExpandableHeading>
