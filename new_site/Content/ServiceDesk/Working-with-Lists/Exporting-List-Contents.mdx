You can export the contents of a list and save the contents in the following file formats:

- Microsoft Excel 2007 and later (.xlsx)
- Microsoft Excel 2003 and earlier (.xls)
- Comma Separated Values (.csv)
- XML (.xml)

The **Export to Excel** quick action performs the export function. To obtain the desired results, you must choose the file format before you export a list.

<hint type="info">
  For any field name that contains a space, the quick action removes the space when you export to XML.
</hint>

## Choosing the Export File Format

1. From the application, click **Action Menu > Edit Actions** to open the **Actions Manager** workspace. The list of business objects that have quick actions appears.
2. Under the **Your Actions** tab, expand the **Form Actions** menu and choose **Export**.

<Paragraph indent="1">
  The application displays the **Export to Microsoft Excel** quick action.
</Paragraph>

3. Click **Edit** to start editing the quick action.
4. From the **Export Format** drop-down menu, select the export file format to use.
5. Click **Save**.

<hint type="info">
  For each column header, you can export a field or a value expression, but not both.
</hint>

## Exporting to an External Connection

By default, the application exports data to your browser. However, if you have an external data connection, for example to a remote host, you can export to that connection.

1. From the application, click **Action Menu > Edit Actions** to open the **Actions Manager** workspace. The list of business objects that have quick actions appears.
2. Under the **Your Actions** tab, expand the **Form Actions** menu and choose **Export**. The application opens the **Export to Microsoft Excel** quick action.
3. Click **Edit** to edit the quick action.
4. In the **Advanced Option** section, from the **Export To** drop-down list, select the remote host connection to use.
5. Click **Save**.

## Exporting the List Contents

Before you export, choose the file format you want.

1. Within the workspace with the records to export, from the Action Menu choose **Form Actions > Export**.
2. In the **Confirm** dialog box, click **Yes** to export all the records or click **No** to export a single selected record.
3. In the **File Download** dialog box, click **Save** to save the exported file and then navigate to a download location. If you click **Open**, the .xlsx, .xls, and .csv files open in Microsoft Excel. The .xml files open in your default XML editor.

<Paragraph indent="1">
  The file name indicates the date and time the file was exported. For example, **Export2018\_29\_8\_14\_05\_37.xml** means that the file was exported at 2:05 PM, plus 37 seconds, on August 29, 2018.
</Paragraph>
