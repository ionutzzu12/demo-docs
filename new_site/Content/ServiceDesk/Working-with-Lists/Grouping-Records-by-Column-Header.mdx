You can group records by their column header and then drill down to view the count of records within the selected group.

1. Log in to the application.
2. Open a workspace. In this example, open the **Incident** workspace.
3. Hover over the column header that you wish to group by. The filter icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/QG3yTJ6W4hdCTFZqENvvI_filter.png" alt caption size="10" width="21" height="15"} is displayed.
4. Click the filter icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/QG3yTJ6W4hdCTFZqENvvI_filter.png" alt caption size="10" width="21" height="15"} beside the column header. In this example, the **Customer** column header.
5. Select **Group By This Field**.

<Paragraph indent="1">
  All the records in the column are grouped by customer and the count indicates the number of records for each state.
</Paragraph>

  *Records grouped by Customer*
</Paragraph>

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/iVAnCz5r-AhhpJDI4bfUo_groupedbycustomer.png" alt="Records grouped by customer" position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/Common/GroupedByCustomer.png" githubPath="Content/Resources/Images/Common/GroupedByCustomer.png" showCaption="false"></Image>

6. Click any of the status rows to view details of each record in that state.
7. To clear the grouping of records, click the filter icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/QG3yTJ6W4hdCTFZqENvvI_filter.png" alt caption size="10" width="21" height="15"} of the column header and clear the **Show in Groups** checkbox.

<hint type="info">
  You cannot group records in columns that have text fields with an unlimited number of characters allowed.
</hint>
