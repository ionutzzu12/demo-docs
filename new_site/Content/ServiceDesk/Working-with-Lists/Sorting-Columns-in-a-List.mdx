You can sort columns and change the order of columns in a list based on your requirements.

## Sorting Columns in a List

You can sort column data by ascending or descending order.

<hint type="info">
  You cannot sort columns that have text fields with an unlimited number of characters allowed.
</hint>

1. Locate the list to sort.
2. Click the column header.

<Paragraph indent="1">
  The column data is sorted and the up/down arrow in the column header is enabled indicating the current sort order (ascending or descending).
</Paragraph>

3. To change the sort order, click the column header again.

<hint type="info">
  By default, the records that display in the list are sorted in ascending order.
</hint>

## Changing Column Order in a List

You can rearrange the column order in the List view. By default, the record ID is always the first column.

1. Locate the column you wish to change the order.
2. Click its column header and without releasing your mouse, drag it to the new location.

<Paragraph indent="1">
  The column appears in the selected order in the List view.
</Paragraph>
