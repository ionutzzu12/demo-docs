In addition to simple sorting (that is, sorting by one column), a multisort feature is available by default in all lists. You can access the multisort feature in the list view by using either of the following:

## Using the Multisort Feature

1. Log in to the application.
2. Open the workspace for which you wish to group records. For example, open the **Incident** workspace.
3. Click **Multisort** to display the **Multisort Column Picker** dialog box.
4. In the **Sort by** field, select a sort choice from the drop-down list.
5. Click the arrow icon beside the **Sort By** field to change sort order (either ascending or descending).
6. Add additional sort choices by clicking the **plus** :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/r1U7p6WF7FkEM2O4Q2Xn9_addicon-blue.png" alt caption size="10" width="12" height="12"} icon.
7. Select the sort option in the additional sort fields.
8. Remove any sort choices by clicking the **delete**:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/vjwqYRKQGOfMRIJjxz0k__redx23x15.png" alt caption size="10" width="23" height="15"}icon.
9. Click **Sort** to complete your action.
10. Click **Close** to close the **Multisort Column Picker**.

## Resetting the Multisort Feature

To reset the multisort feature, do one of the following:

- Click **Reset** in the **Multisort Column Picker** dialog box.
- Click on any column header.
- Click the **Reset List Personalization**:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/VaGowDSt5te53zFr-7k75_resetpersonalizations.png" alt caption size="10" width="16" height="16"} icon in the footer of the list.

<Paragraph indent="1">
  This does the following:
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  Clears the controls (such as drop-down lists and sort order buttons) in the **Multisort Column Picker** dialog box.
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  Resets the sorting in the list behind the dialog box.
</Paragraph>

## Using the Multisort Feature with a Keyboard Shortcut

You can use a keyboard shortcut to invoke the multisort feature by doing the following:

1. Log in to the application.
2. Open the workspace for which you wish to group records.
3. Click the first column header to sort. This sets the first column to be sorted.
4. Press the **CONTROL&#x20;**&#x6B;ey and click the second column header to sort. This sets the second column to be sorted.
5. Set additional columns for sorting by repeatedly pressing the **CONTROL&#x20;**&#x6B;ey and clicking on headers.

<hint type="info">
  You can click the first column header with or without pressing the **CONTROL&#x20;**&#x6B;ey. It always results in a single column sort. However, you must press the **CONTROL&#x20;**&#x6B;ey to perform a second level sort criteria; otherwise, it performs a single column sort again.
</hint>
