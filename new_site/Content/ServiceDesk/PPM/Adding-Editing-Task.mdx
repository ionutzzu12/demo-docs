A task is a unit of work that must be completed for a project phase. A phase can contain multiple tasks. Adding tasks enables you to easily track the progress of each phase throughout a project.

1. From the project page, select the **Phase** tab.
2. Select the phase you want to add a task to, and then click **Go to**.

<Paragraph indent="1">
  The phase record page opens.
</Paragraph>

3. Click the **Tasks** tab.
4. To create a new task, click New Task.
   or
   To edit an existing task, select the task from the given list and click **Go to**.
5. Enter information into the fields.

Mandatory fields are marked with an asterisk.

| Field                             | Description                                                                                                                                                        |
| --------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| **Summary\***                     | Enter a brief summary of the task.                                                                                                                                 |
| **Details**                       | Enter the details of the task. Include all relevant information related to the task.                                                                               |
| **Service**                       | Select the service associated with the task from the dropdown menu.                                                                                                |
| **Team\***                        | Select the team associated with the task from the dropdown menu.                                                                                                   |
| **Owner**                         | Select the owner associated with the task from the dropdown menu.                                                                                                  |
| **Planned Start Date / End Date** | Enter the planned start and end date for the task. Click the calendar icon to select the dates.                                                                    |
| **Start Date / End Date**         | Enter the actual start and end dates for the task. Click the calendar icon to select the dates.                                                                    |
| **Priority**                      | Select the priority for the task from the dropdown menu. Priority ranges from 1 to 5, with 1 being the highest priority.                                           |
| **Planned Effort (Mins.)**        | Enter the planned number of minutes allocated for completing this task.                                                                                            |
| **Actual Effort (Mins.)**         | The actual number of minutes taken to complete this task.                                                                                                          |
| **Percent Complete (%)**          | The completion percentage. Updating this field through the project tracks the progress of both the task and the phase. It also impact the consumed budget figures. |
| **Acceptance Target**             | The amount of time remaining for the task to be accepted. The system automatically updates the graph based on the task details.                                    |
| **Completion Target**             | The amount of time remaining before the task is escalated. The system automatically updates the graph based on the task details.                                   |

6. Click **Save**.
