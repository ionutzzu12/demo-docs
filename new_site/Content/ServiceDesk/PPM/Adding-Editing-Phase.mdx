**Role**: Project Manager or Portfolio Manager

The phases of a project have defined start and end dates. This allows Project Managers to manage whether or not the project is on track.

Follow these instructions to add or edit a phase:

1. Log in to the application as either a Project Manager or Portfolio Manager.
2. Open the **Project** workspace. The system displays a list of all projects.
3. Select the project you want to open and double-click to open it.
4. Select the **Phase** tab and add or edit a phase.
   - To add a new phase, click **New Project Phase** to open a blank form.
   - To edit an existing phase, select the entry and double-click to open it.
5. Enter information into the fields.

Mandatory fields are marked with an asterisk.

| Field                 | Description                                                                                                                                                                                                                                             |
| --------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Title\***           | Title of the phase.                                                                                                                                                                                                                                     |
| **Project**           | Connects the phase to a project. You can enter a partial name and click **Search** or click **Search** to let the system find and list all available projects.<br />This field may already contain the project name, in which case the field is locked. |
| **Phase\***           | The phase number. You must manually enter this number. This allows you to change the phase number in case you need to insert a new phase between existing phases.                                                                                       |
| **Status\***          | Select the phase status from the list. Only phases with an active status are listed on the project calendar.                                                                                                                                            |
| **Schedule Status\*** | Select the schedule status from the list. The status affects the color codes for the portfolio and the project.                                                                                                                                         |
| **Description\***     | Describes what this phase will accomplish.                                                                                                                                                                                                              |
| **Start Date**        | Click the calendar icon and select the start date of the phase.                                                                                                                                                                                         |
| **Target End Date**   | Click the calendar icon and select the end date you targeted for the phase.                                                                                                                                                                             |
| **Actual End Date**   | When the phase is completed, click the calendar icon and select the actual end date of the phase.                                                                                                                                                       |
| **Completion (%)**    | The percentage of completion of the phase. This impacts the **Consumed** field.                                                                                                                                                                         |

6. Click **Save**.

<hint type="info">
  You can add an additional information to the phase by opening it from the **Project Phase** workspace, or select the phase entry in the **Phase** tab, and then click **Go to** to open the record in the phase record page. Adding tasks to a project is done from a phase record page. You can't add tasks directly from the project page.
</hint>

<hint type="info">
  When the **Project Phase** workspace status is set to **Cancelled** or **Closed**, all linked Resource Plans change to **Closed** status.
</hint>
