**Role**: Project Manager

Project Managers can use a project template to help set up new projects. Currently, there is one project template called **Waterfall Model for Software Project**. It contains a project plan based on the waterfall methodology for software development. The project template has project information available in the **Overview** tab and phase information, available in the **Phase** tab. The phase information contains 12 phases that are typically seen within a software development project. You can use these when setting up a new project. You can change the names and add tasks as required.

Follow these steps to use the project template:

1. Log in to the application as a Project Manager.
2. Open the **Project** workspace. The system displays a list of projects.
3. To create a new project using project template based on waterfall methodology, click **New** dropdown arrow > **Waterfall Model for Software Project**.
4. Enter the information into the **Overview** section.

| Field                    | Description                                                                                                                                                                                                                                                                                                             |
| ------------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Project Name\***       | Enter a unique name for the project.                                                                                                                                                                                                                                                                                    |
| **Summary\***            | Enter a summary description for the project.                                                                                                                                                                                                                                                                            |
| **Project Owner\***      | Select the employee who acts as the project owner.                                                                                                                                                                                                                                                                      |
| **Sponsor**              | If there is a sponsor for the project, select the sponsor in this field.                                                                                                                                                                                                                                                |
| **Status**               | Select the status of the project from the dropdown menu: **Initial**, **Active**, **Cancelled** or **On Hold**.&#xA;The default status is **Initial** when the project is created. Once changed, you can't revert to **Initial**.&#xA;If **Cancelled** is selected, the project record is locked and you can't edit it. |
| **Priority**             | Select the project priority: **Low**, **Medium** or **High**.                                                                                                                                                                                                                                                           |
| **Alignment To**         | Indicates how the project aligns with broader organizational elements.<br />Select **Major Project**, **Portfolios** or **Program** from the dropdown menu.                                                                                                                                                             |
| **Estimated Start Date** | The start date of the project. You can edit this date at a later time if required.                                                                                                                                                                                                                                      |
| **Estimated End Date**   | The end date of the project. You can edit this date at a later time if required.                                                                                                                                                                                                                                        |

5. Click **Save**.

Your project is successfully created using project template based on waterfall methodology.
