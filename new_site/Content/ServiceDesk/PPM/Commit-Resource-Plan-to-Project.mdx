You can commit the selected resources to a project using the **Resource Plans** tab available in the Project workspace.

To commit a Resource Plan to a Project, do the following:

1. Open the Project workspace.
2. Select a Project from the list.
3. In the **Resource Plans** tab, select the preferred Resource Plans. Resource Plans can be multi-selected and committed in bulk.
4. Click **Commit Resource Plan** to commit the selected Resource Plans.

Once committed, the status of the resource plan changes to **Committed**, and the Selected Resources associated with it are added to the **Resources** tab.
