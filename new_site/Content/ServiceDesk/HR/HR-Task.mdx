Almost every case or request will have tasks to be completed before resolving the case or fulfilling the request. Tasks can be created and assigned from HR Case and HR Request to any department or user.

## Working Tasks

<ExpandableHeading>
  ## Reassign a task

  If a task is wrongly assigned to you/your team, you can reassign it to a different team.

  1. Select the task and click **Reassign**.
  2. Select the team in the dialog box that pops and click **Ok**.
</ExpandableHeading>

<ExpandableHeading>
  ## Accept, reject, cancel, or delete a task

  You can accept, reject or cancel an accepted task for valid reason.

  - Select the task and click **Accept**, **Reject**, **Cancel**, or **Delete**.

  <Paragraph indent="1">
    You can only cancel tasks that you have already accepted. Canceled tasks are displayed with **Status** as **Cancel**, but deleted tasks remove the record from the **Task** workspace.
  </Paragraph>
</ExpandableHeading>

<ExpandableHeading>
  ## Put a task on hold or waiting

  You can put accepted tasks on hold if you cannot attend to it immediately due to changed priorities.

  1. Select the task and click **Waiting**.
  2. Enter the reason in the pop-up dialog box and click **Ok**.

  <Paragraph indent="1">
    The task's **Status** is updated as **Waiting**.
  </Paragraph>

  3. Click **Continue** when you are ready to work on the task. The **Status** changes to **Accepted**.
</ExpandableHeading>

<ExpandableHeading>
  ## Complete a task

  1. Select the task and click **Complete**.
  2. Enter the number of hours spent on the task in the pop-up dialog box and click **Ok**.

  <Paragraph indent="1">
    You cannot edit a completed task.
  </Paragraph>
</ExpandableHeading>
