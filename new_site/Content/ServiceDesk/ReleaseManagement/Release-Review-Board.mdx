After each milestone is completed, the Release Review Board is notified to do a post-implementation review of the milestone.

By default, the approval workflow is set up only for major types of releases created by templates. See [Working with Milestones](Working-with-Milestones/Working-with-Milestones.mdx).

The Release Review Board is a defined container for approving post-milestone implementation tasks.

1. Sign in to \{\{FRS\_VARS.ProductName}} as a \{\{FRS\_VARS.Neurons\_ITSM\_-\_Release\_Manager}}.
2. Open the **Release Review Board** workspace.

<Paragraph indent="1">
  The default group opens.
</Paragraph>

3. Double-click a record to view details.
