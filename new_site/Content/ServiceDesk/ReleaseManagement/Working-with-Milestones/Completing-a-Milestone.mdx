---
title: Completing a Milestone
docTags: 
createdAt: Tue Jun 30 2026 17:16:48 GMT+0300 (Eastern European Summer Time)
---

You cannot set the status of a milestone to complete, or 100%, until all of the tasks under it have their status complete. If you try to update the status of a milestone to complete or enter 100% in the **% Done** field, the system displays an error message stating that the tasks under it are not all in the complete status.

If you created the milestone from a template, you cannot set the status of the milestone to complete or 100% if the approval is in pending status. The system does not create the associated tasks for the next milestone sequence until the status of the milestone is complete.

To complete a milestone that you created manually (that is, a milestone that was not created from a release template):

1. Sign in to \{\{FRS\_VARS.ProductName}} as a \{\{FRS\_VARS.Neurons\_ITSM\_-\_Release\_Manager}}.
2. Open the **Release** workspace.
3. Open the release record for which you want to create a milestone.
4. Select the **Schedule** tab.
5. Values of the **% Done** field for all of the associated tasks for the milestone should be set to **100%**. To do this:

<Paragraph indent="1">
  a. Select the milestone.
</Paragraph>

<Paragraph indent="1">
  b. Enter **100** in the **% Done** field.
</Paragraph>

<Paragraph indent="1">
  c. In the **Actual End Date** field, enter the date of the milestone completion.
</Paragraph>

<Paragraph indent="1">
  d. Select **Save**.
</Paragraph>
