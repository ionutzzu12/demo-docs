You can edit a milestone from the **Schedule** tab of the release record or by going to the milestone record from the **Milestone** tab of the release record.

1. Sign in to \{\{FRS\_VARS.ProductName}} as a \{\{FRS\_VARS.Neurons\_ITSM\_-\_Release\_Manager}}.
2. Open the **Release** workspace.
3. Open the release record to edit.
4. Select the **Milestones** tab.
5. Select the milestone to edit.
   The **Edit Milestone** dialog box opens.
6. Edit the milestone details.
7. Select **Save**.

## Deleting a Milestone

1. In the **Release** workspace, select the release record to delete.
2. Select **Delete**.
3. On the confirmation dialog box, select **Continue**.
   The milestone is deleted.
