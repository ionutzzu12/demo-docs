### About the Task Catalog

The task catalog lists external tasks used throughout \{\{FRS\_VARS.ProductName}}. You create tasks within an underpinning contract or service level agreement, and the system displays them here automatically.

### Viewing the Task Catalog

1. Log into the Service Desk Console.
2. Open the **Task Catalog** workspace. The system displays a list of task catalogs.

### Creating an External Task Catalog Item

You can only add external tasks to the task catalog.

1. Log in to the Service Desk Console.
2. Open the **Underpinning Contract** workspace.
3. At the bottom, click the **Task Catalog Item** tab.
4. Click **New Task Catalog Item - External Task**.
5. Enter information into the fields.

| Field                | Description                                                                                                                                                                                                                     |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Name                 | A unique name for the external task.                                                                                                                                                                                            |
| Status               | The status of the external task. See [Creating a Task Catalog Status](<Task Catalog Status.mdx>).                                                                                                                               |
| Task Subject         | The title of the external task. This appears in the subject line of subsequent emails.                                                                                                                                          |
| Task Details         | A description of the external task. This appears in the body of subsequent emails                                                                                                                                               |
| Task Team            | The team associated with the external task. The system automatically generates the value in this field according to the parent record.                                                                                          |
| Task Owner           | The owner associated with the external task. Select from the drop-down list or enter a name.                                                                                                                                    |
| Vendor               | The vendor name.                                                                                                                                                                                                                |
| Vendor Contact Name  | The contact name for the vendor. The system automatically generates the value of this field based on the vendor.                                                                                                                |
| Vendor Contact Email | The email address for the contact for the vendor. The system automatically generates the value of this field based on the vendor.                                                                                               |
| Vendor Contact Phone | The phone number for the contact for the vendor. The system automatically generates the value of this field based on the vendor.                                                                                                |
| Job Item             | The job item. The drop-down list is based on the price list item associated with the vendor. See [Working with Price Lists](../RequestOfferings/Working-with-Price-Lists.mdx) and [Using External Tasks](<External Tasks.mdx>). |
| Job Item Cost        | The cost of the job.  The system automatically generates the value of this field based on the job item.                                                                                                                         |

6. Click **Save**. The system displays the external task on the list.
