Create an Evidence request to request and document evidence needed to complete an Audit. You can only create Evidence records through an Audit record.

<hint type="info">
  To create or edit an Evidence request, you must log in to \{\{Primary.Product}} as a **GRC Manager**.
</hint>

**To create an Evidence request:**

1. Open the **Audit** workspace.
2. Open the relevant Audit record.
3. In the **Evidence** tab, click **New Evidence** to open a blank **GRC Evidence** form. You can also create a batch of Evidence records by clicking **New Evidence Records**.
4. Enter the information into the fields as required.

<hint type="info" indent="1">
  Mandatory fields are marked with an asterisk.
</hint>

<Paragraph listStyleType="disc" indent="2">
  **Status** auto-populates as **Evidence Requested**.
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  Use the **Task** tab to create new tasks, and to manage existing Tasks associated to the Evidence request. When a Task is created or edited that is associated with an Evidence record, users are notified of the Task assigned to them, including the associated Evidence record ID.
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  Use the **Journal** and **Attachment** tabs to create/view journal notes and attachments.
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  Use the **Controls** tab to link Controls.
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  Use the **Audits** tab to view a list of Audits the Evidence request is associated with, and to open the Audit records.
</Paragraph>

<hint type="info" indent="1">
  You can restore hidden tabs using the **plus** sign (to the right of the tabs).
</hint>

5. Click **Save**.
6. The Evidence owner provides information on the Evidence record by including attachments or updating the Journal. When the Evidence owner changes the record status to **In Review**, the record is locked and goes back to the Auditor for review.
7. In the **Evidence** tab the Auditor can approve the evidence, request a review, or request a revision. The Auditor can use **Request Review** if the Evidence owner forgets to change the record status to **In Review**. Use **Request Review** to ask the Evidence owner for more information.
8. Click **Save**.

<hint type="info" indent="1">
  Click **Refresh** If changes you made or relevant tabs are not shown in the record after you have saved it.
</hint>

**Notes on Status:**

- When an Evidence record's status is set to **Closed**, the record fields are locked, and the record can no longer be edited. The record cannot be reopened.
- When an associated Audit record's status is set to **Closed**, **Completed** or **Canceled**, the Evidence record's status is set to **Closed**.

## Edit an Evidence request

**To edit an Evidence request:**

1. Double-click the Evidence request to open the details.
2. Edit the information as required.
3. Click **Save**.
