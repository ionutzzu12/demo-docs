A Task within \{\{Primary.Company}} \{\{FRS\_VARS.Neurons\_for\_GRC}} is a unit of work that must be completed while a GRC item is being resolved.

While working on a GRC item you may need to create or work on tasks as part of the business object resolution.

Tasks can be viewed and opened from the **Tasks** workspace. A Task cannot be created from the **Tasks** workspace, only from within a GRC business object record that can be associated with a Task. GRC business objects that can be associated with a Task are:

- Audit
- Controls
- Control Indicators
- Compliance
- Evidence
- Migration Plans
- Risk
- Vendor

For further information on tasks, refer to [Working with Tasks](../Tasks/Tasks.mdx).

**To create a Task:**

<hint type="info">
  To create a new Task, you must log in to \{\{Primary.Product}} as a **GRC Manager**.
</hint>

Refer to [Creating a Task](../Tasks/Creating-a-Task.mdx).

## Edit a Task

<hint type="info">
  Both **GRC Manager** and **GRC Analyst** can edit an existing task.
</hint>

**To edit a Task:**

1. Open the Task:
   - Open the Tasks workspace, select the Task, and then double-click to open the Task record.
   - Select the **Tasks** tab in the associated record, select the Task, and then click **Go to** to open the Task.
2. Edit the information as required.
3. Click **Save**.
