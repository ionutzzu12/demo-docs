Use Controls as mechanisms to ensure adherence to policies and to assist with compliance to laws, regulations, and other authoritative standards.

<hint type="info">
  To create or edit a Control, you must log in to \{\{Primary.Product}} as a **GRC Manager**.
</hint>

**To create a Control:**

1. Open the **Controls** workspace.
2. Click **New GRC Control** to open a blank **Control** form.
3. Enter the information into the fields as required.

<hint type="info" indent="1">
  Mandatory fields are marked with an asterisk.
</hint>

4. Click **Save**.
5. Use the **Policies**, **Citations**, **Audits**, **Exceptions**, **Compliances**, **Mitigation Plans** and **Risks** tabs to link supporting records to the Control.

<Paragraph indent="1">
  Use the Control record tabs to create new records, or edit records that support the Control, such as Attestations and Control Indicators. See Create or Edit a Supporting Record for a Control.
</Paragraph>

6. Click **Save**.

<hint type="info" indent="1">
  Click **Refresh** If changes you made or relevant tabs are not shown in the record after you have saved it.
</hint>

**To edit a Control:**

<hint type="info">
  When a Control record's **State** field is set to **Retired** and saved, the record is locked and cannot be edited. Check that all Control Indicators supporting the Control record require no further actions, and are in **Retired** state prior to retiring the Control record. Control Indicator records are also locked when retired.
</hint>

1. Open the **Controls** workspace.
2. Double-click the Control to open the details.
3. Edit the information as required.
4. Click **Save**.

## Create or Edit a Supporting Record for a Control

You can create various record types that support a Control. These records can be created directly from a Control record, or in some cases from a relevant workspace, and then linked to a Control. Supporting record types are:

- [Attestations](<Create or Edit an Attestation.mdx>)
- [Control Indicators](Create-or-Edit-a-Control-Indicator.mdx)
- [Citations](Create-Citations-and-Link-to-Controls.mdx)
- [Tasks](Create-or-Edit-a-Task.mdx)
- [Compliances](Create-or-Edit-a-GRC-Compliance.mdx)
- [Mitigation Plans](Create-or-Edit-a-Mitigation-Plan.mdx)

**To create a supporting record:**

1. In the **Controls** workspace, select the required Control record and double-click the entry to open it.
2. Select the relevant tab on the **Control** form.

<hint type="info" indent="1">
  Attestations are created by clicking **Attest** on the top row of the Control record.
</hint>

3. Click **New \<supporting record type>** in the selected tab to open a blank record form.
4. Enter the required values/data in the form fields. Fields marked with an asterisk are mandatory.

<Paragraph indent="1">
  Click the links above for information on creating each type of supporting record.
</Paragraph>

5. **Save** the supporting record.

**To edit a supporting record:**

1. In the **Controls** workspace, select the required Control record and double-click the entry to open it.
2. Select the relevant tab on the **Control** form.
3. Double-click the entry in the tab list. The record form opens.
4. Edit the form fields as required.
5. **Save** the supporting record.
