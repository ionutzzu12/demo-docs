Apply Control Indicators to ensure Control records are compliant and demonstrate/ensure they have been reviewed on a regular basis.

<hint type="info">
  To create or edit a Control Indicator, you must log in to \{\{Primary.Product}} as a **GRC Manager** or **GRC Analyst**.
</hint>

Control Indicators are created from within a Control record. You cannot create a Control Indicator from the **Control Indicators** workspace. Further details and supporting information can be added to a Control Indicator record from within the **Control Indicators** workspace.

**To create a Control Indicator:**

1. Open the **Controls** workspace.
2. Open the Control record you want to add a Control Indicator to.
3. Select the **Control Indicators** tab.
4. Click **New Control Indicator**.
   A blank **Control Indicator** record form opens.
5. Enter the required data in the form fields, Fields marked with an asterisk are mandatory:
   - **Title**: Enter a title for the Control Indicator.
   - **Team**: Select the team that the Control Indicator applies to.
   - **Owner**: Select the owner of the Control Indicator.
   - **Indicator Type**: Enter the Control Indicator type, for example Security, Facilities, etc.
   - **Description**: Enter a description of the Control Indicator
   - **Status**: Select the Control Indocator status. When you create a new Control Indicator, this field is set to Draft.
   - **Response Target**: Select or enter the date that the Control Indicator must be responded to.
   - **Last Result**: Select the last result, can be **Passed** or **Failed**.
   - **Last Collection Date**: Read-only field.
6. Click **Save**.

**To add further details and supporting information:**

When a Control Indicator record is opened from within the **Control Indicators** workspace, a series of tabs are displayed where you can add further details, supporting information, and link/unlink Controls.

1. Open the **Control Indicators** workspace.
2. Select the required Control Indicator record, and double-click to open it.
3. Select the required tab and enter details, or link an existing entry to the Control Indicator record.
   - **Details**: Use this tab to define the instructions or method used for collecting data. Enter the details in the **Data Collection Method** and **Desired Result** fields. **Save** the Control Indicator record when done.
   - **Schedule**: Set the desired frequency/schedule for when the Indicator should be re-evaluated, and new data collection tasks triggered. **Save** the Control Indicator record when done.
   - **Results**: Record the status of the Control Indicator and date for collection.
   - **Controls**: Link or unlink a Control record to the Control Indicator.
   - **Tasks**: Create a task to associate with the Control Indicator. Refer to [Create or Edit a Task](Create-or-Edit-a-Task.mdx) for more information.
   - **Journals**: Create a Journal to associate with the Control Indicator. Select the **New Record Menu** drop-down, and then select the type of Journal to create, **New Activity History**, **New Email** or **New Notes**. Refer to [Adding a Journal Entry to a Business Object](<../ActivityHistory/Adding Notes and Comments.mdx>) for more information.
   - **Attachments**: Add an attachment to the Control Indicator. Click **New Attachment** to attach a file, or click **New URL** to add a link.
   - **Audit History**: View audit history for the Control Indicator.

**To edit a Control Indicator from a Control record:**

<hint type="info">
  When a Control Indicator record's **State** field is set to **Retired** and saved, the record is locked and cannot be edited.
</hint>

1. Open the Control record that the Control Indicator is associated with.
2. Click the **Control Indicators** tab.
3. Select the Control Indicator in the list, and double-click the entry.
   The Control Indicator record opens.
4. Edit the fields as required,
5. Click **Save**.
