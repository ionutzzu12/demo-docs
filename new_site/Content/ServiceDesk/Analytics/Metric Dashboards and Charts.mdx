<ExpandableHeading>
  ## Creating a Dashboard

  1. Log into the \{\{FRS\_VARS.Neurons\_ITSM\_-\_Service\_Desk\_Console}} console.
  2. Open the **Analytic Metrics** workspace.
  3. Click **Edit Mode**.

  <Paragraph indent="1">
    The application displays edit options at the top of the window.
  </Paragraph>

  <Paragraph indent="1">
    *Analytic Metrics Edit Mode*
  </Paragraph>

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/LxAKEXod20BbTE0h614P5_analytics-edit-mode.png" position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/analytics edit mode.png" githubPath="Content/Resources/Images/help-images/analytics edit mode.png" showCaption="false"></Image>

  4. From the upper leftcorner, click the **add dashboard** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/JtIxGMfjslfCSitXWBu4o_addgreen.png" alt caption size="10" width="17" height="15"}.

  <Paragraph indent="1">
    The application displays the **Create Dashboard** dialog box.
  </Paragraph>

  5. Enter information into the fields.

  | Field       | Description                                                                                                                                                         |
  | ----------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
  | Title       | Enter a title for the dashboard. The title appears in the tab.                                                                                                      |
  | Icon        | Select an icon from the dropdown list. The icon appears in the tab before the title. If you do not select an icon, the application uses the default dashboard icon. |
  | Snap        | - Defines the height and width of charts on the dashboard. **Both** is a fully automatic setting .                                                                  |
  | Width (px)  | Appears when horizontal snap is disabled.                                                                                                                           |
  | Height (px) | Appears when vertical snap is disabled.                                                                                                                             |

  6. Click **Save**.

  <Paragraph indent="1">
    The application create a new, blank dashboard.
  </Paragraph>

  7. Add charts to the dashboard.
</ExpandableHeading>

<ExpandableHeading>
  ## Adding Charts to a Dashboard

  1. Log into the \{\{FRS\_VARS.Neurons\_ITSM\_-\_Service\_Desk\_Console}}.
  2. Open the **Analytic Metrics** workspace.
  3. Choose the dashboard to which you want to add a chart.
  4. Do one of the following:
     - From the top right side of the window, drag and drop **Drag me to Add Chart** onto the chart.
     - If a chart already exists, position icons appear. Drop the new chart in one of the available positions: top, left, bottom, or right of the existing chart.

  <Paragraph indent="1">
    *Dashboard Position Icons*
  </Paragraph>

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/-UyenqT_n4xmuLWCNSluS_analytic-position-icons.png" position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/analytic position icons.png" githubPath="Content/Resources/Images/help-images/analytic position icons.png" showCaption="false"></Image>

  <Paragraph indent="1">
    The application displays the **Select Metric and Chart Type** dialog box.
  </Paragraph>

  5. Choose a metric template from the list on the left side. The application displays various chart types that are relevant to your data.
  6. Choose a chart type and then click **Advanced Configuration** to further define your chart. See :Link[Chart Types]{label="Chart Types" overridedLabel="Chart Types" spaceId docId="Odz29FwTIQccM-QaEZEo0" version="v2" docAnchorId="#Chart_Types" loadingMethod="dynamic" newTab="false" href="Metric Dashboards and Charts.mdx"}.
  7. Click **Done**.

  <Paragraph indent="1">
    The application generates the chart.
  </Paragraph>

  <Paragraph indent="1">
    *Select Metric and Chart Type Dialog Box*
  </Paragraph>

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/djzwcRLJQRR93tflYxSK0_select-metric-and-chart-type-db.png" position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/select metric and chart type db.png" githubPath="Content/Resources/Images/help-images/select metric and chart type db.png" showCaption="false"></Image>

  8. Click **Edit Mode** again to turn off edit mode.

  <Paragraph indent="1">
    The application automatically saves the dashboard.
  </Paragraph>
</ExpandableHeading>

### Chart Types

The selection of available charts depends on the metric that the chart represents.

<ExpandableHeading>
  ## Area Chart

  *Area Chart for Analytic Metrics*

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/BQk-tRkycus8nDxmGOSJt_chart-area.png" position="flex-start" size="40" initialPath="../../Resources/Images/help-images/chart area.png" githubPath="Content/Resources/Images/help-images/chart area.png" showCaption="false"></Image>

  The area chart shows graphically quantitative data. Based on the line chart, it compares an area with two or more quantities.

  | Option                      | Description                                                                                                                                                                                                                                                                                                                                             |
  | --------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
  | Chart Title and Description |                                                                                                                                                                                                                                                                                                                                                         |
  | Title                       | The title of the chart, which appears above the chart. The application automatically generates this, but you can edit it.                                                                                                                                                                                                                               |
  | Description                 | Optional. A description of the chart. Describes why the chart was created and what data it contains. This information does not appear on the chart.                                                                                                                                                                                                     |
  | Visualization               | **Count**, **Total Cost**, and **Average**. Each tab is configured separately.                                                                                                                                                                                                                                                                          |
  | Visualization and colors    | Double-click the color sample to change the color. Move the slider to set saturation value, 10% to 100%.                                                                                                                                                                                                                                                |
  | Before Value                | Check to enable. Append text before the reported value. Example: $.                                                                                                                                                                                                                                                                                     |
  | After Value                 | Check to enable. Append text after the reported value. Example: %.                                                                                                                                                                                                                                                                                      |
  | Precision                   | Displays 0 to 5 places to the right of the decimal point.                                                                                                                                                                                                                                                                                               |
  | Use 1000 Separator          | Check to enable. Inserts a comma (,) before every 3 places to the left of the decimal point.                                                                                                                                                                                                                                                            |
  | Period Type                 | Choose from the drop-down menu. Displays the reported value as a period of time. Choose a period in days, hours, minutes, or seconds.  <br />**NOTE**: Use this option only when the reported value is a period of time.                                                                                                                                |
  | Format                      | Appears when you choose a period type. Choose the format best suited to your needs.                                                                                                                                                                                                                                                                     |
  | Series Name                 | Includes the series type (such as count or average) in the hover-over information.                                                                                                                                                                                                                                                                      |
  | Source Name                 | Includes the chart title in the hover-over information.                                                                                                                                                                                                                                                                                                 |
  | Label                       | Includes the field name in the hover-over information.                                                                                                                                                                                                                                                                                                  |
  | Options                     |                                                                                                                                                                                                                                                                                                                                                         |
  | Use Logarithmic Y scale     | Specifies to use a logarithmic Y scale.                                                                                                                                                                                                                                                                                                                 |
  | Axes and Labels             |                                                                                                                                                                                                                                                                                                                                                         |
  | Horizontal Axis Title       | The label for the horizontal axis.                                                                                                                                                                                                                                                                                                                      |
  | Vertical Axis Title         | The label for the vertical axis.                                                                                                                                                                                                                                                                                                                        |
  | Labels Font Size            | The font size for the labels. 1 is the smallest, 10 is the largest.                                                                                                                                                                                                                                                                                     |
  | Labels Orientation          | Specifies orientation of the labels:  <br />**Horizontal**: Labels may overlap in some cases.  <br />**Vertical**: Rotated 90 degrees.  <br />**Diagonal**: Rotated 45 degrees.                                                                                                                                                                         |
  | Legend                      |                                                                                                                                                                                                                                                                                                                                                         |
  | Legend Font Size            | The font size for the legend. 1 is the smallest, 10 is the largest.                                                                                                                                                                                                                                                                                     |
  | Legend Presence             | Specifies where the legend appears in relation to the chart:  <br />**None**: Does not display the legend.  <br />**Left**: Displays the legend at the left of the chart.  <br />**Right**: Displays the legend at the right of the chart.  <br />**Top**: Displays the legend above the chart.  <br />**Bottom**: Displays the legend below the chart. |
</ExpandableHeading>

  ## Modifying a Dashboard Name or Icon

  1. Log into the \{\{FRS\_VARS.Neurons\_ITSM\_-\_Service\_Desk\_Console}}.
  2. Open the **Analytic Metrics** workspace.
  3. Click **Edit Mode**. The application displays the edit options at the top of the window.
  4. Click the down arrow next to the name of the dashboard you want to modify and then choose the **edit** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/pUcDCwFqa9aQFPQhBx47U_edit.png" alt caption size="10" width="15" height="16"}.

  </Paragraph>

  <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/OpwvysBlaFgebknxdSjFx_analytics-edit.png" position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/analytics edit.png" githubPath="Content/Resources/Images/help-images/analytics edit.png" showCaption="false"></Image>

  The application displays the **Edit Dashboard** window.

  5. Enter information into the fields.

  | Field       | Description                                                                                    |
  | ----------- | ---------------------------------------------------------------------------------------------- |
  | Title       | The title for the dashboard.                                                                   |
  | Icon        | The icon to display. Choose an icon from the drop-down list.                                   |
  | Snap        | Determines height and width of charts on the dashboard. **Both** is a fully automatic setting. |

<ExpandableHeading>
  ## Modifying a Chart

  You can modify a chart at any time.

  <hint type="info">
    Some charts, such as the **Weekly Average Response and Resolution Times** chart, are designed to show weekly data for the last 3 months based on the current selection. Weekly reports require some recursive query and cannot scale to year. You can modify the report to monthly, then customize the query.
  </hint>

  1. Log into the \{\{FRS\_VARS.Neurons\_ITSM\_-\_Service\_Desk\_Console}}.
  2. Open the **Analytic Metrics** workspace.
  3. Click **Edit Mode**.

  <Paragraph indent="1">
    The application displays edit options at the top of the window
  </Paragraph>

  4. Choose the dashboard for which to modify the chart.
  5. Hover over the top right corner of the chart, then click the **edit** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/pUcDCwFqa9aQFPQhBx47U_edit.png" alt caption size="10" width="15" height="16"}. The application displays the **Configure Chart** dialog box, showing the advanced options.
  6. Modify the chart as desired.
  7. Click **Save**.
</ExpandableHeading>

  ## Deleting a Chart

  Removing a chart from a dashboard deletes it.

  1. Log into the \{\{FRS\_VARS.Neurons\_ITSM\_-\_Service\_Desk\_Console}}.
  2. Open the **Analytic Metrics** workspace.
  3. Click **Edit Mode**.

  <Paragraph indent="1">
    The application displays edit options at the top of the window
  </Paragraph>

  4. Choose the dashboard from which to remove the chart.
  5. Hover over the top right corner of the chart and click :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/USPl9Egu4Y2Blx7HKC3mA_deleteredx.png" alt caption size="10" width="14" height="13"}. The application displays the **Delete Confirmation** window.

  <Paragraph indent="1">
    The application deletes the chart.
  </Paragraph>

  1. Log into the \{\{FRS\_VARS.Neurons\_ITSM\_-\_Service\_Desk\_Console}}.

  3. Click the down arrow next to a dashboard name.
