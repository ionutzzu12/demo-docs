This example shows how to create an analytic metrics chart that uses multiple series and multiple business objects to display trends.

1. Log into the \{\{FRS\_VARS.Neurons\_ITSM\_-\_Service\_Desk\_Console}}.
2. Open the **Analytic Metrics** workspace.
3. Click **Edit Mode**.

<Paragraph indent="1">
  The application displays the **Settings** menu.
</Paragraph>

4. Go to **Settings > Manage Metrics**.

<Paragraph indent="1">
  The application displays the **Manage Metrics** window.
</Paragraph>

5. On the **General** tab, enter information into the fields:
   - **Category**: Select a category from the list, or enter a new category name. For example, type **Business Value Metrics**.
   - **Name**:Enter a name for the metric. For example, type **Target Cost Saving**.
   - **Description**: Enter a description of the metric.
6. On the **Data Source** tab, enter information into the fields:
   - **Name**: Enter a name for the data set. This name appears in the **Data Source** tab.
   - **Business Object**: Select the business object from the list.
   - **Saved Search**: Select the saved search from the list.
   - **Group By**: Select how to group the data. The application displays only the first 500 items in the list.
   - **Format**:Select the grouping format, such as hour, day, month, or year.
   - **2nd Level Group By**: \[Optional] Select an additional grouping level, if needed. The application displays only the first 500 items in the list.
   - **Split in Series By**: \[Optional] Select a field to split the data into multiple series.
   - **Statistic Fields**:Select labels and fields to use as statistics. Add more fields if needed.
   - **Order by**: Select the sort order. Choose Do not sort to reset the sort order.
   - **Return up to**: Enter the maximum number of items to return.
   - **Hide Null or 0 values**: Select this option to hide records without data.

<hint type="info" indent="1">
  When you select or clear **Hide Null or 0 values** verify that the correct Drill Down values are configured for the associated Data Source. See [About Analytic Metrics Dashboards and Charts](<Metric Dashboards and Charts.mdx>) for more information.
</hint>

7. Click **Add Data Source** to create a new **Data Source** tab.
8. Click **Run** to test the output.
9. Click **Save** to save your parameters.
10. In the **Manage Metrics** window, click **Close**.
11. From the toolbar, select **Drag me to Add Chart** and drop it over the dashboard.

<Paragraph indent="1">
  The application displays the **Select Metric and Chart Type** window.
</Paragraph>

12. In the left pane, select **Target Cost Saving** (the metric you created).
13. In the right pane, select the column chart, click **Done**.
