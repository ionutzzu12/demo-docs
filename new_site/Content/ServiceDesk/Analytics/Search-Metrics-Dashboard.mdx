The Search Metrics dashboard allows you to track searched keywords and their search volumes in the \{\{FRS\_VARS.Self\_Service\_Mobile\_portal}} portal. It is a default dashboard within Analytic Metrics, providing insights such as the total number of keywords searched in the past six months, individual keyword search volumes, and top trending keywords.

The dashboard refreshes automatically every hour. You can disable metric collection by setting the **SearchMetrics** global constant.

You can also configure the retention period for search records. For more information, see [Archive for Search Workflow Block](../../Configure/Workflows/WorkflowObjects/Workflow_Object_Blocks.mdx).

The following dashboards are available for Search Metrics:

| Dashboard Parts Name                    | Description                                                                                                           |
| --------------------------------------- | --------------------------------------------------------------------------------------------------------------------- |
| Search Feature Usage by Users on Device | Displays a pie chart of total keyword searches performed on various devices, including desktops, phones, and tablets. |
| Total Keywords Searched                 | Shows a gauge chart of the total number of keywords searched in the past six months.                                  |
| Keywords with No Result                 | Lists all keywords that returned no results, along with their search volumes.                                         |
| Top Trending Keywords with Results      | Lists the most frequently searched keywords that returned results, along with their search volumes.                   |
