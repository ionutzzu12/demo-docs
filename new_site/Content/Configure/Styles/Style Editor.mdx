Use the Style Editor to customize your workspaces using background color, fields, menus, buttons, and text, as well as the logo of your choice. You can preview the changes you make in the Style Editor instantly on right pane. This feature enables you to experiment with different color combinations, logos, and images for your \{\{FRS\_VARS.ProductName}} and **Self Service Mobile** user interfaces, and the login page.

## Accessing the Style Editor

To access the Style Editor, do the following:

1. From the Configuration console, click **Build > User Interface Styles > Style Editor**. The application displays th&#x65;**&#x20;User interface Styles&#x20;**&#x70;age. In the **User interface Styles&#x20;**&#x70;age, under **All Roles** , click **Edit**.
2. Click the tab corresponding to the item to customize:
   - **Application**: Configures the \{\{FRS\_VARS.ProductName}} user interface (applicable to all workspaces other than the Self Service Mobile).
   - **Self Service**: Configures the **Self Service Mobile** user interface.
   - **Login Page**: Configures the \{\{FRS\_VARS.ProductName}} login page.
