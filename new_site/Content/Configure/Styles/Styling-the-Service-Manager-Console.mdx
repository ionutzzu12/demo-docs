You can configure and customize the \{\{FRS\_VARS.ProductName}} console using the **Application** tab in the **Style Editor**. The settings that you configure on the **Application** tab affect all roles in \{\{FRS\_VARS.ProductName}} except Self Service Mobile. See [Configuring the Self Service Mobile User Interface](Styling-the-Self-Service-Mobile-Portal.mdx).

## Configuring the Styles for a Role

The interface for a specific user role can be configured. Changes made to the default role affect all roles, except for any roles that were configured individually. For more information on roles, see [Working with Roles](../../ServiceDesk/Roles/roles.mdx).

To configure the style for a role:

1. In the \{\{FRS\_VARS.Configuration\_console}}, click **Build** > **User Interface Styles** > **Style Editor**. The application displays the Style Editor.
2. From the **Role** drop-down menu, choose a role.

<hint type="info" indent="1">
  The **Remove Customization** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/8MTnlIsqNFdggXIj_Vr-B_removecustomizationicon23x23.png" alt caption size="10" width="23" height="23"} disappears from the page on selecting a role. However, after saving the customizations, the icon reappears.
</hint>

3. Configure the color specifications for the selected role.
4. Click **Save**. The drop-down menu lists this role as customized.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/lU4SAAxx68x-uaVkX2Ixm_styleeditoradministratorcustomizedinlist.png" position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/style_editor_administrator(customized)_in_list.png" githubPath="Content/Resources/Images/help-images/style_editor_administrator(customized)_in_list.png" showCaption="false"></Image>

<hint type="info" indent="1">
  The **Save** button is only enabled when a value is entered for any of the parameters.
</hint>

To restore a role to its default style, do the following:

1. In the \{\{FRS\_VARS.Configuration\_console}}, click **Build** > **User Interface Styles** > **Style Editor**. The application displays the Style Editor.

<Paragraph indent="1">
  The **Application** tab is selected by default.
</Paragraph>

2. From the **Role** drop-down menu, select a customized role.

<hint type="info" indent="1">
  A customized role is identifiable from the rest of the roles in the list. For example, **Administrator (customized)**.
</hint>

3. Click the **Remove Customization** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/8MTnlIsqNFdggXIj_Vr-B_removecustomizationicon23x23.png" alt caption size="10" width="23" height="23"} next to the **Role** drop-down list. The colors for this role are restored back to its default settings. So in our example, the role is now listed as **Administrator**. See also [Working with Roles](../../ServiceDesk/Roles/roles.mdx).

## Adding a New Logo to the Header or Footer

In \{\{FRS\_VARS.ProductName}}, logos are scaled to fit a 50-pixel vertical space in the header. Because scaling reduces image quality, the preferred logo dimensions are 50 pixels high and up to 200 pixels wide. The maximum logo dimensions are 650 pixels high by 650 pixels wide.

<hint type="info">
  You cannot upload a new logo if you are using Microsoft Internet Explorer Release 9 or earlier.
</hint>

To add a new logo to the header or footer in \{\{FRS\_VARS.ProductName}}, do the following:

1. In the \{\{FRS\_VARS.Configuration\_console}}, click **Build** > **User Interface Styles** > **Style Editor**. The application displays the Style Editor.

<Paragraph indent="1">
  The **Application** tab is selected by default.
</Paragraph>

2. Under **Header & Footer** > **Logo**, click the image.

<Paragraph indent="1">
  The image is enlarged.
</Paragraph>

<hint type="info" indent="1">
  Users can upload only JPEG or PNG image formats. No other formats are accepted by the application.
</hint>

3. Click the **select file** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/46JidEisQOn8omfny09Cu_logofoldericon.png" alt caption size="10" width="16" height="16"} to upload the image.

<hint type="info" indent="1">
  The image size must be less than or equal to 256 KB.
</hint>

4. Navigate to the file and double-click the file name. In the right pane, the new logo replaces the existing logo on the top-left side of the header.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/8AruyJBeMkZUcRESNoG5K_updating-logo-heat-application.png" position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/updating-logo-heat-application.png" githubPath="Content/Resources/Images/help-images/updating-logo-heat-application.png" showCaption="false"></Image>

5. In the **Select Theme** field, the application marks the unsaved value with a red triangle :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/EHiyYfMVQWob8kZOS2-vm_styleeditorredtriangle.png" alt caption size="10" width="7" height="14"} at the left and the **revert** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/uzqDpiJCmf14BiE72Ypet_stylereverticon.png" alt caption size="10" width="17" height="16"} at the right.
6. Click **Save**.

## Customizing a Tenant

You can customize the user interface of tenants and vanity URL tenants. For information on configuring vanity URLs and Managed Service Providers, see [Vanity URL](<../Vanity URL/Configuring-Vanity-URL.mdx>).

To customize a specific tenant, do the following:

1. From the \{\{FRS\_VARS.Configuration\_console}} , click **Build** > **User Interface Styles** > **Style Editor**.
2. From the **Adjust Style for** and **Role** options, choose the tenant and role to customize the application.

<Paragraph indent="1">
  *Choosing an MSP Tenant to Customize*
</Paragraph>

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/smUUzDPtqw95zCRrRmgJf_msptenantstyleeditor.png" position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/MSPTenant_StyleEditor.png" githubPath="Content/Resources/Images/help-images/MSPTenant_StyleEditor.png" showCaption="false"></Image>

3. Make appropriate changes from the tabs - **Application**, \{\{FRS\_VARS.Self\_Service\_Portal-Self-Service}} and **Login Page**.
4. Click **Save**.
5. Log in to the \{\{FRS\_VARS.ProductName}} tenant with to view the application's new user interface as designed.

## Restoring a tenant and role to the default configuration

To restore the customization made to a tenant and role, do the following:

1. From the \{\{FRS\_VARS.Configuration\_console}} , click **Build** > **User Interface Styles** > **Style Editor**.
2. From the **Adjust Style for** and **Role** options, choose the tenant and role to restore to default configuration.
3. Click the **Remove customization** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/8MTnlIsqNFdggXIj_Vr-B_removecustomizationicon23x23.png" alt caption size="10" width="23" height="23"} at the top of the page.

## Restoring the Default Logo

To restore the default logo in the header or footer in \{\{FRS\_VARS.ProductName}}:

1. From the \{\{FRS\_VARS.Configuration\_console}}, click **Build** > **User Interface Styles** > **Style Editor**.
2. Under **Header & Footer**, click the current logo.
3. Click the **delete uploaded file** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/e-zlSKp1AvX6F9KZjn0t5_logodeleteicon.png" alt caption size="10" width="16" height="16"}. The application deletes the current logo and displays the default \{\{FRS\_VARS.ProductName}} logo in its place.
4. In the **Logo** field, the application marks the unsaved value with a red triangle :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/EHiyYfMVQWob8kZOS2-vm_styleeditorredtriangle.png" alt caption size="10" width="7" height="14"} at the left and the **revert** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/uzqDpiJCmf14BiE72Ypet_stylereverticon.png" alt caption size="10" width="17" height="16"} at the right.
5. Click **Save**.
