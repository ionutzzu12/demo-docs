Use the **Login Page** tab to configure the \{\{FRS\_VARS.ProductName}} login page. The same login page is displayed for all roles and user interfaces.

**For Cloud:&#x20;**&#x54;he customizations to the login page are saved in the tenant that you are configuring. Use the URL that is configured for the Tenant record in the ConfigDB in order to access the login page

**For On-Premise:&#x20;**&#x54;he application will select which customizations to displayed based on the address used in the browser. For the customizations to be correctly displayed, you must use the URL that is configured in the ConfigDB in order to access the customized login page

To restore the login page to its default configuration, click the **remove customization** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/8MTnlIsqNFdggXIj_Vr-B_removecustomizationicon23x23.png" alt caption size="10" width="23" height="23"} at the top of the Style Editor.

## Adding a Background Image to the Login Page

When you fill a background using an imported image, \{\{FRS\_VARS.ProductName}} anchors the image at the upper left corner and either repeats the image to fill the background or crops the image to fit the background.

When using a small image, such as a logo, you expect the background to be repeated, so the login page should look OK regardless of its background size.

When using a large image, such as a landscape, you want the image to fill the background of the login page without repeats or excessive cropping.

In the table below, we show the background image size of the \{\{FRS\_VARS.ProductName}} login pages in a maximized browser window on two common monitor resolutions, without any cropping or repeating of the image.

| Layout | 1680 x 1050 Monitor | 1920 x 1080 Monitor |
| ------ | ------------------- | ------------------- |
| Legacy | 1680 x 871 pixels   | 1920 x 901 pixels   |
| Others | 1680 x 991 pixels   | 1920 x 1021 pixels  |

To add a new background image to the login page, do the following:

1. Access the Style Editor. See [Accessing the Style Editor](<Style Editor.mdx>).
2. Click the **Login Page** tab.
3. Under **Background**, click **Upload file**.
4. Click the **select file** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/46JidEisQOn8omfny09Cu_logofoldericon.png" alt caption size="10" width="16" height="16"}.
5. Navigate to the file of the image to add and double-click the file name. The application displays the image in place of the default background. This example shows the legacy layout.

<Paragraph indent="1">
  The application marks the unsaved value with a red triangle :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/EHiyYfMVQWob8kZOS2-vm_styleeditorredtriangle.png" alt caption size="10" width="7" height="14"} at the left and the **revert** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/uzqDpiJCmf14BiE72Ypet_stylereverticon.png" alt caption size="10" width="17" height="16"} at the right.
</Paragraph>

6. Click **Save**.

## Editing the Title Text on the Login Page

To edit the title text on the login page, do the following:

1. Access the Style Editor. See [Accessing the Style Editor](<Style Editor.mdx>).
2. Click the **Login Page** tab.
3. Under **Title**, click in the **Title Text** field.
4. Edit the information.

<hint type="info">
  The application provides a title in English. You cannot localize this text.
</hint>

<Paragraph indent="1">
  The application marks the unsaved value with a red triangle :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/EHiyYfMVQWob8kZOS2-vm_styleeditorredtriangle.png" alt caption size="10" width="7" height="14"} at the left and the **revert** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/uzqDpiJCmf14BiE72Ypet_stylereverticon.png" alt caption size="10" width="17" height="16"} at the right.
</Paragraph>

5. To keep your text changes, click **OK**.
6. Click **Save**.

## Adding a New Logo to the Login Page

The maximum logo dimensions are 650 pixels high by 650 pixels wide. You can scale your logo to fit the login page.

To add a new logo in the login page, do the following:

1. Access the Style Editor. See [Accessing the Style Editor](<Style Editor.mdx>).
2. Click the **Login Page** tab.
3. Under **Logo**, verify that Show is set to *true*.
4. Click **Upload file**.
5. Click the **select file** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/46JidEisQOn8omfny09Cu_logofoldericon.png" alt caption size="10" width="16" height="16"}.
6. Navigate to the file of the logo to add and double-click the file name. The application displays the new logo in place of the default logo on the login page.
7. Adjust the scale of the logo.

<hint type="info">
  If a logo is too large, it displaces the other objects on the login page, forcing users to scroll to enter their user names and passwords.

  Scaling a logo greater than 100% reduces its image quality.
</hint>

<Paragraph indent="1">
  The application saves the value with a red triangle at the left and the **revert** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/uzqDpiJCmf14BiE72Ypet_stylereverticon.png" alt caption size="10" width="17" height="16"} at the right.
</Paragraph>

8. Click **Save**.

## Adding Customized Text Above the Copyright on the Login Page

You cannot edit the \{\{FRS\_VARS.ProductName}} copyright statements. But you can add text that appears above them on the login page. To add text above the \{\{FRS\_VARS.ProductName}} copyright, do the following:

1. Access the Style Editor. See [Accessing the Style Editor](<Style Editor.mdx>).
2. Click the **Login Page** tab.
3. Under **Copyright**, click in the **Additional Text&#x20;**&#x66;ield to open the **Copyright** dialog box.
4. Edit or add copyright text.

<Paragraph indent="1">
  To access the other text tools, click the **menu** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/xp2i9MMHwwvdFfSbNbW0V_menuicon223x18.png" alt caption size="10" width="23" height="18"}. You can also widen the **Translation** window until all text tools are visible.
</Paragraph>

5. Enter your text:
   - To apply a text style, highlight the text to be styled and then click the appropriate text tool.
   - To enter text in a different language, click the **Language** menu and choose a language.
   - To add a hyperlink, highlight a word or phrase in the text and click the **hyperlink** tool :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/YEZ9r5JDYeGve_FpqRrFu_linkicon.png" alt caption size="10" width="17" height="16"}. Enter the link in the pop-up window and click **OK**.
6. Click **OK**.
7. Optional: To test a hyperlink, in the right pane, right-click the link and choose **Open in new tab**. If the address is valid, the destination site opens in the new browser tab.

<Paragraph indent="1">
  The application marks the value with a red triangle :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/EHiyYfMVQWob8kZOS2-vm_styleeditorredtriangle.png" alt caption size="10" width="7" height="14"} at the left and the **revert** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/uzqDpiJCmf14BiE72Ypet_stylereverticon.png" alt caption size="10" width="17" height="16"} at the right.
</Paragraph>

8. Click **Save**.

## Adding an Image to the Left Bar on the Login Page

When you fill the left bar using an imported image, \{\{FRS\_VARS.ProductName}} anchors the image at the upper left corner and crops the image to fit the bar or enlarges the image to fill the bar. You can set the width of the left bar but not its height:

- On a 1680 x 1050 monitor, the left bar can be up to 991 pixels high.
- On a 1920 x 1080 monitor, the left bar can be up to 1021 pixels high.

First determine the dimensions of the left bar as it appears on your production servers. Then choose an image, without excessive enlargement or cropping, to fill the left bar.

To add a new background image to the login page, do the following:

1. Access the Style Editor. See [Accessing the Style Editor](<Style Editor.mdx>).
