You can attach documents to Knowledge Base articles. When you search a Knowledge Base article, the application also indexes and searches all attachments. \{\{FRS\_VARS.ProductName}} supports the *txt*, *doc*, *docx*, *xls*, *xlsx*, and *pdf* file formats. You can add attachments in other formats, but the application does not search them.

<hint type="info">
  Ensure that your Microsoft SQL server is enabled to search PDF files, if you plan to use this feature.
</hint>

You can only attach one searchable document to a Knowledge Base article at a time. If you attempt to upload another document, the application overwrites the original document with the new document.

## Attaching a Document to a Knowledge Base Article

1. Log in to \{\{FRS\_VARS.ProductName}} as an \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
2. Open a Knowledge Base article in the **Knowledge** workspace.
3. Click the **Details** tab.
4. Click **Upload** button in the **Attachment** section and click **Save**.

## Viewing a Document Attached to a Knowledge Base Article

You can view the attached document within a Knowledge Base article.

1. Log in to \{\{FRS\_VARS.ProductName}} as an \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
2. Open a Knowledge Base article in the **Knowledge** workspace.
3. Click the **Details** tab.
4. Click the attached file name before the **Upload** button in the **Attachment** section.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/0er47FTZEGUqDTCGcnLXG_attachment-flow-view.png" position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/attachment-flow-view.png" githubPath="Content/Resources/Images/attachment-flow-view.png" showCaption="false"></Image>

## Removing a Document from a Knowledge Base Article

1. Log in to \{\{FRS\_VARS.ProductName}} as an \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
2. Open a Knowledge Base article in the **Knowledge** workspace.
3. Click the **Details** tab.
4. Click the attached file name before the **Upload** button in the **Attachment** section.
5. Click the :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/USPl9Egu4Y2Blx7HKC3mA_deleteredx.png" alt caption size="10" width="14" height="13"} icon next to attached file name.
   The application removes the attachment from the Knowledge Base article.
