Follow these steps to support a URL in a dashboard: 

1. Log into the Service Desk Console.
2. Click the name of your default dashboard. For example, if you are logged in as a Service Desk Analyst, click on the down arrow next to the words Service Desk Analyst.
3. Click **All Dashboards** on the bottom right of the drop-down list. The application displays a list of dashboards available to your role.
4. Click the **edit** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/WTLKbMa5PI7qeazzbj0FD_edit-old.png" alt caption size="10" width="16" height="16"}next to a dashboard.
5. Expand **Special Parts** in the **Dashboard Objects** section.
6. Click **Display URL**  at the bottom of the **Special Parts** section.

*Display URL Icon*

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/VkWRtlmhm6EC7Ht33A6q4_displayurl.png" alt="Image of the Display URL icon." position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/Display_URL.png" githubPath="Content/Resources/Images/help-images/Display_URL.png" showCaption="false"></Image>

7. In the **Create New Command Link List Dashboard Part** dialog box, do the following:

*Create New Command Link List Dashboard Part*

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/iPw9Z_5tq6UeROycO8Aq4_command-link-dashboard-part.png" alt="Creating a new command link list dashboard part." position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/Command Link Dashboard Part.png" githubPath="Content/Resources/Images/help-images/Command Link Dashboard Part.png" showCaption="false"></Image>

<Paragraph indent="1">
  a. In the **Name** field, enter a name.
</Paragraph>

<Paragraph indent="1">
  b. Optionally, enter a description in the **Description** field.
</Paragraph>

<Paragraph indent="1">
  c. In the **Dashboardparts** field, select **Display URL** from the drop-down list.
</Paragraph>

<Paragraph indent="1">
  d. In the **Parameters** field, enter a valid URL. For example, \{"URL":"http\://www\.ivanti.com"}.
</Paragraph>

<Paragraph indent="1">
  e. Click **Preview** to view your output.
</Paragraph>

<Paragraph indent="1">
  f. Click **Save**.
</Paragraph>

8. Return to the Service Desk Console and view the dashboard containing the URL dashboard part.

*URL in Dashboard*

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/yameIygNpCbYHSWdDRNLh_heatsoftware.png" alt="Image of a URL displaying in a dashboard." position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/HEat_Software.png" githubPath="Content/Resources/Images/help-images/HEat_Software.png" showCaption="false"></Image>

If the application cannot display the external URL in another application, it displays an empty page.
