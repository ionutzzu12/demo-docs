## Adding a Microsoft Azure Application

1. Log into [https://portal.azure.com](https://portal.azure.com/)
2. Select Azure Active Directory.
3. Create a new Active Directory or use an existing Active Directory.
4. From **App Registration**, create a new Application Registration and provide the following information:
   1. **Name**: A contextual name for the application, such as \{\{FRS\_VARS.ProductName}} Prod.
   2. **Redirect URI**: Select the option Web and enter the \{\{FRS\_VARS.ProductName}} tenant URL in the format - https\://\{\{tenant\_url}}/handlers/sso/OIDC/AuthResultHandler.ashx.

## Creating a Neurons for ITSM Authentication Provider

  To set up \{\{FRS\_VARS.ITSM}} users for authentication via the \{\{FRS\_VARS.Neurons\_platform}}, refer to [Setting up ITSM users for authentication via the Neurons Platform](../../../setting-up-itsm-users-for-authentication-via-the-neurons-platform.mdx) and [Authentication](https://help.ivanti.com/ht/help/en_US/CLOUD/vNow/authentication.htm). ## Security Considerations When Using Microsoft Azure

\{\{FRS\_VARS.ProductName}} application servers must be able to initiate outbound connections to the following endpoints:

- Token verification URL
- Certificate URL

All URLs have the following URL pattern: https\://login.microsoftonline.com/\*.
