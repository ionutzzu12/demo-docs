To ensure data segregation, each tenant organization is assigned its own secure database within the \{\{FRS\_VARS.ProductName}} Data Center.

Users access \{\{FRS\_VARS.ProductName}} over the Internet using a browser. Communication between the browser and \{\{FRS\_VARS.ProductName}} is secured using HTTPS.

Application functions, such as email, quick actions, or business rules, use internal safeguards to prevent hacking.
