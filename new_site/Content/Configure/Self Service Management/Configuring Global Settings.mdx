**Role**: Administrator

***Minimum Version****:&#x20;*\{\{Primary.Company}}\{\{Primary.Product}}*&#x20;2024.2.*

The Global Settings page lets you customize the user interface to make it clear and easy to use. You can change the page layout, banner style, company logo, headers, fonts, text colors, and button colors. This helps match the look to your brand. You can also edit the welcome message, organize menu items, and turn on chat support to help self service users to connect with support team for any issues.

You access the Global Settings user interface only after you configure the user for UI v3 in the Style Editor.

The **Global Settings** page displays the following sections:

<ExpandableHeading>
  ## Page Layout

  Use the **Page Layout** to modify the content width, margins, and grid spacing for the home page layout.

  To configure the page layout:

  1. Log in to \{\{Primary.Product}} as a **Self Service** user, and change role to **Administrator**.
  2. Click **Edit Page** on the top right corner of the page banner.

  <Paragraph indent="1">
    The **Page Designer** page is displayed.
  </Paragraph>

  3. Click **Global Settings**.
  4. In the **Page Layout** section, in **Content width** select either **Boxed** or **Full Width** from dropdown list.
     - For Full Width, you can specify **Left margin**, **Right margin**, and **Grid Spacing**.
     - For Boxed, you can specify only **Grid Spacing**.

  <Paragraph indent="2">
    Margin Settings: Set the left and right margins between 24px and 48px (this is applicable only for full width).
  </Paragraph>

  <Paragraph indent="2">
    Grid Spacing: Set spacing between columns (horizontal) and rows (vertical) from 8px to 24px.
  </Paragraph>
</ExpandableHeading>

<ExpandableHeading>
  ## Default Banner Style

  Use the **Default Banner Style** section to modify image, color, and text across the workspaces.

  1. To modify the image, click **Edit** :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/XoBHbFgyDeNnNtn2gCeXy_edit-icon.png" alt caption size="10" width="19" height="19"} icon, select the image from your library.
  2. To change the background color, click color box and select the desired background color from the palette.
  3. Turn on the **Use color as image overlay** toggle.

  <Paragraph indent="1">
    The selected color appear as a transparent layer over the image, this improves the text readability and visual consistency.
  </Paragraph>

  4. You can customize a banner from the **Settings** panel, select **Use custom banner style** in center panel.
  5. From the **Banner Size** dropdown list, select desired size **Small**, or **Medium**, or **Large**.
     - Banner sizes varies for different page types, by default the sizes are:
       - Home Page - **Large** size
       - Enterprise Landing Page - **Medium** size
       - Inner Pages - **Small** size
     - Recommended image formats are **.png** and **.jpg**
     - Recommended image dimensions:
       - **Large** size - **width = 1920px** and **height** = **512px**
       - **Medium** size - **256px**
       - **Small** size - **128px**
  6. Turn on/off the **Display Banner** toggle to show/hide the banner in each workspace.
  7. To enable welcome messages, turn on **Welcome Message** toggle, and you can add **Message Text** and **Message Sub Text** on the banner.
  8. To enable search option, turn on the **Search** toggle, enter text in **Place Holder Text** to set as default for user to view the search options , and then select **Search Bar Size** from dropdown list.

  Default Banner Style settings for Service Catalog

The maximum supported file size for banner images is **2 MB**. If the uploaded file exceeds this limit, the system displays an error message.

Also, if you have an old sections with background image as banner you need to remove that section, refer to To add sections and rows in [Working with Page Settings - Home Page](<Working with Page Settings.mdx>) and enable the banner again to reflect the custom Banner on Self Service home page.

<ExpandableHeading>
  ## Typography

  Use the **Typography** section to set to define the default fonts and sizes.

  1. To set the font type and size in banner section for body text, specify the following:
     - Select the desired font from the **Body font** dropdown to define the font style for the text beneath the title.
     - Enter or select above arrow from the **Body font default size** dropdown to define the font style for the body text.
  2. To set the font type and size in banner section for header text, specify the following:
     - Select the desired font from the **Header font** dropdown to define the font style for the title.
     - Enter or select above arrow from the **Header font default size** dropdown to define the font style for the header.
</ExpandableHeading>

<ExpandableHeading>
  ## Buttons

  Use the **Buttons** section to configure the look and feel of the buttons displayed across the pages.

  1. Click Edit :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/XoBHbFgyDeNnNtn2gCeXy_edit-icon.png" alt caption size="10" width="19" height="19"} icon adjacent to **Primary Button** or **Secondary Button** to modify the background color, text color, border type, and border radius.
  2. Click **Ok**.
  3. You can also click **Reset to default style** to reset to default.
</ExpandableHeading>

<ExpandableHeading>
  ## Header

  Use th&#x65;**&#x20;Header** section to customize the page header. You can set the title text color, background color, and icon color.

  1. Toggle the **Show title** button to display or hide the title.
  2. Enter the desired title in **Title text** field.
  3. Select the desired background color from the palette, to set the color in banner section above title.
  4. Select the desired background color from the palette, to set the color for icons.

  <hint type="info">
    The icon color will be applied to all icons in the header, including the menu icon.
  </hint>

  5. Optionally, you can enable or disable the **Search** bar, **Alerts**, and **Shopping cart** features based on your requirements.
</ExpandableHeading>

  ## Configure Alerts

  You can configure alerts to notify users when requests are updated or to display relevant articles. To configure alerts, do the following:

  1. Select turn on **Alerts** toggle.

  <Paragraph indent="1">
    The **Configure** link is enabled.
  </Paragraph>

  2. Click Configure.

  <Paragraph indent="1">
    The **Configure Alerts** window is displayed and has following tabs:
  </Paragraph>

  <Paragraph listStyleType="disc" indent="2">
    **Data Source** – Select options to configure business objects and alert display settings.
  </Paragraph>

  <Paragraph listStyleType="disc" indent="2">

  </Paragraph>

  ## Chat Support

  Use the **Chat Support** tab to enable, disable, or edit the **Chat with Support** and **Chatbot** features in the \{\{FRS\_VARS.Self\_Service\_Portal-Self-Service}}.

  <hint type="info">
    You also must enable the **Enable "Chat with support" button** option in the **Role Details** configuration to use chat support in the \{\{FRS\_VARS.Self\_Service\_Portal-Self-Service}}. Refer to :Link[Creating a Role]{label="Creating a Role" overridedLabel="Creating a Role" spaceId docId="iUGD9ji_VgJkk9DVIGvHo" version="v2" docAnchorId="#roles_1236189643_1266760" loadingMethod="dynamic" newTab="false" githubPath="../Roles/Setting-up-Roles.htm#roles_1236189643_1266760" href="../Roles/Setting-up-Roles.mdx"}.
  </hint>

  **To enable the Chat with support option**

  1. Log into \{\{Primary.Product}} as an \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
  2. In the \{\{FRS\_VARS.Configuration\_console}}, set the **ChatEnabled** global constant to **True**.
  3. Select **Build** > **User Interface Styles** > **Style Editor**, in the **UI** column, select **UI v3**.
  4. Select \{\{FRS\_VARS.Self\_Service\_Portal-Self-Service}} role to enable the chat feature.
  5. In the page designer, go to **Global Settings** > **Chat Support** and select **Chat with support**.
  6. Click **Save**.

  **To enable or disable chat support:**

  1. Select **Global Settings**, and select **Chat Support**.
  2. Select or clear the **Chat with Support** or the **Chatbot** checkbox to enable or disable the chat options.

  When both options are enabled, the **Chat with Support** and **Chatbot** icons are visible in the \{\{FRS\_VARS.Self\_Service\_Portal-Self-Service}} UI.

  **To edit the Chatbot button and integration settings:**

  1. Select the **Edit** icon for the **Chatbot** entry to open the **Chatbot Settings** form.
  2. Edit the button styling message and color settings as required.
  3. Select the **Integration** tab. The **Integration Type** drop-down dispalys "URL"; this is currently the only selection available.
  4. Enter the URL for the Chatbot you want to connect to in the text box.
