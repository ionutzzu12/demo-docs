When creating an incident from Self Service, the system suggests related documents and articles, such as help guides, FAQs, and knowledge base articles. These documents are fetched using the Matching panel configuration.

Each business object there are Matching Panel settings within its associated Layouts. Using these settings, you can:

- Configure different sections such as FAQs, Knowledge Base, Help Guides, and more to organize and list a variety of documents.
- Create call-to-action buttons that generate incidents with information mapped to the corresponding incident fields.
- Disable case deflection steps by turning off the Matching Panel for a given layout.

## Configuring the Matching Panel

To create and edit Matching Panel settings:

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/bTVg5XaBHoj7emqQc1bJv_create-edit-match-panel-settings.png" position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/SelfServiceMgmt-images/create-edit-match-panel-settings.png" githubPath="Content/Resources/Images/SelfServiceMgmt-images/create-edit-match-panel-settings.png" showCaption="false"></Image>

1. Log in to \{\{FRS\_VARS.ProductName}} as an \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
2. Open the Configuration console > **Build** > **Business Objects** > \{\{FRS\_VARS.Incident}}.
3. Click **Layouts** tab, select a layout, and open the **Form Edit** view.
4. Click **Edit Matching Settings** next to **Enable Matching panel**.
5. Select **Edit Preview** of **Object Name**. You can also add another object using the :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/Kc7G2l98uGFtWPlc5ZMqC_add-icon.png" alt caption size="10" width="15" height="15"} button.
6. Enter the **Preview Item format** details such as **Title Field**, **Preview Text Field**, select a **Form for Hover Panel**, and click **Add New Button**.
7. Enter **Button Label**, check or enable **Link matched Problem with main Incident#** select the **Incident#.** from the dropdown, and **Save** the configuration.

## Creating an Incident from Service Catalog

While creating an incident from Service Catalog as a Self Service user you select the template, the incident fetches all the information from the template and update the request with the same information.

1. Log in to \{\{FRS\_VARS.ProductName}} as \{\{FRS\_VARS.Neurons\_ITSM\_-\_Self\_Service\_User}}.
2. Go to Service Catalog and select a service request card.
3. Enter title and description, and click **Continue**.
4. Select the related resource based on the **Object Name** list and click the button created from the **Matching Panel** settings.
5. Select **Urgency** and click **Submit**.
