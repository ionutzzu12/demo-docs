title: Enabling the Chat with Support Option
docTags: 
createdAt: Tue Jun 30 2026 17:16:50 GMT+0300 (Eastern European Summer Time)
---

Self Service users can chat with your support teams. Enable this feature in the \{\{FRS\_VARS.Configuration\_console}} and set the appropriate global constant. The way you enable the chat feature depends on the **UI&#x20;**&#x73;election for the role.

**To enable the Chat with support option for UI V2 (formerly Modern):**

1. Sign in to Ivanti \{\{FRS\_VARS.ProductName}} as an Administrator.
2. In the Configuration console, set the **ChatEnabled** global constant to **True**.
3. Select **Build** > **User Interface Styles** > **Style Editor**.
4. In the **UI** column, select **UI v2 (formerly Modern)**.
5. Select **Save**.
6. From the left navigation bar, select **Users and Permissions** > **Roles and Permissions**.
7. Select **Self Service**.
8. In the **Role Details&#x20;**&#x74;ab under **Branding Options**, select **Enable "Chat with support" button**.
9. Set the **ChatEnabled** global constant to **True**.

**To enable the Chat with support option for UI V3:**

1. Sign in to Ivanti \{\{FRS\_VARS.ProductName}} as an Administrator.
2. In the Configuration console, set the **ChatEnabled** global constant to **True**.
3. Select **Build** > **User Interface Styles** > **Style Editor**.
4. Find the role for which you want to enable the chat feature.
5. In the **UI** column, select **UI v3**. In the page designer, go to **Global Settings** > **Chat Support** and select **Chat with support**.
6. Select **Save**.

**To Configure the Chat User Interface for UI V3:**

1. Sign in to Ivanti \{\{FRS\_VARS.ProductName}} as an Administrator.
2. In the Configuration console, go to **User Interface Styles** > **Style Editor**.
3. Find the role for which you want to configure the chat user interface and click on the :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/qJpTqwX_2CkRd6SG_I7nP_2025-08-1217-57-4623x21.png" alt caption size="10" width="23" height="21"} Edit icon.
4. On the Designer page, go to **Global Settings**.
5. Click on **Chat Support**.
6. Click on the **Edit** :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/ky_kCI6JiZsrHLw5WNhU__2025-08-1218-24-0323x16.png" alt caption size="10" width="23" height="16"} icon next to **Chat with Support**.
7. In the **Chat with Support Settings** window, go to the **Chat Window** panel.
