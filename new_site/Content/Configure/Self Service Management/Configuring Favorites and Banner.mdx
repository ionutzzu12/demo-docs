**Role**: Administrator

The following procedure describes how to configure Banners and enable the **Add to Favorite** option in the Self Service Portal V3. You can set a default banner across all workspaces or override it with page-specific banners. Banner settings allow you to define images, overlay colors, text, and search behavior. You can also configure static or dynamic Hero banners with multilingual content and call-to-action links. Additionally, you can enable the **Add to Favorite** icon to let Self Service users quickly access frequently visited pages.

**Banner image requirements**

- **Recommended formats**: **.png**, **.jpeg**, **.gif**, and **.svg**
- **Recommended dimensions**:
  - Large: **width = 1920px** and **height** = **512px**
  - Medium: **width = 1920px** and **height** = **256px**
  - Small: : **width = 1920px** and **height** = **128px**
- **Maximum file size**: **2 MB**

<Paragraph indent="1">
  If the uploaded file exceeds this limit, the system displays an error.
</Paragraph>

- To use **.svg** files, add **\*.svg** to the allowed attachment types. By default, **.svg** is not included in the out-of-the-box (OOTB) list. If not added, the system returns an error when uploading an **\*.svg** file. To enable it, refer to [Allowed File Extensions](../Attachments/Attachment_Types.mdx).

<ExpandableHeading>
  ## Configuring a (Home Page) Banner with Search bar

  To configure a banner for a specific page:

  1. Open **Page Designer**.
  2. Select the target page from the **Page Selector**.
  3. Click the **Page Settings** tab.
  4. In the **Banner section**, configure the following settings:
     - **Display Banner**: Toggle on or off to show or hide the banner.
     - **Banner Style**: Use default or custom banner style.
     - **Banner Size**: Large or medium or small.
     - Click **Edit** to upload a **Background Image**.
     - **Background Color**: Choose a color for the background.
     - **Use Color as Image Overlay**: Overlay the background color on the image.
     - **Text Color**: Set the color for banner text.
     - **Welcome Message**: Toggle on to enable message display.
       - Message Text: Enter a greeting (e.g., “Hello #first\_name!”).
       - Message Sub Text: Add supporting text (e.g., “All the Facilities services you need”).
     - **Search**:
       - Set placeholder text.
       - Choose Search Bar Size (**Small**, **Medium**, or **Large**).
  5. Click **Publish** to apply the changes.
  6. Click the **Preview** button to preview prior to publishing.

  ## Recommended Banner Widths by Screen Resolution

  To ensure correct banner display across layouts, use approximately 75% of screen resolution.

  **Recommended Width (Based on Screen Resolution)**

  | Screen Resolution (x) | Recommended Width (0.75 × x) |
  | --------------------- | ---------------------------- |
  | 1920px (Full HD)      | 1440px                       |
  | 1600px                | 1200px                       |
  | 1440px                | 1080px                       |
  | 1366px                | 1025px                       |
  | 1280px                | 960px                        |

  **Banner Height Options**

  Banner height is determined by the selected size setting:

  | Banner Size | Height |
  | ----------- | ------ |
  | Large       | 480px  |
  | Medium      | 320px  |
  | Small       | 160px  |

  ## Configuring the Default Banner Style

  The Default Banner Style settings allow you to define a consistent banner across all workspaces in the Self Service Management Portal V3. You can configure default images, overlay colors, text, and search behavior that appear globally unless overridden at the page level.

  Use the **Default Banner Style** tab to set a banner with a default image, color, and text across the workspaces.

  To configure the default banner:

  1. Log in to Neurons for ITSM as **Administrator**.
  2. Go to Configuration console > **Build** > **User Interfaces Styles** > **Style Editor**.
  3. Select the edit icon in **Action** column of th&#x65;**&#x20;Self Service** role in **Exceptions / Custom roles** column.

  <Paragraph indent="1">
    The **Page Designer** appears.
  </Paragraph>

  4. Click **Global Settings** > **Default Banner Style** or
  5. Or click banner section in the center pane, you can customize a banner from the **Settings** pane.
  6. Enter the following details:
     - Upload a **Banner Image**.
     - Define a **Background color**.
     - Toggle on **Use color as image overlay** to overlay the background color on banner.
     - Select a **Text Color**.

  <hint type="info">
    Default Banner Sizes for Home page - **Large** size, Enterprise Landing Page - **Medium** size, and for other Inner Pages with **Small** size.
  </hint>

  **Customize the Banner Section for a Workspace**

  To customize banner section for \{\{FRS\_VARS.Self\_Service\_Portal-Self-Service}} or other workspaces:

  1. In the **Settings** panel, toggle **Display Banner** to show or hide the banner.
  2. Select **Use Custom Banner Style** to apply a unique style for the selected workspace.
  3. From the **Banner Size** dropdown, select **Small**, **Medium**, or **Large**.
  4. To update the banner image:
     - Click **Edit** the image to upload a new image
     - Click **Delete** to remove the existing image.
  5. To change the banner's background color, click the **Background Color** and select the desired color.
  6. To update the font color of the **Welcome Message** and **Message Sub Text**, click **Text color** and select the required color.
  7. Toggle **Welcome Message** to enable or disable **Message Text** and **Message Sub Text** as required.
  8. Toggle **Search** to enable search bar. Enter custom **PlaceHolder Text** or retain the default text.
  9. From the **Search Bar Size** dropdown, select **Large** or **Medium** to define the input field size.
  10. To enable the transparent background for the search bar, toggle the **Semi-transparent background**.

  <hint type="info" indent="1">
    The **Background Color** option is grayed out if **Semi-transparent background** is enabled.
  </hint>

  11. To set a solid background color for search bar, click **Background Color** and select a color.

  <Paragraph indent="1">
    To apply the selected color, toggle off the **Semi-transparent background**.
  </Paragraph>

  12. To change the search bar text color, click **Text Color** and select the required color.
  13. To adjust the corner radius of the search bar, enter a value or choose from the **Border radius** field.

  <Paragraph indent="1">
    The radius supports a minimum of 4 pixels and a maximum of 24 pixels. The default value is 24 pixels.
  </Paragraph>

  ## Configuring a Page-Specific dynamic Banner

  You can override the default banner by configuring a static banner for a specific page.

  **To configure a page level banner:**

  1. Open **Page Designer**.
  2. Select the target page from the **Page Selector**.
  3. Click the **Page Settings** tab.
  4. In the **Banner section**, configure the following options:
     - Upload an image.
     - Set the background and text colors.
     - Enter a title and optional subtext.
     - Enable or disable the search bar and other icons (e.g., Add to Favourite).
  5. Click **Publish** to apply the changes.

  <hint type="info">
    The application displays the default banner only if no page-specific banner is configured.
  </hint>
</ExpandableHeading>

<ExpandableHeading>
  ## Enable the Add to Favorite Option

  The **Add to Favorite** option allows users to mark specific pages as favorites in Self Service Management Portal V3.

  To enable this feature:

  1. Log in to \{\{Primary.Product}} as \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
  2. Open **Page Designer**.
  3. Select the **Service Catalog** or **Knowledge Base** page from the **Page Selector** dropdown list.
  4. Select the **Self Service** page from the **Workspace Selector** dropdown list.
  5. Select the page view you want to edit from the **View Selector** dropdown list - **Desktop**, **Mobile**, or **Tablet**.
  6. Click the **Page Settings** tab.
  7. In the **Additional Settings** section, locate the **Add to Favourite** Icon toggle.
  8. Toggle **Add to Favourite** Icon to **On**.
  9. Click **Publish** to apply the changes.

  <hint type="info">
    The Favorite feature is exclusively available for the Self Service role and doesn't apply to other roles. Furthermore, selecting the **Favorites** checkbox does not show any templates.
  </hint>
</ExpandableHeading>
