## To access the \{\{FRS\_VARS.Service\_Catalog-Service-Catalogue.}} page settings:

1. Log in to \{\{FRS\_VARS.ProductName}} \{\{FRS\_VARS.Self\_Service\_Portal-Self-Service}} portal, and then change the role to \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
2. Click **Edit Page** at the top right of the window. The **Page Designer** opens.
3. Select the \{\{FRS\_VARS.Service\_Catalog-Service-Catalogue.}} page from the **Page Selector** drop-down list.
4. Select the page view you want to edit from the **View Selector** drop-down list - **Desktop**, **Mobile**, or **Tablet**.

## To Configure Preset Filters

You can use or create a preset filter to categories different services and show Service Catalog options customized for a target user or audience.

To create a preset filter:

1. Log in to \{\{FRS\_VARS.ProductName}}\{\{FRS\_VARS.Self\_Service\_Portal-Self-Service}} portal as \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
2. Switch to **Service Catalog** workspace.
3. Click **Preset Filters** and select **Add new preset**.
   - Enter **Filter Name**.
   - Select **Categories** to show in the preset.
   - Optionally, select **Set as Default** to set the particular preset as default.

<hint type="info" indent="2">
  A preset filter has a flash icon if it is set as default and only one preset can be set as default. The **Publish** button remains disabled while you select **Set as Default** to a preset, further edits are required to enable the **Publish** button.
</hint>

<Paragraph listStyleType="disc" indent="2">
  Click **Save** to save the preset.
</Paragraph>

**To edit or delete a preset filter:**

- To edit - select the eye icon next to the preset name.
- To delete - select the bin icon next the eye icon.

## To Configure the Generic Request Config

If a Self Service user can't find any results on the **Service Catalog** workspace, they have the option to create a Generic Request. They can use the **Generic Request Template&#x20;**&#x64;rop-down list to select from existing request offerings, form offerings, and external offerings.

**To configure a generic request:**

1. Log in to \{\{Primary.Product}} as \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
2. Click **Edit Page**. The **Page Designer** page appears.
3. From the workspace drop-down list available at top center, select **Service catalog**. The **Generic Request Config** section is displayed in the left pane.
4. Expand **Generic Request config**. Select a request from the **Generic Request Template** drop-down list. The list displays the existing request offerings, Form offerings, and external offerings.
5. Click **Publish**. A confirmation message is displayed, indicating that the configuration has been published successfully.

## Manage Additional Settings in Service Catalog

The **Service Catalog** workspace enables you to activate the following features:

| Feature                           | Description                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| --------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Submission Dialog Box for Mobile  | This feature allows you to toggle the display of a dialog box on a mobile device when a Self Service user submits a request or incident from a request offering or form offering, respectively. By default, this option is enabled. If disabled, the dialog will not appear, and users will be directly navigated to the created request page or form offering page                                                                           |
| Submission Dialog Box for Desktop | This feature allows you to toggle the display of a dialog box on a desktop when a Self Service user submits a request or incident from a request offering or form offering, respectively. By default, this option is enabled. If disabled, the dialog box will not appear, and users will be directly navigated to the created request page or form offering page.                                                                            |
| Add to favorites                  | This feature allows you to toggle the display of a **Favorites&#x20;**&#x69;con for Self Service users to mark records as favorites.                                                                                                                                                                                                                                                                                                          |
| Share as Service Catalog          | This feature allows you to toggle the display of the **Copy link** and **Send email&#x20;**&#x6F;ptions, allowing Self Service users to copy the service request link and paste it into another application, such as an email. Additionally, users can directly send the email using Outlook.                                                                                                                                                 |
| Save for later                    | This feature enables you to toggle the display of the **Save for later&#x20;**&#x6F;ption for Self Service users when creating a service request in the Self-Service portal without immediately submitting it. For instance, if you do not have all of the information needed to submit the service request, but want to start the service request, a service request that has been saved for later appears as a draft to Self Service users. |

<ExpandableHeading>
  ## Enable or Disable the Submission Dialog Box for Mobile

  1. Log in to the \{\{FRS\_VARS.ProductName}} \{\{FRS\_VARS.Self\_Service\_Portal-Self-Service}} portal, and then change the role to Administrator.
  2. Click **Edit Page**. The **Page Designer&#x20;**&#x70;age appears.
  3. Select the **Service Catalog** workspace from the drop-down list.
  4. Click the **Page Settings** tab, and then click **Additional Settings**.
  5. Under **Additional Settings**, click the **Submission Dialog Box for Mobile** toggle button to enable or disable the display of the dialog box on the mobile device.
  6. Click **Publish** to publish the configuration.

  <hint type="info">
    If this feature is disabled, the Self Service user will be redirected to **My Items** while submitting a Service Request or an Incident.
  </hint>
</ExpandableHeading>

  ## Enable or Disable the Favorites icon

  Enabling the Global Constant:

  1. Log in to \{\{Primary.Product}} as \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
  2. Open the Configuration Console.
  3. Select **Global Constants**.
  4. Set the **SSMFavoritesEnabled** option to **True**.
  5. Click **Save**.

  Enabling or disabling the Favorites icon from the Page Designer:

  1. Log in to \{\{FRS\_VARS.ProductName}} \{\{FRS\_VARS.Self\_Service\_Portal-Self-Service}} portal, and then change the role to Administrator.
  2. Click **Edit Page**. The **Page Designer** page appears.
  3. Select the **Service Catalog** workspace from the drop-down list.
  4. Click the **Page Settings** tab, and then click **Additional Settings**.
  5. Click the **Add to Favorites** toggle button to enable or disable the display of the icon in the Self Service portal.
  6. Click **Publish** to publish the configuration. The Self Service user can now click the :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/NxEYwUffliA4T4mJe-nhK_favorites.png" alt caption size="10" width="17" height="14"} icon that appears under each service record and mark them as favorite in the Self Service Management portal.

  ## Enable or Disable the Copy Link and Send Email Option in the Service Catalog

  1. Log in to \{\{FRS\_VARS.ProductName}} \{\{FRS\_VARS.Self\_Service\_Portal-Self-Service}} portal, and then change the role to Administrator.
  2. Click **Edit Page**. The **Page Designer** page appears.
  3. Select the **Service Catalog** workspace from the drop-down list.
  4. Click the **Page Settings** tab, and then click **Additional Settings**.
  5. Under **Additional Settings**, click the **Share in Service Catalog** toggle button to enable or disable the display of the **Copy link** and **Send email** option in the Self Service Management portal.
  6. Click **Publish** to publish the configuration. You can now view the :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/iUy8ySdKi8SG2NhtV_8g1_share-icon14x13.png" alt caption size="10" width="14" height="13"} button under each service record and use the options in the Self Service Management portal.

  Using the share button, you can **Copy link** or **Send email** to share a service request or an incident.

<ExpandableHeading>
  ## Enable or Disable Save for Later

  1. Log in to \{\{FRS\_VARS.ProductName}}\{\{FRS\_VARS.Self\_Service\_Portal-Self-Service}} portal, and then change role to \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
  2. Click **Edit Page**. The **Page Designer** page appears.
  3. Select the **Service Catalog** workspace from the drop-down list.
  4. Click the **Page Settings** tab, and then click **Additional Settings**.
  5. Under **Additional Settings**, click the **Save for later** toggle button to enable or disable the display of the **Save for later** option for Self Service users.
  6. Optional. Click the **Auto Save** toggle button to automatically save the service requests after the set interval duration. Set the timeout value (in seconds) in **Auto save Interval** field.
  7. Click **Save**.
  8. Click **Publish** to publish the configuration. You can now view the **Save as Draft** option in the Self Service Management portal.

  Also, as a Self Service user, you can access all the draft items in the draft side bar from the **Draft** button in the **Service Catalog** page.
