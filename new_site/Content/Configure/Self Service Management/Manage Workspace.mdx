**Role**: Administrator

A workspace in the Self Service Portal V3 is a role based, configurable area that organizes key modules into a unified interface for easy access. Workspaces are designed to improve usability by displaying relevant content and enabling direct interaction based on the user’s role and permissions.

**By default, the workspace includes the following modules:**

- Home
- Service Catalog
- My Items
- Announcements
- Knowledge
- Search

**In addition to the default modules, administrators can configure the following:**

- FAQ
- Alerts

<hint type="info">
  You have the option to include the **Alert** and **FAQ** workspaces on the Home page. To enable alert notifications and FAQs, refer to the [Creating and Configuring Workspace](<Creating and Configuring Workspace.mdx>) in the Administrator Help.
</hint>

## **Customization of Workspace Using Page Designer**

Use Page Designer to tailor workspace elements and provide a personalized user experience. Administrators can:

- Modify background colors, fields, menus, buttons, and text.
- Apply diverse color palettes, logos, and images.
- Drag and drop interface components for quick layout adjustments.
- Apply themes to individual sections such as **Home**, **Service Catalog**, **Knowledge Base**, **FAQ**, **My Items**, and **Announcements**.
- Instantly preview design changes using the Preview option within the Page Designer editor.

<hint type="info">
  All workspace layout and content configurations are managed through Page Designer. For more information, refer to [Page Designer](<Working with the Page Designer.mdx>) in the Administrator Help.
</hint>

## To Access the Workspace from the Style Editor

Administrators can enable or disable the Self Service portal V3 workspace for specific roles using the Style Editor.

To access the workspace from the Style Editor:

1. Sign in to **Neurons for ITSM** as an administrator.
2. From the Configuration Console, click **Build** > **User Interface Styles** > **Style Editor**.
3. Under the **Role** column, locate the **Self Service** section.
4. Use the dropdown in the **UI** column, select the **UI v3** option.

For more information please refer to [Working with Style Editor](<Customize User Interfaces with the Style Editor.mdx>) in the Administrator Help.
