If the application contains translations, you can edit the translated values.

1. From the Configuration Console, open the Translation Tool as described in :Link[Accessing the Ivanti Translation Tool]{label="Accessing the Ivanti Translation Tool" overridedLabel="Accessing the Ivanti Translation Tool" spaceId docId="_EuTeCZuBV-NNqINHqp4H" version="v2" docAnchorId="#Accessing_the_Translation_Tool" loadingMethod="dynamic" newTab="false" githubPath="Using the Translation Tool.htm#Accessing_the_Translation_Tool" href="Using the Translation Tool.mdx"}.

*Translation Tool*

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/DN3jvbdaS_15LJ1CIIWdD_translationtool2.png" position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/Translation_tool_2.png" githubPath="Content/Resources/Images/help-images/Translation_tool_2.png" showCaption="false"></Image>

<Paragraph listStyleType="disc" indent="2">
  The first column shows the property, that is, the specific place, where a word or phrase is used in the current form.
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  The third column shows the word or phrase as it appears right now for the language that you selected.
</Paragraph>

2. In the third column, click a translated string to open it for editing.
3. Do one of the following:
   - Select a string from the drop-down list, if the application lists any suggestions.
   - Enter new text.
4. Press **Enter** to record (but not save) the change. The application displays a red triangle next to the unsaved text string. This marks the changed string for reference.
5. Change additional strings as necessary.
6. Do one of the following:
   - To save your changes, click **Save**. New strings are indicated by logged-in user name and date and time created. Modified strings are indicated by logged-in user name and date and time changed.
   - To remove all of the changes that made, click **Revert**. The application returns the strings to their original settings.
7. Optional. Import the translation strings to a spreadsheet. See :Link[Importing Language Records from Microsoft Excel]{label="Importing Language Records from Microsoft Excel" overridedLabel="Importing Language Records from Microsoft Excel" spaceId docId="8dYPJVhCQi23Zi1YghMZP" version="v2" docAnchorId="#Importin" loadingMethod="dynamic" newTab="false" githubPath="Importing Exporting Translation Spreadsheets.htm#Importin" href="Importing Exporting Translation Spreadsheets.mdx"}.
