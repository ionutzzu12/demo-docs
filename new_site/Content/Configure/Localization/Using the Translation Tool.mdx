## About the Ivanti Translation Tool

\{\{FRS\_VARS.ProductName}} provides a translation tool that lets you customize the text strings used in the application for each supported locale. You can use the translation tool to do the following:

- Specify which user interface areas use customized translation strings.
- Update and modify the translation strings of one of the supported languages.
- Import or export a spreadsheet containing some or all of the translation strings for a language.

Do not attempt to add translation strings by importing a spreadsheet to which you have added rows or columns. See :Link[About Importing or Exporting a Translation Spreadsheet]{label="About Importing or Exporting a Translation Spreadsheet" overridedLabel="About Importing or Exporting a Translation Spreadsheet" spaceId docId="8dYPJVhCQi23Zi1YghMZP" version="v2" docAnchorId="#About_Importing_or_Exporting_a_Translation_Spreadsheet" loadingMethod="dynamic" newTab="false" githubPath="Importing Exporting Translation Spreadsheets.htm#About_Importing_or_Exporting_a_Translation_Spreadsheet" href="Importing Exporting Translation Spreadsheets.mdx"}.

## Accessing the Ivanti Translation Tool

You can access the Translation Tool through the **Languages** workspace.

1. From the Configuration Console, click **Configure > Organization Profile > Languages** to open the **Languages** workspace.
2. Click **Translation Tool**. The application displays the **Translation Tool** workspace.

*Translation Tool Workspace*

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/CU-itsGUSoVyP4SCe_poD_translationtoolworkspace.png" position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/translation_tool_workspace.png" githubPath="Content/Resources/Images/help-images/translation_tool_workspace.png" showCaption="false"></Image>

3. In the **To** field, select a target language.

You can also access the Translation Tool while editing a form, layout, or list.

1. From the Configuration Console, click **Build > Business Objects**.
2. Open a business object.
3. Click the **Forms** tab, **Lists** tab, or **Layouts** tab.
4. Open a specific form, list, or layout.
5. Do the following:

<Paragraph indent="1">
  a. Click **Translation Tool** next to the business object to translate.
</Paragraph>

<Paragraph indent="1">
  b. Select a language from the drop-down list.
</Paragraph>

## Using the Ivanti Translation Tool

To see the results of your changes, go to the \{\{FRS\_VARS.ProductName}} application and change your locale to the language that you made the changes to. For example, if you made changes in French, set your locale to French.

1. From the Configuration Console, open the Translation Tool as described in :Link[Accessing the Ivanti Translation Tool]{label="Accessing the Ivanti Translation Tool" overridedLabel="Accessing the Ivanti Translation Tool" spaceId docId="_EuTeCZuBV-NNqINHqp4H" version="v2" docAnchorId="#Accessing_the_Translation_Tool" loadingMethod="dynamic" newTab="false" href="Using the Translation Tool.mdx"}. The application displays the **Translation Tool** workspace.
2. Select the translation criteria.

| Field                                   | Description                                                                                                                                                                                                                                                                                                               |
| --------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Translate                               | Narrows the translation criteria:<br />Business object<br />Role<br />Object matching                                                                                                                                                                                                                                     |
| Name                                    | The object name. Select from the drop-down list. This list changes according to the item that you selected in the **Translate** field.                                                                                                                                                                                    |
| Area                                    | The area to translate. Select from the drop-down list:<br />Properties<br />Fields<br />Lists<br />List Properties<br />Forms<br />Form Properties<br />Searches<br />Quick actions<br />Layouts                                                                                                                          |
| From                                    | The source language. Choose from the drop-down list.                                                                                                                                                                                                                                                                      |
| To                                      | The target language. Choose from the drop-down list.                                                                                                                                                                                                                                                                      |
| Properties                              | Specific properties. This list changes according to the object name and area chosen. Select from the drop-down list or select **All**. If there are no options, the drop-down list is grayed out.                                                                                                                         |
| Autofill suggested existent translation | This lists all text strings, implemented and non-implemented.<br />If selected, the display area shows the implemented translation strings and all other translation strings in the database associated with the original-language term.<br />If not selected, only the currently implemented translation strings appear. |

3. Click **Search**. The application displays the translation strings that match the criteria.
