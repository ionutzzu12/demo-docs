**Role**: \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.

**Minimum Version**: \{\{Primary.Company}} \{\{Primary.Product}} 2023.4.

**Availability**: Both cloud deployments and on-premises environment.

<hint type="info">
  The language is translated for all UI elements except for text entered by the user. The text in the fields will automatically display from right-to-left but for it to display in Arabic, you need to write in Arabic using an Arabic supported keyboard.
</hint>

## Enable the Language

<hint type="info">
  This section is applicable only for on-premises customers. Cloud customers should reach out to Ivanti Operations team to get the language enabled.
</hint>

This feature is managed from the ConfigDB on each tenant. For example, if there is a production environment on one server and a staging/UAT environment on another, the feature must be enabled on both ConfigDBs for Arabic language to be available across all tenants.

If the feature record is not available, create it manually.

**To enable the Arabic language feature:**

1. In the ConfigDB, open the **Feature Management** workspace.
2. Change the **EnableArabicRTL** feature status to **Unconditionally ON**.

## Adding the Language

1. Log in to \{\{Primary.Product}} as an \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
2. Open the \{\{FRS\_VARS.Configuration\_console}}.
3. Select **Organizational Profile** > **Languages** > **Add New**.
4. Select **Arabic**. Wait for the application to process and add the language.
5. Click **Ok** in the prompt message after the language is successfully added.

## Enabling Self Service UI V3

<hint type="info">
  The Arabic language support is available only for \{\{FRS\_VARS.Self\_Service\_Mobile\_portal}} V3 UI.
</hint>

1. From the Configuration console, navigate to **Style Editor**.
2. Scroll down till you find the **Self Service** row.
3. Select the checkbox in **Use Self-Service** column.
4. Click the **UI** column and select **UI V3** from the dropdown list.
5. Click **Save**.

<Paragraph indent="1">
  You have now successfully setup the Arabic language for your \{\{FRS\_VARS.Self\_Service\_Mobile\_portal}} V3 UI users.
</Paragraph>
