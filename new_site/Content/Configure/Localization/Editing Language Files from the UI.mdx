You can edit the language files directly from \{\{FRS\_VARS.ProductName}}. Use this method primarily to update stored data.

1. Locate the database of language files.
   1. Log in as an \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}} and navigate to the **Configuration Console**.
   2. From the **Settings** menu, select **Organization Profile** > **Languages**.
   3. The **Languages** workspace displays a table of language files.
2. In the **Edit Language Draft File** column, click **Edit Draft File**.
3. The **Manage Language Files** workspace opens; find the language file you want to modify and click **Edit**.

<hint type="info" indent="2">
  The **Delete** link permanently removes the language file.
</hint>

<Paragraph indent="1">
  The file opens in the **Localization Workbench and Settings** workspace.
</Paragraph>

5. To change a translation:
   1. Use the column headers to filter the data in order to find the label you want to translate in the **Source** column.
   2. Enter the new translation in the **Target** column for that object. The corresponding **Translation Status** property updates to Modified.
   3. Click **Save File** to save the changes without republishing the translations.
   4. Click **Publish Changes** to save and republish the translation immediately.

## Updating Multiple Translation Records

You can simultaneously select and update multiple translation records with the same translation value by filtering those with identical source values.

To update mutiple translation records:

1. From the Configuration console, click **Configure > Organizational Profile > Languages** to display the **Languages** workspace.
2. In the **Edit Language Draft File** column, click **Edit Draft File**.

3. Select the language file from the list, or import a language file from an external source, clic&#x6B;**&#x20;Import Language File**.
4. Click **Edit** to open the language file you want to modify. The application displays the list of translation records.

## Grouping Multiple Translation Records

You can update multiple translation records by grouping them based on the same source value and applying filters.

To group multiple translation records:

1. From the Configuration console, click **Configure > Organizational Profile > Languages** to display the **Languages** workspace.
2. In the **Edit Language Draft File** column, click **Edit Draft File**.
3. Click **Filter Options** drop-down list and select **Group by Duplicates Source** option. The application displays only the translation records that are grouped together based on the same source value.
4. Modify the translation value in th&#x65;**&#x20;Target&#x20;**&#x63;olumn for the object.
5. A confirmation dialog box appears. Click **Yes**.

<Paragraph indent="1">
  The application updates the corresponding Translation records with the modified string value.
</Paragraph>
