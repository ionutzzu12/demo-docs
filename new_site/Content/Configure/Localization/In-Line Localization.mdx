In-line localization allows you to update language files used by the \{\{Primary.Product}} Translation Tool without importing or exporting a translation file, allowing Administrators to make small changes to live data without having to exit the application.

<hint type="info">
  At this time, in-line localization occurs only at the business object level.
</hint>

\{\{FRS\_VARS.ProductName}} uses language files to enable client-side localization. Read more about [Working with Localization](Localization.mdx) to understand how \{\{Primary.Company}} \{\{FRS\_VARS.ProductName}} facilitates localization.

<hint type="info">
  \{\{FRS\_VARS.ProductName}} does not validate any changes you make to the language files.
</hint>

You can use this tool in two different ways:

- [Editing Language Files from the UI](<Editing Language Files from the UI.mdx>) for updating stored data, or
- [Localizing Live Data](<Localizing Live Data.mdx>) for updating live data.
