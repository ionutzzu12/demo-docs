You can add HTML content to a workspace. Follow these steps to translate that HTML content into another language.

Before you begin this procedure, create the HTML content.   See :Link[Adding a Workspace that Displays HTML Content]{label="Adding a Workspace that Displays HTML Content" overridedLabel="Adding a Workspace that Displays HTML Content" spaceId docId="IBjJ7By5-pJ__G5-0odUy" version="v2" docAnchorId="#Adding-a-Workspace-that-Displays-HTML-Content" loadingMethod="dynamic" newTab="false" githubPath="../Attachments/Support Object Workspace as URL.htm#Adding-a-Workspace-that-Displays-HTML-Content" href="../Attachments/Support Object Workspace as URL.mdx"} for complete information about how to do this.

1. Log in to \{\{FRS\_VARS.ProductName}}.
2. Open the **URL Workspace Content** workspace. The application displays a list of HTML content.
3. Open the HTML content to translate.
4. Click the **translation tool** :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/6dvWSPBhAdJFxmQvInMVF_translateblue.png" alt caption size="10" width="17" height="17"} icon. The application displays the **Translate: \[Details]** dialog box.

*Translating HTML Content*

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/WrpZq9iRAGdDDIh8NRVg6_translationtoolhtml.png" position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/translation_tool_HTML.png" githubPath="Content/Resources/Images/help-images/translation_tool_HTML.png" showCaption="false"></Image>

5. Click the tab of the language to translate into.  For example, to translate the HTML content into French, click **French**.
6. Enter the translation of the HTML content.
7. Click **Save**. The application saves the translation of the HTML content.
8. Verify this by doing the following:

<Paragraph indent="1">
  a. Log in to \{\{FRS\_VARS.ProductName}}.
</Paragraph>

<Paragraph indent="1">
  b. Change your locale to the one that you added translated content to.
</Paragraph>

<Paragraph indent="1">
  c. Open the workspace with the HTML content and ensure that it is the translated content that you entered.
</Paragraph>
