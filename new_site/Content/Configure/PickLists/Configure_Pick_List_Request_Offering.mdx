This example shows how to configure a pick list for a request offering.

For a pick list to constrain the value based on another parameter in the request offering, it must have a special value of "\{\{0}}" as a constraint value in the pick list definition.

1. From the Configuration Console, click **Build > Pick Lists** to open the **Pick Lists** workspace.
2. Click a pick list to open it.
3. At the bottom of the window, click the **add** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/9SKkTHEVfabOeACXWkky6_addblue.png" alt caption size="10" width="16" height="12"} to add a line under **Constraints**.
4. Enter the values as illustrated: **AND Department Equal to&#x20;****\{\{0}}**.

*Add Constraint*

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/0XzbwgpX6pAAwTDTNDtPG_constraints.png" position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/constraints.png" githubPath="Content/Resources/Images/help-images/constraints.png" showCaption="false"></Image>

<hint type="info">
  The other parameter that constrains this list (such as department) must be a drop-down list control.
</hint>

5. Click **Save**.
