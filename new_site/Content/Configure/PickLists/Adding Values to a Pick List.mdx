You can add values to a pick list from the business object. You can add validation list values from the **Validation Business Objects** page. See :Link[Adding Validation List Values]{label="Adding Validation List Values" overridedLabel="Adding Validation List Values" spaceId docId="71-6PJQBxTSOBECO85niE" version="v2" docAnchorId="#Add_Validation_List_values" loadingMethod="dynamic" newTab="false" githubPath="Validation_Objects.htm#Add_Validation_List_values" href="Validation_Objects.mdx"}.

1. From the Configuration Console, click **Build > Business Object** to open the **Business Object** workspace.
2. Open the business object containing the pick list to add values for.
3. Click the **Fields** tab.
4. Click the field holding the pick list. The field information appears. See [Working with Fields](../BusinessObjects/Using-Fields.mdx)
5. Next to the Pick List Name, click **values**. The application displays the list of values for this pick list.
6. Click **Add**. The application displays the **New** dialog box.
7. Enter information into the fields. The fields that appear are different for each pick list.
8. (Optional) Translate the fields as needed.
9. Click **Save**.
10. Click **Close**.
