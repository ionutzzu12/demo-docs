The following table contains the default survey business objects.

| Business Object Name        | Description                                                                                                                              |
| --------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
| FRS\_Survey                 | Stores surveys.                                                                                                                          |
| FRS\_SurveyOwnerTeamView    | Contains a list of survey owner teams.                                                                                                   |
| FRS\_SurveyOwnerView        | Contains a list of survey owners.                                                                                                        |
| FRS\_SurveyResults          | Contains a list of survey results.                                                                                                       |
| FRS\_SurveyServiceView      | Shows the survey service type, organizational unit, average score, and total number of surveys.                                          |
| FRS\_Survey Session         | Contains a list of survey sessions.                                                                                                      |
| FRS\_SurveyAnswer           | Contains a list of survey answers.                                                                                                       |
| SurveyMetricView            | Shows survey metrics, such as configuration item owner, closed date and time, numeric value, owner team, organization unit, and service. |
| FRS\_SurveyChoice           | Contains a list of survey response choices.                                                                                              |
| FRS\_SurveyQuestion         | Contains a list of survey questions.                                                                                                     |
| FRS\_SurveyQuestionCategory | Contains a list of survey questions categories.                                                                                          |
