**Role**: Administrators.

**Minimum Version**: \{\{Primary.Company}} \{\{Primary.Product}}*&#x20;2024.3 to access global features via&#x20;*\{\{FRS\_VARS.Neurons\_platform}}*&#x20;features.*

**Feature Eligibility**: *Existing tenants for Classic&#x20;*\{\{FRS\_VARS.ITSM}}*&#x20;and new tenants only for&#x20;*\{\{FRS\_VARS.Neurons\_platform}}*&#x20;features.*

**Feature Prerequisites**: *ConfigDB feature flag for Global Settings EnableGlobalSettings*.

The Configuration Console can be accessed only by users logged in as Administrator. This is where administrators can set up the application, create dashboards, give permissions to user roles, and perform many other configuration tasks. See [Initially Configuring Neurons for ITSM](../CONFIGURE.mdx) for information on how to use the specific items in the Configuration console.

### Accessing the Configuration Console

<hint type="info">
  Only administrators can access the Configuration Console.

<ExpandableHeading>
  ## ITSM via Neurons Platform

  1. Log in using your usual external authentication method. Refer to [Setting up ITSM users for authentication via the Neurons Platform](../../../setting-up-itsm-users-for-authentication-via-the-neurons-platform.mdx).
  2. In the main menu, there are options for **Settings** and **Global Settings**.
     - **Settings:** Select this option to see the Classic \{\{FRS\_VARS.ITSM}} **Settings&#x20;**&#x6D;enu as described below.
     - **Global Settings:** Select this option to see features now hosted in the \{\{FRS\_VARS.Neurons\_platform}} (example: Authentication).
     - When you open any option in the **Global Settings** menu you see a screen from the \{\{FRS\_VARS.Neurons\_platform}} itself. Depending on your role, you may not see the **Global Settings** option in the menu or be able to edit settings in the screen you are viewing.
</ExpandableHeading>

## About the Settings Menu

The **Settings** menu appears on the left side and lists the administrative items by category.

*Settings Menu*

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/YE1v2QDA1SxTIeii-tA39_configuresettingsmenu.png" position="flex-start" size="40" initialPath="../../Resources/Images/Configure/ConfigureSettingsMenu.png" githubPath="Content/Resources/Images/Configure/ConfigureSettingsMenu.png" showCaption="false"></Image>

Navigate in the **Settings** menu as follows:

- Click **Toggle**:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/yH617sZ-ogcVhqFh82lvY_config-close.png" alt caption size="10" width="17" height="16"} to open or close the **Settings** menu.
- Click **Search**:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/MjCUA2oVy8y7EAJiL_Jl1_config-search.png" alt caption size="10" width="20" height="16"} to open the **Search Workspace** window. This lets you open any configuration workspace window.
- Click **Expand All**:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/NI9cfW_C1Rwq9rVLVCg43_config-expand.png" alt caption size="10" width="17" height="16"} to expand all the groups listed in the **Settings** menu and view the links. Groups are indicated by a side arrow :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/FNtOA3yV5W3bUReE8cGRS_config-side-arrow.png" alt caption size="10" width="10" height="13"}.
- Click an item from the list to open a workspace in the center pane. Breadcrumbs appear at the top to remind you where the workspace is located on the list.
- Some workspaces can be opened in a secondary tab. To open a workspace in a new tab, hover over an item and click **Open**:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/0UMNZH9Cf_5Pz3Z6GopSs_config-open.png" alt caption size="10" width="16" height="11"}.

## About the Center Pane

The center pane holds direct links to all administrative items. This allows you to navigate to them with a single click.

Click a link to open a workspace.

## About the Quick Links Menu

The **Quick Links** menu appears to the right of the main window and provides access to some of the most-used items.

Click a link to open a workspace.
