## Viewing a List of Teams

1. Do one of the following:
   - From the Configuration Console, click **Configure > Users and Permissions > Teams** to open the **Teams** workspace to view a list of standard user teams.
   - Log in to the application and open the **Employee Team** workspace.
2. Open a team to view its properties and members.

## Creating a Team

1. From the Configuration Console, click **Configure > Users and Permissions > Teams** to open the **Teams** workspace.
2. Click **New Standard User Team**. The application displays a blank **Standard User Team** workspace.

*Standard User Team Workspace*

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/16RhYuf_1tKi13txtb4qx_userteamcosting.png" alt="A standard user team workspace." position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/user_team_costing.png" githubPath="Content/Resources/Images/help-images/user_team_costing.png" showCaption="false"></Image>

3. Enter information into the fields.

| Field                       | Description                                                                                        |
| --------------------------- | -------------------------------------------------------------------------------------------------- |
| Team                        | A unique name for the team.                                                                        |
| Service Desk                | Adds this team to the Service Desk workflows.                                                      |
| Team Email                  | The name of the email team that relays messages to each team member.                               |
| Team Manager                | The team manager. You can start entering a manager name to automatically search from the database. |
| Team Manager Email          | The email address for the team manager.                                                            |
| Monthly Availability (Hrs)  | How many hours the team is available each month.                                                   |
| Average Cost Per Minute ($) | Average cost per minute.                                                                           |

4. Click **Save**. The team now appears on the team list.

## Modifying a Team

1. From the Configuration Console, click **Configure > Users and Permissions > Teams** to open the **Teams** workspace.
2. Open the team to modify.
3. Edit the team using the options described in :Link[Creating a Team]{label="Creating a Team" overridedLabel="Creating a Team" spaceId docId="vJl4A_IKdZ7b1akgxPONd" version="v2" docAnchorId="#teams_3378263838_1266376" loadingMethod="dynamic" newTab="false" href="Teams.mdx"}.
4. Click **Save**.

## Deleting a Team

You can delete any team, including \{\{FRS\_VARS.ProductName}} default teams.

Deleting a team can cause unexpected results. Delete the team from a workflow before permanently deleting the team here.

1. From the Configuration Console, click **Configure > Users and Permissions > Teams** to open the **Teams** workspace.
2. Highlight the team to delete.
3. Click **Delete**.
4. Click **Continue** at the confirmation message.

## Adding Users to a Team

You can add new users, link users, and unlink users from the **Teams** workspace or under the employee section. Refer to :Link[Creating an Employee]{label="Creating an Employee" overridedLabel="Creating an Employee" spaceId docId="gO65DKWOS0RTWm_hpdz38" version="v2" docAnchorId="#employees_2653137062_1265285" loadingMethod="dynamic" newTab="false" githubPath="Working_with_employees.htm#employees_2653137062_1265285" href="Working_with_employees.mdx"}.

1. From the Configuration Console, click **Configure > Users and Permissions > Teams** to open the **Teams** workspace.
2. Open a team.
3. Click the **Employees** tab.
4. Click **Link**. The application displays a list of users.
5. Select users from the list. Press **Ctrl + click** to select multiple users.
6. Click **Select**. The users appear in the member list.
7. Click **Save**.

## Unlinking Users from a Team

1. From the Configuration Console, click **Configure > Users and Permissions > Teams** to open the **Teams** workspace.
2. Open a team.
3. Click the **Employees** tab.
4. Select a user name from the member list.
5. Click **Unlink**. The user no longer appears on the team list.

## Adding a New User to a Team

1. From the Configuration Console, click **Configure > Users and Permissions > Teams** to open the **Teams** workspace.
2. Open a team.
3. Click the **Employees** tab.
4. Click **New Employee**. The application displays the **New Employee** pages. Refer to :Link[Creating an Employee]{label="Creating an Employee" overridedLabel="Creating an Employee" spaceId docId="gO65DKWOS0RTWm_hpdz38" version="v2" docAnchorId="#employees_2653137062_1265285" loadingMethod="dynamic" newTab="false" githubPath="Working_with_employees.htm#employees_2653137062_1265285" href="Working_with_employees.mdx"} for more information.
