## Viewing the Contact Store List

1. Log into the Service Desk Console.
2. Open the **Contact Store** workspace. The application displays a list of all users in the application.
3. Open a record from the list to see details.

## Showing Contact Details

1. Log into the Service Desk Console.
2. Open the **Contact Store** workspace. The application displays a list of all users in the application.
3. Highlight a user name from the list.  The application displays user details in the bottom panel of the window.
4. For more information about a contact, highlight a user from the list and click **Show Contact Details** from the toolbar.
   - If the contact type is **Employee**, the application opens the **Employee Details** workspace with user details. You can edit employee details from the **Employee** workspace.
