## Enabling the Global Constant

To make the Favorite feature available for the \{\{FRS\_VARS.Self\_Service\_Mobile\_portal}} users, enable the Global Constant:

1. Log in to \{\{FRS\_VARS.ProductName}} as an \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
2. Open the Configuration console > **Global Constants** > set the **SSMFavoritesEnabled** option as **True** > **Save**.

## Setting up Favorite items to be displayed in the Home page

If the items marked as Favorite should be displayed in the \{\{FRS\_VARS.Self\_Service\_Mobile\_portal}} Home page:

1. Log in to \{\{Primary.Product}} Self Service Mobile portal.
2. Change your role to \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
3. Click the Settings icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/CTkrjuf-Q_ub2C3KlUIfW_setting-icon-2.png" alt caption size="10" width="16" height="17"} from the top menu.
4. Click the Settings icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/fuFTRyW0y5G4AEsv7GOJ3_settingsicon.png" alt caption size="10" width="15" height="16"} from the widget you want to edit to change the **Data Source**.
5. Click the drop-down list of the **Data Source** field and select **Favorite**.

<hint type="info" indent="1">
  Only List widget will the Favorite option.
</hint>

6. Click **Save**.

<hint type="info" indent="1">
  - This feature is available only if the Global Constant **SSMFavoritesEnabled** is enabled.
  - At anytime if the Global Constant **SSMFavoritesEnabled** is disabled, and the widgets were configured to display the Favorite data, the list will be blank. You have to change the Data Source of the widget or enable the Global Constant **SSMFavoritesEnabled**.
</hint>
