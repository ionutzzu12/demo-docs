The term *block* refers to specific items on the **Home** page of the Self Service Mobile portal that provide users with information. A *template* contains preset values to speed up the process of making new blocks.

The home page displays the following blocks by default:

- Alert banner
- **My Items** block
- **Need Help?** block
- Popular requests (from the Service Catalog)
- **Announcements** block
- Meet your staff
- **Featured Knowledge Articles** block
- Contact information

## Viewing Block Templates

1. Log in to \{\{Primary.Product}} Self Service Mobile portal.
2. Change your role to \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
3. From the **Home** page click the gear icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/3M1-R4UE-zFYmbQrJA_fX_settings-icon.png" alt caption size="10" width="20" height="17"}.
4. To view a list of block templates, click **Parts**. The application displays a list of block templates.

## Creating a New Block Template

1. Log in to \{\{Primary.Product}} Self Service Mobile portal.
2. Change your role to \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
3. From the **Home** page, click the gear icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/3M1-R4UE-zFYmbQrJA_fX_settings-icon.png" alt caption size="10" width="20" height="17"}.
4. Click **Parts** to view the list of block templates.
5. Click the **Add** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/UoDmK67T2pkOdLMaqxDQk_addtemplatessm.png" alt caption size="10" width="17" height="16"}. The application displays a blank **Template Settings** form with **General** settings.
6. Enter information into the fields:
   - **Title** - The name of the block. The title can have a maximum of 50 characters.
   - **Description Header** - The description of the block header.
   - **Description** - The description of the block.
   - **Category** - The block category.
   - **Widget&#x20;**- The widget type from the options - **List**, **Text**, **Shortcut**, **Banner**, **Placeholder**, **Search**, and **Docked Carousel**.
7. Click **Save**.

## Creating a New Block Template from an Existing Block

1. Log in to \{\{Primary.Product}} Self Service Mobile portal.
2. Change your role to \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
3. From the **Home** page, click the gear icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/3M1-R4UE-zFYmbQrJA_fX_settings-icon.png" alt caption size="10" width="20" height="17"}.
4. Click the **Settings** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/xJW1_6fIiGiCIIUIXbuF0_settingsssm.png" alt caption size="10" width="17" height="16"} in the header of an existing block from which you want to create a new block. The application displays the **Settings** dialog box.

<Paragraph indent="1">
  All the fields are auto-filled with the selected block template's details.
</Paragraph>

5. Make necessary changes and click **Save As Custom Template**.
6. Click **Yes** to confirm. The application displays the new template in the list.

## Editing a Block Template

1. Log in to \{\{Primary.Product}} Self Service Mobile portal.
2. Change your role to \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
3. From the **Home** page, click the gear icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/3M1-R4UE-zFYmbQrJA_fX_settings-icon.png" alt caption size="10" width="20" height="17"}.
4. Click **Parts** to view the list of block templates.
5. Select the template and click the **edit** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/mOv-znjoS_I8ZoCkFHcif_editcategoryssm.png" alt caption size="10" width="16" height="16"}.
6. Make necessary changes and click **Save**.

## Deleting a Block Template

1. Log in to \{\{Primary.Product}} Self Service Mobile portal.
2. Change your role to \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
3. From the **Home** page, click the gear icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/3M1-R4UE-zFYmbQrJA_fX_settings-icon.png" alt caption size="10" width="20" height="17"}.
4. Click **Parts** to view the list of block templates.
5. Select the template and click the **delete** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/OZzjGNCV5YpWreyS_O-P1_deletecategoryssm11x10.png" alt caption size="10" width="11" height="10"}.
6. Click **Yes** to confirm.
