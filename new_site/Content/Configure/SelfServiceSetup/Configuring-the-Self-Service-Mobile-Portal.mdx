## **Home**

You can customize the **Home** page to display screen elements as configured by you.

The **Home** page settings has the following options:

- **Parts** - Opens the **Parts** pane. Use it to configure the **Home** page.

<Paragraph indent="1">
  To add a part to the **Home** page, select a part and then drag and drop it on the **Home**.
</Paragraph>

<Paragraph indent="1">
  Click the **Parts** button to close the pane.
</Paragraph>

<Paragraph indent="1">
  For more information on how to position, customize, and configure a part, and to create a new part template and categories, refer to:
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  [Working with Template Blocks in the Self Service Mobile Portal](Working-with-Template-Blocks-in-SS-Mobile-Portal.mdx)
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  [Customizing Block and Template Parts](Customizing-Block-and-Template-Parts.mdx)
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  [Managing Block Templates with Category Folders](Managing-Block-Templates-with-Category-Folders.mdx)
</Paragraph>

- **Phone** - Displays how the **Home** page renders on a phone. Use it to configure the phone layout settings.

- **Save** - Saves the layout setting changes.
- **Revert** - Reverts to the last saved version of the layout(s).

## **Service Catalog**

You can configure the **Filter Options** options on this page. The following filters are available as defaults, but you can configure new filters for your installation.

**Creating a custom filter**

<Paragraph indent="1">
  a. Click the **Create new search** <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/IIqGPiIz6sfBxbtfVa-SH_add-button-self-service-catalog.png" size="10" isUploading="false" initialPath="../../Resources/Images/buttons/Add-button-Self-Service-Catalog.png" githubPath="Content/Resources/Images/buttons/Add-button-Self-Service-Catalog.png" width="16" height="16" showCaption="false"></Image> icon.
</Paragraph>

<Paragraph indent="1">
  b. Enter details in the **Search Name** field.
</Paragraph>

<Paragraph indent="1">
  c. Select the categories for the new filter.
</Paragraph>

<Paragraph indent="1">
  d. Click **Save**.
</Paragraph>

**Setting a filter as default**

<Paragraph listStyleType="disc" indent="2">
  After creating the filter and selecting the categories, select the **Set filter as default** check box.
</Paragraph>

**Deleting a filter**

<Paragraph indent="1">
  a. Select the filter from the **Filter Options**.
</Paragraph>

<Paragraph indent="1">
  b. Click the **Delete this Search** <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/e8C0RDnLf7lEpSWD_bTjj_delete-button-self-service-catalog.png" size="10" isUploading="false" initialPath="../../Resources/Images/buttons/Delete-button-Self-Service-Catalog.png" githubPath="Content/Resources/Images/buttons/Delete-button-Self-Service-Catalog.png" width="14" height="16" showCaption="false"></Image> icon.
</Paragraph>

<hint type="info">
  Administrators must use a laptop or desktop to change default settings, since these settings cannot be changed from mobile devices.
</hint>

### Loading Service Catalogs in Optimal Time

If your organization is dealing with a large number of Service Catalogs, you can configure the application to load minimal records at a time. As you scroll, the records keeping loading.

This feature is controlled by a Global Constant. From the Configuration Console > **Global Constants**, set the value for **SSM Catalog load more on Scroll** as **True**.

## Additional Settings

From the **Settings** page you can further customize and design the Self Service Mobile portal. The following are the options:

### Personal Settings

Changes the default language and role.

### Application Settings

- **Branding** - Customizes the application header options.
  - **Title** - Select the check box to display the title on the application header. Click the :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/Fe8_f6s_MttP-IJBkaB6u_ssmobiletranslationicon.png" alt caption size="10" width="16" height="16"} icon to change the language.
  - **Logo** - Select the check box to display the logo on the application header.
- **Active Workspaces** - Customize workspaces that should be displayed on the left pane.

<Paragraph indent="1">
  Select or clear the workspace check boxes to display or hide them in the left pane. By default, all workspaces are selected.
</Paragraph>

- **Service Catalog** - Customize Service Catalog options.
  - **Save for Later** - Service Requests are saved as drafts for later use when the check box is selected.
    - **Auto Save** - Enabled when **Save for Later** is selected. Service Requests are automatically saved after the set interval.
  - **Shopping Cart** - Saved Service Requests are added to the **Shopping Cart** when the check box is selected.
  - **Show Request Offering Confirmation Dialog** - To customize the action on clicking a \{\{FRS\_VARS.Service\_Request-Service-Request}}.
    - Select the check box if you want a confirmation dialog to display when you click a \{\{FRS\_VARS.Service\_Request-Service-Request}}.
    - Clear the check box if you want to directly submit the request when you click a \{\{FRS\_VARS.Service\_Request-Service-Request}}.
- **My Items** - Customize the **My Items** workspace options.
  - **Filter** - Select the filter option: **All**, **Approval**, **Issue**, and **Request**.
  - **Sort By** - Select the sort by option: **ID**, **Summary**, **Type**, **Status**, **Created On**, and **Modified On**.
  - **Sort Dir** - Select the sort direction as either **Asc** (ascending) or **Desc** (Descending).
  - **Items per page** - Enter a value or use **+** (plus) or **-** (minus) to change the number of items that appear on the page. The default setting is 20.
