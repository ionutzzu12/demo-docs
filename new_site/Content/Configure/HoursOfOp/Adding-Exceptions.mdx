You can add holidays and other non-working days as exemptions to the hours of operation.

There are two types of exceptions:

- **Recurring** - For repeated exception.
- **Manual** - For one time exception.

## Adding Recurring Exception

There are two types of recurring exceptions:

- **Working time**
- **Non-working time**

The following are the steps to create a recurring exception:

1. From the Configuration console, click **Configure > Organization Profile > Business Hours**.
2. Open an existing schedule.
3. Click **Recurrent** to add a recurrence pattern. The application displays the **Add Exception** page.
4. Enter information into the fields.

| Field               | Description                                                                                                                                   |
| ------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| Name                | A unique name for the exception.                                                                                                              |
| Exception type      | The exception type. You can select one of these:<br />Working time<br />Non-working time                                                      |
| Whole day           | Applies the recurrence to a 24-hour day.                                                                                                      |
| Overnight           | Adds this recurrence as an overnight (after business hours) service.                                                                          |
| Start               | The start and end time of your operation (not applicable if you select **Whole day**).                                                        |
| End                 |                                                                                                                                               |
| Recurrence pattern  | The number of occurrences. Select the pattern options according to the chosen interval. Select the interval (daily, weekly, monthly, yearly). |
| Range of recurrence | The start and end date ranges for this schedule.<br />No end date<br />End after (enter number) occurrences<br />End by (enter date)          |

5. Click **Save**. The application no longer highlights the day and the exception appears at the bottom of the page.

## Adding Manual Exception

1. From the Configuration Console, click **Configure > Organization Profile > Business Hours**.
2. Open an existing schedule.
3. Click **Manual dates** to add a manual exception, such as a half-day.
4. Enter information into the fields.

| Field          | Description                                                                              |
| -------------- | ---------------------------------------------------------------------------------------- |
| Name           | A unique name for the exception.                                                         |
| Exception type | The exception type. You can select one of these:<br />Working time<br />Non-working time |

5. Click the **Add Hours of Operation** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/9SKkTHEVfabOeACXWkky6_addblue.png" alt caption size="10" width="16" height="12"} to add a row, then enter the date information.

| Field      | Description                                                                                  |
| ---------- | -------------------------------------------------------------------------------------------- |
| Date       | Date of the exception.                                                                       |
| Whole day  | Applies the exception to a 24-hour day.                                                      |
| Start Time | The start and end time of your operation (not applicable if you select **Whole day**).<br /> |
| End Time   |                                                                                              |
| Overnight  | Adds this exception as an overnight service (after business hours).                          |
| Name       | Name each row of the exception for future reference (optional).                              |

6. Click the **delete**:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/USPl9Egu4Y2Blx7HKC3mA_deleteredx.png" alt caption size="10" width="14" height="13"} icon to remove a row.
7. Click **Save**.

  On the calendar, the application displays the hours of operation settings with an orange background.
</Paragraph>
