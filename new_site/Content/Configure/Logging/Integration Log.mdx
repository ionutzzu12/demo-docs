The **Integration** log is included by default and lists data integration import jobs. See :Link[Viewing the Data Integration Log - File Integration]{label="Viewing the Data Integration Log - File Integration" overridedLabel="Viewing the Data Integration Log - File Integration" spaceId docId="SZ6PVnM0THPrdssRKn1FH" version="v2" docAnchorId="#integrationLog" loadingMethod="dynamic" newTab="false" githubPath="../ImportExportData/Integration Status and History.htm#integrationLog" href="../ImportExportData/Integration Status and History.mdx"}.

1. From the Configuration Console, click **Monitor > Application Logs > Integration Log** to open the **Integration Log** workspace. The application displays a list of import jobs.
2. If you do not see any recent jobs, click the **refresh** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/85bBZ5gsPcufovXgtjmFo_refresh-old.png" alt caption size="10" width="18" height="14"} from the toolbar.
