The **LDAP Import** log is included by default and lists all imports executed through LDAP.

1. From the Configuration Console, click **Extend > Integration Tools > LDAP Import Log** to view the **LDAP Import Log** workspace.  The application displays a list of import jobs appears.
2. If you do not see any recent jobs, click the **Refresh** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/85bBZ5gsPcufovXgtjmFo_refresh-old.png" alt caption size="10" width="18" height="14"} from the toolbar.

<hint type="info">
  You cannot export the list of imports executed through LDAP.
</hint>
