## Integrate Google Tag Manager with ITSM

Following are the steps to set up the GTM with \{\{FRS\_VARS.ITSM}}:

1. Log in to \{\{FRS\_VARS.ITSM}} as an \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
2. Open the Configuration console > **Organization Profile** > **Account Information**.
3. Select the **Enable google tag manager** check box and enter the **Container ID** created in Google Tag Manager in the **Enter Google Tag Manager Code**.

## Use Cases

Create a tracking metric for the number of clicks on the \{\{FRS\_VARS.Self\_Service\_Mobile\_portal}} portal left menu.

1. Log in to \{\{FRS\_VARS.ITSM}} as an \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
2. Log in to your [Google Tag Manager workspace](https://tagmanager.google.com/), and select **Add a new tag** from the **New Tag** section to [create a Tag](https://support.google.com/tagmanager/answer/14842872).
3. Select **Tag Configuration** > **Google Analytics** > **Google Analytics: GA4 Event**.
4. Add the :Link[Measurement ID]{label="Measurement ID" overridedLabel="Measurement ID" spaceId docId="Uy33imwpFgnGSL0KaxBJ0" version="v2" docAnchorId="#measure-id" loadingMethod="dynamic" newTab="false" href="Setting-up-Google-Analytics-using-Google-Tag-Manager.mdx"} from [Google Analytics](https://support.google.com/analytics/answer/12270356?hl=en).
5. Click the add + icon and select the **Event Name**.
6. Under the **Event Parameter** section, set the **Parameter Name** as link\_text and the **Value** as \{\{Click Text}}.

<hint type="info" indent="1">
  For information on Parameter Name and Values, refer to the [analytics documentation on Google Support page](https://support.google.com/analytics/answer/13675006?hl=en).
</hint>

7. From the **Triggering** section, click the **Add** button and configure when the event should be triggered.
8. Set the **Trigger Type** as **Click - All Elements**.
9. Select **Some Clicks** for **This trigger fires on** option.
10. Conditions - **Click Element**, **matches CSS selector**, gtm-left-nav-menu .x-treelist-item-icon.fontawesome-icon.

<hint type="info" indent="1">
  You can get the CSS element by right-clicking on the application and selecting **Inspect**.
</hint>

11. Save and exit.

<Paragraph indent="1">
  The event, Click - All Elements, is created. Now every time the left menu is clicked, the tag is triggered and the analytics are captured.
</Paragraph>

## Concepts

<ExpandableHeading>
  ## Measurement ID

  **Measurement ID** is a unique identifier for web data stream in Google Analytics. The Measurement ID format is 'G-' followed by a mix of numbers and letters, such as 'G-PSW1MY7HB4.' For more information refer to [\[GA4\] Measurement ID](https://support.google.com/analytics/answer/12270356?hl=en).

<ExpandableHeading>
  ## Container ID

  **Container ID** is an identification code of container created in the Google Tag Manager. The Container ID format is \`GTM-\` followed by a mix of alpha-numeric string, such as \`GTM-T63FA3VC\`. For more information refer to [Install a web container](https://support.google.com/tagmanager/answer/14847097).
