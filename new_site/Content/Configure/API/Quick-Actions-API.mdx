Performs quick actions for a business object. For example, close, clone or resolve an incident or a problem.

For more information on quick actions and viewing a list of quick actions, see :Link[Viewing a List of Quick Actions for a Business Object]{label="Viewing a List of Quick Actions for a Business Object" overridedLabel="Viewing a List of Quick Actions for a Business Object" spaceId docId="L1rMpfFaNyzNJyXZb8B_j" version="v2" docAnchorId="#Viewing-a-List-of-Quick-Actions-for-a-Business-Object" loadingMethod="dynamic" newTab="false" githubPath="../QuickActions/Using_Quick_Actions.htm#Viewing-a-List-of-Quick-Actions-for-a-Business-Object" href="../QuickActions/Using_Quick_Actions.mdx"}.

**Base URL:**

https\://\{tenant url}/api/odata/businessobject/\{business object name}('\{business object unique key}')/\{quick action name}

<hint type="info">
  - Ensure the business object name is suffixed with an "s".
  - You can get the quick action ID from the [Metadata API](Metadata.mdx).
</hint>
