Use a session ID to log out of the tenant.

**Base URL:**

https\://\{tenant url}/api/rest/authentication/logout

Logging out using a session id

Use this information to log out of a tenant using the session ID authorization.

| URL             | https\://\{tenant url}/api/rest/authentication/logout |
| --------------- | ----------------------------------------------------- |
| Method          | DELETE                                                |
| Header Name     | Authorization : Sessionkey                            |
| Request Payload | \{<br />"tenant" :"\{tenant url}"<br />}              |
| Status Code     | 200                                                   |
