<hint type="success">
  Using web script to update the **Attachment** business object are limited to the ATTACHDESC field. To update additional fields, follow these guidelines:

  - **Cloud:** Use the APIs listed on this page.
  - **On-Premise**: A configuration change in the appserver configuration file is required to allow updates to additional fields. Contact your system administrator or Ivanti Support for guidance. Also, enabling this option should only be done in secure environments.
</hint>
