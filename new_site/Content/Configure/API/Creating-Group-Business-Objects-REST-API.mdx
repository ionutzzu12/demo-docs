title: Creating Group Business Objects via REST API
docTags: 
createdAt: Tue Jun 30 2026 17:17:01 GMT+0300 (Eastern European Summer Time)

You can create records for group business objects (for example, Journal#Notes) using the REST OData API. Group names containing a hash (#) must be converted to a URL‑safe format before use. The following rules apply:

1. If the group name contains a hash symbol (#), replace it with a double underscore (\_\_).
   - Example:
     - Original name: Journal#Notes
     - API‑formatted name: Journal\_\_Notes
2. All business object names used in the URL must be pluralized by appending an “s”.
   - Example:
     - Incident → Incidents
     - Change → Changes
     - Journal#Notes → Journal\_\_Notess
3. The final URL/endpoint format is: https\://\{\{tenant-url}}/api/odata/businessobject/\{\{object}}\_\_\{\{group}}s
4. REST Query format: GET https\://\{\{tenant}}/api/odata/businessobject/Journal\_\_Notess

**Base URL**:

https\://\{\{tenant-url}}/api/odata/businessobject/\{\{object}}\_\_\{\{group}}s
