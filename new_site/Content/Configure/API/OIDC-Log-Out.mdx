Use a JWT to log out of the tenant.

**Base URL:**

https\://\{tenant url}/api/rest/authentication/logout

**Logging out using a JWT**

Use this information to log out of a tenant using the OIDC authorization.

| URL             | https\://\{tenant url}/api/rest/authentication/login |
| --------------- | ---------------------------------------------------- |
| Method          | DELETE                                               |
| Header Name     | Authorization Bearer: JWT Token                      |
| Request Payload | \{<br />"tenant" :"\{tenant url}"<br />}             |
| Status Code     | 200                                                  |
