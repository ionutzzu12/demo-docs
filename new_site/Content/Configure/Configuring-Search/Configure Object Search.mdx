You can set the primary display fields and additional detail fields for related items (refer to [Using Related Items and Object Matching](<../../ServiceDesk/Search/Related Items.mdx>)) searches.

When you are in the Service Desk Console and open the business object, if you enter a term in the **Search** text box, the application displays certain information depending on the settings below.

*Search Parameters*

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/4I5jL568zVIcf6lsDvoBP_search-new.png" alt="Image of search parameters." position="flex-start" size="40" initialPath="../../Resources/Images/help-images/search-new.png" githubPath="Content/Resources/Images/help-images/search-new.png" showCaption="false"></Image>

Follow these steps:

1. From the Configuration Console, click **Build > Business Objects** to open the **Business Object** workspace.
2. Open a business object.
3. Click the **Search** tab. The application displays the **Search Settings** page.
4. Enter information into the fields.

| Field                    | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| ------------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Primary Display Fields   | Automatically shows the name of the primary display field based on the parent business object.<br /><br />To change it, hover over the field, then click **Change Field**. Select a new field from the drop-down list.<br /><Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/ju6C9ae2cuINSVKaDgwfZ_object-search-change-field.jpg" alt="Changing the primary display fields." position="flex-start" size="40" initialPath="../../Resources/Images/help-images/object search change field.jpg" githubPath="Content/Resources/Images/help-images/object search change field.jpg" showCaption="false"></Image> |
| Additional Detail Fields | Contains additional detail fields to help you select the correct record.<br /><br />To add more detail fields, do the following:<br />Click **Add Field**.<br />Select a field from the drop-down list.<br />Repeat as needed.<br /><br />To remove a field, hover over the field and click **Delete**.                                                                                                                                                                                                                                                                                                                                        |
| Advanced Search Settings | Determines which fields appear when you click **Advanced Search**.<br /><br />To add more detail fields, do the following:<br />Click **Add Field**.<br />Select a field from the drop-down list.<br />Repeat as needed.                                                                                                                                                                                                                                                                                                                                                                                                                       |

5. Click **Save**.
