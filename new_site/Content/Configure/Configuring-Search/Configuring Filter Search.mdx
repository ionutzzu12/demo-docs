You can create and configure filter panels to improve the way the application searches in \{\{FRS\_VARS.ProductName}}.

In the Service Desk Console, you can use a filter panel to add additional search options.

*Search Filter Panel*

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/XPLDccIm9fKW2ZfZiin1l_filter-search-panel.png" alt="Image of the search filter panel." position="flex-start" size="40" initialPath="../../Resources/Images/help-images/filter search panel.png" githubPath="Content/Resources/Images/help-images/filter search panel.png" showCaption="false"></Image>

You can define filter panels for business objects, including custom business objects. (Custom business objects are business objects that users can customize for their own individual business requirements.)

Each filter panel is made up of one or more filter groups. The figure below shows the filter panel for the **Change** business object. If you click **more...** at the bottom of a filter group, the application displays the **More Options** dialog box.

Each filter group is made up of one or more individual filters.

## Adding a Filter Panel

By default, \{\{FRS\_VARS.ProductName}} provides filter panels for several key business objects. To add a filter panel for another business object, follow these steps:

1. From the Configuration Console, click **Build > Search Tools > Filter Searches** to display the **Filter Searches** workspace.
2. Click **Add**.
3. Select a business object from the drop-down list.
4. Click **Save**.

### Adding a Filter Group to a Filter Panel

Follow these steps to add a new filter group to a filter panel:

1. Perform steps 1 and 2 in :Link[Editing a Filter Panel]{label="Editing a Filter Panel" overridedLabel="Editing a Filter Panel" spaceId docId="AcRY6-q_oBDbQ3IT2QzmZ" version="v2" docAnchorId="#Using" loadingMethod="dynamic" newTab="false" href="Configuring Filter Search.mdx"}.
2. Click **Add Filter**.
3. In the **Condition Name** field, enter the name for the filter group. This is the header that the application displays in the Service Desk Console.
4. In the **Condition Help** field, enter information that the application displays as a tooltip.
5. Click **Save**.
6. Click **Preview** on the toolbar to see what the filter panel looks like.

*Filter Preview Workspace*

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/5-Ll2wkHqtf5Fhw2bhGze_newfilterpreviewpanel.png" alt="Image of the Filter Preview workspace." position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/NEW_Filter_Preview_Panel.png" githubPath="Content/Resources/Images/help-images/NEW_Filter_Preview_Panel.png" showCaption="false"></Image>

7. To translate the values in the filter panel, click **Translation Tool**. Refer to [Using the Ivanti Translation Tool](<../Localization/Using the Translation Tool.mdx>) for more information about using the Ivanti Translation Tool.
8. Click **Save**.

### Enabling and Disabling Filter Groups in a Filter Panel

Before you begin, perform steps 1 and 2 in :Link[Editing a Filter Panel]{label="Editing a Filter Panel" overridedLabel="Editing a Filter Panel" spaceId docId="AcRY6-q_oBDbQ3IT2QzmZ" version="v2" docAnchorId="#Using" loadingMethod="dynamic" newTab="false" href="Configuring Filter Search.mdx"}.

If a filter group is enabled, the application displays **ON** next to its name. 

1. To disable the filter group, click **ON** to toggle it to **OFF**.

*Disabled Filter Group*

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/4T2yzdCbJlpKmDLA4Usoo_status-off.PNG" alt="Image of the Disabled Filter group." position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/Status OFF.PNG" githubPath="Content/Resources/Images/help-images/Status OFF.PNG" showCaption="false"></Image>

2. To enable the filter group, click **OFF** to toggle it to **ON**.

### Editing the Title of a Filter Group

Follow these steps to edit the title of a filter group:

1. Perform steps 1 and 2 in :Link[Editing a Filter Panel]{label="Editing a Filter Panel" overridedLabel="Editing a Filter Panel" spaceId docId="AcRY6-q_oBDbQ3IT2QzmZ" version="v2" docAnchorId="#Using" loadingMethod="dynamic" newTab="false" href="Configuring Filter Search.mdx"}.
2. To edit the header title, click on it. The application displays the **Condition Filter** dialog box.
3. Enter a new name in the **Condition Name** field.
4. (Optional) In the **Condition Help** field, enter information that displays as a tool tip.
5. Click **Save**.
6. Do the following to verify the new header title:

<Paragraph indent="1">
  a. Go to the Service Desk Console.
</Paragraph>

<Paragraph indent="1">
  b. Open the selected business object.
</Paragraph>

<Paragraph indent="1">
  c. Click **Show Filters**.
</Paragraph>

<Paragraph indent="1">
  d. Verify that the new header title displays.
</Paragraph>

<Paragraph indent="1">
  e. Hover over the header title to verify that the tool tip displays correctly.
</Paragraph>

### Adding an Individual Filter to a Filter Group

Follow these steps to add an individual filter to a filter group.

1. Perform steps 1 and 2 in :Link[Editing a Filter Panel]{label="Editing a Filter Panel" overridedLabel="Editing a Filter Panel" spaceId docId="AcRY6-q_oBDbQ3IT2QzmZ" version="v2" docAnchorId="#Using" loadingMethod="dynamic" newTab="false" href="Configuring Filter Search.mdx"}.
2. Click either **Add filter criteria** or **Add another filter criteria**. The application displays the **Filter Criteria** dialog box.
3. Enter the name of the individual filter in the **Label** field.
4. Select the type of control to use from the drop-down list in the **Control** field. Choose from None, Calendar, Checkbox, Pick List, or Textbox.
5. Add one or more expressions to your individual filter. The control type you chose in step 4 determines the content displayed in the **Expression** area. For example, if you selected **Calendar**, the application displays the following choices:

*Filter Criteria*

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/7G7wIcdFEP5rRdJ6yLsR3_brandnewfiltercriteria.png" alt="Image of filter criteria." position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/Brand_New_Filter_Criteria.png" githubPath="Content/Resources/Images/help-images/Brand_New_Filter_Criteria.png" showCaption="false"></Image>

<Paragraph indent="1">
  An individual filter can be a single search condition or a group of multiple search conditions. For example, an incident that is considered to be "open" can actually contain a number of different status settings such as logged and resolved. However, you can configure one individual filter to contain multiple parameters.  See the figure below.
</Paragraph>

*Multiple Conditions*

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/gol1LJYlzX-GxAUNEkfnp_openshowfiltersfromsundar.png" alt="An individual filter can have multiple conditions." position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/Open_Show_Filters_From_Sundar.png" githubPath="Content/Resources/Images/help-images/Open_Show_Filters_From_Sundar.png" showCaption="false"></Image>

<Paragraph indent="1">
  We recommend that you use a unique name for an individual filter if it contains multiple conditions.
</Paragraph>

6. To display an individual filter in a filter group in a filter panel, check **Show by default**. If you do not check **Show by default**, you must click **more....&#x20;**&#x69;n the filter panel to display this individual filter.
7. Click **Save**.

### Editing an Individual Filter in a Filter Group

1. Perform steps 1 and 2 in :Link[Editing a Filter Panel]{label="Editing a Filter Panel" overridedLabel="Editing a Filter Panel" spaceId docId="AcRY6-q_oBDbQ3IT2QzmZ" version="v2" docAnchorId="#Using" loadingMethod="dynamic" newTab="false" href="Configuring Filter Search.mdx"}.
2. To edit the name or type of an individual filter, click **Edit** on the right-hand side.
3. To remove an individual filter, click **Remove** on the right-hand side.

### Removing a Filter Group from a Filter Panel

Follow these steps to remove a filter group from a filter panel:

1. Perform steps 1 and 2 in :Link[Editing a Filter Panel]{label="Editing a Filter Panel" overridedLabel="Editing a Filter Panel" spaceId docId="AcRY6-q_oBDbQ3IT2QzmZ" version="v2" docAnchorId="#Using" loadingMethod="dynamic" newTab="false" href="Configuring Filter Search.mdx"}.
2. Click the **Remove filter** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/USPl9Egu4Y2Blx7HKC3mA_deleteredx.png" alt caption size="10" width="14" height="13"} to remove the filter group and all of the associated individual filters.
3. Click **Yes** in the confirmation dialog box.
4. Click **Save**.
