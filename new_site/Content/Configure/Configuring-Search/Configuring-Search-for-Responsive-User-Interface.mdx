The search functionality in the responsive user interface is configurable. You can configure the business objects that should be included when performing a search. By default, the application searches for keywords in the FAQ and Announcement business objects, but you can configure the search results to include the Request Offerings and My Items business objects as well.

Use the following steps to configure the search feature for the responsive user interface:

1. From the Configuration Console, select **Users and Permissions** > **Roles and Permissions**.

<Paragraph indent="1">
  A list of available roles is displayed.
</Paragraph>

2. Select either the Admin or Self Service Mobile role to configure the search settings. In this scenario, the Admin role is configured.

<Paragraph indent="1">
  The **Role Details** page is displayed.
</Paragraph>

<hint type="info">
  Only the Admin or Self Service Mobile roles have access to the responsive user interface.
</hint>

3. Select **Top Level Tabs** > **Search**.
4. In the **Self Service Search's Configuration** section, you can see **FAQ** and **Announcements** are configured by default.

<Paragraph indent="1">
  *Top-level Tabs Search - Default Configuration*
</Paragraph>

5. Configure search settings for the Request Offering:

<Paragraph indent="1">
  a. Click the <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/qZk3vL6CcYm1sU0AkWumk_addbuttoninsearchconfig.png" size="10" isUploading="false" initialPath="../../Resources/Images/buttons/AddButtonInSearchConfig.png" githubPath="Content/Resources/Images/buttons/AddButtonInSearchConfig.png" width="16" height="16" showCaption="false"></Image> icon. A new line is added.
</Paragraph>

<Paragraph indent="1">
  b. Click the **Object Name** field's drop-down arrow and select **Service Request Template**.
</Paragraph>

<Paragraph indent="1">
  c. Click the **Edit Preview** link; the **Preview Item Format for: Service Request Template** dialog box is displayed.
</Paragraph>

<Paragraph indent="1">
  d. Configure as given in the table below:
</Paragraph>

| Field Name                             | **Settings**                                                         |
| -------------------------------------- | -------------------------------------------------------------------- |
| Title Field                            | Set as **Description**.                                              |
| Preview Text Field                     | Set as **Name**.                                                     |
| Number of Preview Lines to show:       | Set the number of lines that the record details should be displayed. |
| Highlightmatching text in the preview: | Select the checkbox.                                                 |
| Form for Hover Panel:                  | ServiceReqTemplate                                                   |

<Paragraph indent="2">
  *Preview Item Format for: Service Request Template Dialog Box*
</Paragraph>

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/DyFdpU2aDikO79T2BWesW_previewservicerequesttemplatedialogbox.png" alt="Preview item format for the Service Request Template dialog box." position="flex-start" size="40" indent="2" initialPath="../../Resources/Images/Configure/PreviewServiceRequestTemplateDialogBox.png" githubPath="Content/Resources/Images/Configure/PreviewServiceRequestTemplateDialogBox.png" showCaption="false"></Image>

<Paragraph indent="1">
  e. Click **Save**.
</Paragraph>

6. Configure search settings for My Items:

<Paragraph indent="1">
  a. Click the <Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/qZk3vL6CcYm1sU0AkWumk_addbuttoninsearchconfig.png" size="10" isUploading="false" initialPath="../../Resources/Images/buttons/AddButtonInSearchConfig.png" githubPath="Content/Resources/Images/buttons/AddButtonInSearchConfig.png" width="16" height="16" showCaption="false"></Image> icon. A new line is added.
</Paragraph>

<Paragraph indent="1">
  b. Click the **Object Name** field's drop-down arrow and select **My Item**.
</Paragraph>

<Paragraph indent="1">
  c. Click the **Edit Preview** link; the **Preview Item Format for: My Item** dialog box is displayed.
</Paragraph>

<Paragraph indent="1">
  d. Configure as given in the table below:
</Paragraph>

| Field Name                             | **Settings**                                                         |
| -------------------------------------- | -------------------------------------------------------------------- |
| Title Field                            | Set as **Display Text**.                                             |
| Preview Text Field                     | Set as **Display Text**.                                             |
| Number of Preview Lines to show:       | Set the number of lines that the record details should be displayed. |
| Highlightmatching text in the preview: | Select the checkbox.                                                 |
| Form for Hover Panel:                  | Set as **FRS\_MyItems.Brief**.                                       |

<Paragraph indent="2">
  *Preview Item Format for: My Item Dialog Box*
</Paragraph>

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/D7Fy-mT7h_29X22vT9UO__previewmyitemtemplatedialogbox.png" alt="Preview item format for the My Item dialog box." position="flex-start" size="40" indent="2" initialPath="../../Resources/Images/Configure/PreviewMyItemTemplateDialogBox.png" githubPath="Content/Resources/Images/Configure/PreviewMyItemTemplateDialogBox.png" showCaption="false"></Image>

Now, when you perform a search in the responsive user interface **Home** page, the application returns results for FAQs, Announcements, Request Offerings, and My Items business objects.
