When you work with layouts in a language other than English, you might observe a word or phrase that is not properly translated. Do the following to add or correct a translated word or phrase:

1. From the Configuration console, click **Build > Business Objects**.
2. Select a business object > **Layouts** tab > open a layout.
3. Click the **translation tool** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/6dvWSPBhAdJFxmQvInMVF_translateblue.png" alt caption size="10" width="17" height="17"} and choose the locale language from the drop-down menu. The application displays the **Translation Tool** workspace and it is preset for the layout that you are working on.
4. Optional. To view words and phrases from all layouts, choose **All** in the **Layouts** drop-down menu.
5. Click **Search** to view the words and phrases from the current layout. The application displays the search results.
   - The first column shows the property, that is, the specific place, where a word or phrase is used in the current form.
   - The third column shows the word or phrase as it appears right now for the language that you selected.
6. In the third column, click a translated string to open it for editing.
7. Do one of the following:
   - Select a string from the drop-down list, if the application lists any suggestions.
   - Enter new text.
8. Press **Enter** to record (but not save) the change. The application displays a red triangle next to the unsaved text string. This marks the changed string for reference.
9. Change additional strings as necessary.
10. Do one of the following:
    - To save your changes, click **Save**.
    - To remove all of the changes that made, click **Revert**. The application returns the strings to their original settings.
11. Optional. Import the translation strings to a spreadsheet.
