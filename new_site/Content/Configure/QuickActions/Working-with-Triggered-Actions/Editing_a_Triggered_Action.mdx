docTags: 
createdAt: Tue Jun 30 2026 17:16:57 GMT+0300 (Eastern European Summer Time)

To edit a triggered action:

1. In the Graphical Action Designer (GAD), select the **Saved Actions** tab.
2. Select **Triggered Actions&#x20;**&#x66;rom the category, and click the arrow icon to display the list of triggered actions.
3. Select the Triggered action that you want to edit. Modify the **Block&#x20;**&#x74;ab or **Common** tab based on your requirements.

<hint type="info">
  When you select the **Triggered&#x20;**&#x6F;ption, the following actions are removed from the Action Designer list: Go to Record, Run for Collection, Go to Action, Decision Action, and Show a Popup. These actions cannot be used with triggers.
</hint>

4. Click **Save**.

<Paragraph indent="1">
  A confirmation message appears that the triggered action is updated successfully.
</Paragraph>
