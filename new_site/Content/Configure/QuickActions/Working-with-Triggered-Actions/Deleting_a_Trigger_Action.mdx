To delete a triggered action:

1. In the Graphical Action Designer (GAD), select the **Saved Actions** tab.
2. Select **Triggered Actions&#x20;**&#x66;rom the category, and click the arrow icon to display the list of Triggered actions.
3. Select the Triggered action that you want to delete, and click :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/dAn_YD-HPMMUCHmqWSmqk_delete-button-self-service-catalog.png" alt caption size="10" width="16" height="18"}. A confirmation dialog box appears prompting you to confirm the action.
4. Click **Yes**. The triggered action is deleted.
