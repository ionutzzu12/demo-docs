### Action Fields

Each action template has different fields, but they all have the following:

| Field                    | Description                                                                                                                                                                                  |
| ------------------------ | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Enter action name        | A unique name for this action.<br /><br />**NOTE**: Actions with duplicate names produce unexpected results. The application only runs the first action of the same name that it encounters. |
| Enter action description | Contains a description for this action.                                                                                                                                                      |
| Enter category           | Activates the drop-down menu that contains the action categories. Saves the action to this folder.                                                                                           |

### Action Settings Tab

| Field                 | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| --------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Apply Action To<br /> | (Only shown for the **Send Email** action) Determines the object or child to which the action applies. See [Example: Using the Send Email Action (CAD)](<Examples of Using Actions (CAD)/Using Send Email QA.mdx>).                                                                                                                                                                                                                                            |
| Plain Text            | (Only shown for the **Send Email** action) Sends the email in plain text (as opposed to HTML or formatted text).                                                                                                                                                                                                                                                                                                                                               |
| Publish to            | Activates the drop-down menu that contains the roles to which to publish this action. The application saves the action to the roles you select, as well as to the administrator and action owner roles.                                                                                                                                                                                                                                                        |
| UI Options            | Determines which of the following options users have:<br />**Show as toolbar button**: Creates a button on the toolbar that performs this action.<br />**Hide in UI**: Removes this action from the Action Menu.                                                                                                                                                                                                                                               |
| Auto-save             | When selected, all actions carried out on records, individually, or as part of a composite action, are saved automatically. When this option is not selected, changes are not automatically saved, and you are asked to click **Save**.<br />If the following action(s) are added to a Composite Action, auto-save is automatically switched on and cannot be switched off: Go to Record, Run for Collection, Decision Action, Show a PopUp, and Go to Action. |
| Hide in UI when       | Hide this action under certain circumstances.  Enter the appropriate expression.                                                                                                                                                                                                                                                                                                                                                                               |
| Disable in UI when    | Disables this action under certain circumstances.  Enter the appropriate expression.                                                                                                                                                                                                                                                                                                                                                                           |
| Confirm before run    | Prompts you to confirm before running the action.                                                                                                                                                                                                                                                                                                                                                                                                              |
| Confirmation prompt   | Contains the confirmation message.                                                                                                                                                                                                                                                                                                                                                                                                                             |
