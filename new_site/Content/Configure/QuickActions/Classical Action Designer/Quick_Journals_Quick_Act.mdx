| Action Menu Option | Action Taken                                                                        |
| ------------------ | ----------------------------------------------------------------------------------- |
| Create Memo        | Creates a new notes journal entry.                                                  |
| Email Customer     | Creates an email customer journal entry.                                            |
| Left Voice Mail    | Creates a left voicemail journal entry, with "Left Voicemail" in the summary field. |
