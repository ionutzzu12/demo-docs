<hint type="info">
  This \{\{FRS\_VARS.Quick\_Action}} has been deprecated in \{\{FRS\_VARS.ITSM}} 2024.3. New \{\{FRS\_VARS.Quick\_Actions}} cannot be created from deprecated action types. Existing \{\{FRS\_VARS.Quick\_Actions}} can still be edited, but cause warning messages to appear. We recommend customers discontinue use of this \{\{FRS\_VARS.Quick\_Action}} now, ahead of its removal in the future release.
</hint>

As an alternative, use a combination of the [Insert Object Action](<Insert Object.mdx>) and the [Search and Link Action](<Search and Link.mdx>) to replicate the functionality of this \{\{FRS\_VARS.Quick\_Action}}.
