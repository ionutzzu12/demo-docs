## About the Insert Object Action

The **Insert Object** action inserts a business object into a form or list, such as a new incident, field value, or a note.

Use this action to add levels of granularity, constructing additional expressions restricting the viewing and usage of the action to certain field states.

### Parameters

<hint type="info">
  Some parameter field names may differ slightly in the Graphical Action Designer UI.
</hint>

| Field                    | Description                                                                                                    |
| ------------------------ | -------------------------------------------------------------------------------------------------------------- |
| Enter action name        | A unique name for the action.                                                                                  |
| Enter action description | Optional. A description of what the action does.                                                               |
| Enter category           | Optional. The category. Select from the drop-down list.                                                        |
| Business Object          | The business object to insert. Select from the drop-down list. The fields that appear are based on this value. |
| Field Name               | The name of the field.                                                                                         |
| Field Value              | The value of the field.                                                                                        |
