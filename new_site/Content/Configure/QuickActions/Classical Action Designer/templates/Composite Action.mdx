## About the Composite Action

The **Composite Action** action assembles complex actions based on other actions.

### Parameters

| Parameter                | Description                                                                               |
| ------------------------ | ----------------------------------------------------------------------------------------- |
| Enter action name        | A unique name for the action.                                                             |
| Enter action description | Optional. A description of what the action does.                                          |
| Enter category           | Optional. The category. Select from the drop-down list.                                   |
| Add new action           | Opens the drop-down list and displays the actions that can be used in a composite action. |
