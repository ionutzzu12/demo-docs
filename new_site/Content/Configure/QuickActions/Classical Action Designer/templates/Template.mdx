## About the Template Action

The **Template** action displays a template where you can create a business object with prepopulated field values and child business objects.

### Parameters

| Parameter                | Description                                                                                                                                                                 |
| ------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Enter action name        | A unique name for the action.                                                                                                                                               |
| Enter action description | Optional. A description of what the action does.                                                                                                                            |
| Enter category           | Optional. The category. Select from the drop-down list.                                                                                                                     |
| Field Name               | The field name.                                                                                                                                                             |
| Field Value              | The field value.                                                                                                                                                            |
| Additional actions       |                                                                                                                                                                             |
| Add new action           | Creates a composite action to run with the business object.                                                                                                                 |
| Open                     | Configures the action that you selected. The additional fields that appear depend on the action that you select. Fill in these fields as necessary to configure the action. |
| Edit Label               | Renames the action that you added.                                                                                                                                          |
| Remove from List         | Removes the action from the list.                                                                                                                                           |
