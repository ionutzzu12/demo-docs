## About the Update Object Action

The **Update Object** action updates an existing business object. Use this action to optionally add levels of granularity, constructing additional expressions restricting the viewing and usage of the action to certain field states.

### Parameters

| Parameter                | Description                                             |
| ------------------------ | ------------------------------------------------------- |
| Enter action name        | A unique name for the action.                           |
| Enter action description | Optional. A description of what the action does.        |
| Enter category           | Optional. The category. Select from the drop-down list. |
| Field Name               | The field name.                                         |
| Field Value              | The field value.                                        |
