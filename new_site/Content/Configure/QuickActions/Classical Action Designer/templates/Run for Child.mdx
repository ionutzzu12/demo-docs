## About the Run for Child Action

The **Run for Child** action applies the action to a child business object of the current business object. For example, you can run an update action and then export data to Microsoft Excel.  

### Parameters

| Field                    | Parameter                                                                                                                                                                            |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Enter action name        | A unique name for the action.                                                                                                                                                        |
| Enter action description | Optional. Enter a plain-text description of what the action does.                                                                                                                    |
| Enter category           | Optional. The category. Select from the drop-down list.                                                                                                                              |
| Relationship             | The relationship between the current business object and a child object.<br />The action is applied to the child object in the relationship that you select from the drop-down list. |
| Child Object Filter      | Optional. Opens the Expression Editor. Enables you to constrain the child object being evaluated.                                                                                    |
| Action to run            | The action to run for the child business object. Choose from the drop-down list.                                                                                                     |
