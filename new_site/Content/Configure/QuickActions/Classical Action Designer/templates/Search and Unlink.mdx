## About the Search and Unlink Action

The **Search and Unlink** action searches for and unlinks a business object record from another record. For example, you can unlink a configuration item record from an incident.

*Search and Unlink Action*

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/pEgF7DrSF44Ma3z8P8XVW_search-and-unlink.png" position="flex-start" size="40" indent="1" initialPath="../../../../Resources/Images/help-images/search and unlink.png" githubPath="Content/Resources/Images/help-images/search and unlink.png" showCaption="false"></Image>

### Parameters

| Parameter                | Description                                                                                              |
| ------------------------ | -------------------------------------------------------------------------------------------------------- |
| Enter action name        | A unique name for the action.                                                                            |
| Enter action description | Optional. A description of what the action does.                                                         |
| Enter category           | Optional. The category. Select from the drop-down list.                                                  |
| Relationship             | The relationship between the current business object and the business object to which you are unlinking. |
| Find where               | Specifies where to search.                                                                               |
| Operator                 | A comparison operator.                                                                                   |
| String                   | A string to use for the comparison.                                                                      |
