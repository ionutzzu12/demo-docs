## About the Run Report Action

The **Run Report** action runs a specified report that has been created to support actions.

### Parameters

| Parameter                    | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| ---------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Enter action name            | A unique name for the action.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| Enter action description     | Optional. A description of what the action does.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                             |
| Enter category               | Optional. The category. Select from the drop-down list.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      |
| Report Name                  | The report to run. Select from the drop-down list.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| Can run for multiple records | Runs the report for multiple records.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| Report Parameters            | The parameters automatically appear based on the report name chosen. The values are populated by reports checked as **Can Run from Run Report Action**. See :Link[Step 3: Creating a Report]{label="Step 3: Creating a Report" overridedLabel="Step 3: Creating a Report" spaceId docId="lqyFeWH0iCqzwXKTAJ5TX" version="v2" docAnchorId="#Step3" loadingMethod="dynamic" newTab="false" githubPath="../../../../ServiceDesk/Reports/Creating_Custom_Reports.htm#Step3" href="../../../../ServiceDesk/Reports/Creating_Custom_Reports.mdx"}. |

### Actions

| Button | Action                 |
| ------ | ---------------------- |
| Save   | Commits your settings. |
| Cancel | Exits without saving.  |

<hint type="info">
  This action does not respect the **Auto-save** checkbox option in the **Action Settings** tab. It does save changes directly to the database, but please ensure that the behaviour you expect still occurs.
</hint>
