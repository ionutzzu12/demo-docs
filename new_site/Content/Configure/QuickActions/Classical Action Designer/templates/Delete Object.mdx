## About the Delete Object Action

The **Delete Object** action deletes the current context business object. Use this action within a search.

Use extreme caution if using this in a composite action as it could potentially delete the wrong value.

### Parameters

| Parameter                | Description                                             |
| ------------------------ | ------------------------------------------------------- |
| Enter action name        | A unique name for the action.                           |
| Enter action description | Optional. A description of what the action does.        |
| Enter category           | Optional. The category. Select from the drop-down list. |

### Actions

| Button | Action                 |
| ------ | ---------------------- |
| Save   | Commits your settings. |
| Cancel | Exits without saving.  |
