## About the Insert Child Object Action

The **Insert Child Object** action inserts a child business object into a form or list, such as a new incident, field value, or a note.

<hint type="info">
  This action is listed as **Create a Child Record** in the Graphical Action Designer.
</hint>

Use this action to add levels of granularity, constructing additional expressions restricting the viewing and usage of the action to certain field states.

### Parameters

<hint type="info">
  Some parameter field names may differ slightly in the Graphical Action Designer UI.
</hint>

| Field                    | Description                                                                                                                                                                                                                                                                                                                                 |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Enter action name        | A unique name for the action.                                                                                                                                                                                                                                                                                                               |
| Enter action description | Optional. A description of what the action does.                                                                                                                                                                                                                                                                                            |
| Enter category           | Optional. The category. Select from the drop-down list.                                                                                                                                                                                                                                                                                     |
| Child Object             | The child business object to insert. Select from the drop-down list. The fields that appear are based on this value.                                                                                                                                                                                                                        |
| Field Name               | The name of the field.                                                                                                                                                                                                                                                                                                                      |
| Field Value              | The value of the field.                                                                                                                                                                                                                                                                                                                     |
| Web UI parameters        | Generally for internal use only.<br /><br />Only used to post the object to the Social Board. This field takes only one expression:<br />**\{socialshare\:true, elementId: $(elementId)}**.<br /><br />For example, in the **Incident** business object, this field is used in the **Post Social message associated with Incident** action. |
