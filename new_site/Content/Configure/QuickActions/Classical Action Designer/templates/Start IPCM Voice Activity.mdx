## About the Start Ivanti Voice Activity Action

The **Start&#x20;**\{\{FRS\_VARS.Company}}**&#xA0;Voice Activity** action configures and calls a web service for the \{\{FRS\_VARS.Company}} Voice application. Before configuring this action, you must set up the voice integration to \{\{FRS\_VARS.ProductName}}. See [Configuring Ivanti Voice Integration](../../../Voice/Configure_Voice_Integrat.mdx).

When the action executes, the \{\{FRS\_VARS.ProductName}} server calls the \{\{FRS\_VARS.Company}} Voice web service, which in turn launches the voice application chosen, and sets any parameters specified as properties available to the voice application.

When a voice application is started via the web service, there is no active phone call; it is simply an interaction in \{\{FRS\_VARS.Company}} Voice terms. If the action initiates a phone call, you can use the **Make Call** block to initiate the call.

If the \{\{FRS\_VARS.Company}} Voice web service is not configured correctly, \{\{FRS\_VARS.ProductName}} cannot retrieve the list of available voice applications and displays an error message.

### Parameters

| Parameter                | Description                                             |
| ------------------------ | ------------------------------------------------------- |
| Enter action name        | A unique name for the action.                           |
| Enter action description | Optional. A description of what the action does.        |
| Enter category           | Optional. The category. Select from the drop-down list. |
| Voice Application Name   | The voice application to start.                         |
| Property Name            | The property name to pass to the voice application.     |
| Property Value           | The property value to pass to the voice application.    |
