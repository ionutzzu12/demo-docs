## About the Clone Object Action

The **Clone Object** action creates a copy of the selected record and overwrites the specified fields, including copying from one-to-one relationships.

### Parameters

| Parameter                    | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                 |
| ---------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Enter action name**        | Enter a unique name for the action.                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| **Enter action description** | Optional. Add a description.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| **Enter category**           | Optional. Select an action category from the drop-down list.                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| **Field Name**               | Choose the name of the field you want to update when you clone the record. Click :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/JtIxGMfjslfCSitXWBu4o_addgreen.png" alt="add" caption size="10" width="17" height="15"} to add multiple fields.<br />When updating a field that uses counters (such as **IncidentNumber**), enable the **Unique** property (see [Creating a Field](../../../BusinessObjects/Creating-a-Field.mdx)) to avoid duplicate records. |
| **Field Value**              | Enter the desired replacement value for the **Field Name**. Use the **Expression Editor** to enter an expression for the value. See [Using the Expression Editor](<../../../../Reference/Expressions/Using Expression Editor.mdx>) for more information.<br />To avoid copying values for any field (such as **ClosingEscLink**), leave this field empty.                                                                                                                                                   |
| **Relationship Name**        | Use this parameter to indicate the objects to associate and clone with the record. Click :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/JtIxGMfjslfCSitXWBu4o_addgreen.png" alt="add" caption size="10" width="17" height="15"} to add multiple related objects.                                                                                                                                                                                               |
| **Publish to**               | Use this parameter to publish the \{\{FRS\_VARS.Quick\_Action}} to different Roles.                                                                                                                                                                                                                                                                                                                                                                                                                         |
