## About the Discovery 9.x Export Action

Use the **Discovery 9.x Export** action to set up a data export to Discovery Release 9.x. Before you create this action, you must define an export connection, since this action uses the export connection. For information about creating an export connection, see [Exporting Data](../../../ImportExportData/Data-Export.mdx).

Alternate Method: You can also do this from the **CI** workspace, however, to ensure that this action is applicable to all configuration items ensure that no record is selected in the **CI** workspace. For example, if a computer record is selected, the Action Designer Form will display information for the CI#Computer asset type.

### Parameters

| Parameter                | Description                                                                                                                                                                                                        |
| ------------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Enter action name        | A unique name for the action.                                                                                                                                                                                      |
| Enter action description | Optional. A description of what the action does.                                                                                                                                                                   |
| Enter category           | Optional. The category. Select **Configuration Management** from the drop-down list.                                                                                                                               |
| Connection               | The connection configured for Discovery Release 9.x exports. See [Exporting Data From Neurons for ITSM to Ivanti HEAT Discovery Release 9.x](../../../ImportExportData/Disc_Data_Export.mdx) for more information. |

### Actions

| Button | Action                 |
| ------ | ---------------------- |
| Save   | Commits your settings. |
| Cancel | Exits without saving.  |
