This topic describes how to create an action that executes a file or script on a remote server.

See [Run Program Action](<../templates/Run Program.mdx>).

1. From the Configuration console, click **Build > Automation Tools > Quick Actions** to open the **Actions Manager** workspace. The list of business objects that have actions appears.
2. Click **Add/Edit** for a business object. The application displays the Actions Center.
3. Click the **Create New Actions** tab.
4. Click **Run Program**. The application displays the **Run Program Execution** page.
5. Enter the name of the action, then enter information into the fields. See [Run Program Action](<../templates/Run Program.mdx>) action parameters.

*Run Program Action*

6. Click **Save**.
