When you open the \{\{FRS\_VARS.Graphical\_Action\_Designer}} from a business object, a **New Actions** list is displayed containing action templates relevant to that context. See [Using the Graphical Action Designer (GAD)](../Using_the_Graphical_Action_Designer.mdx) for more information.

**To select and configure record actions:**

1. Select the **New Actions** list, and then select **Record Actions**.
2. Select the action you require, and drag it into the Design Board of the \{\{FRS\_VARS.Graphical\_Action\_Designer}}.
   When the **Action Block** is selected in the Design Board, its parameter fields and options are displayed in the **Settings** pane.
3. Configure the action block parameters.

<hint type="info">
  For information on the individual record action parameter settings, see the action topics in this help section.
</hint>
