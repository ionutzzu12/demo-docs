## About the Insert Parent Record Quick Action

The **Insert Parent Record** action inserts a new record into a form or list as a parent record.

## Parameters

To access the parameters, select the **Action Block** to display them in the **Settings** pane.

### Action Block Settings

**Block Label**: A unique name for the block. This is useful to identify each action when you have more than one of the same action block added or cloned in the design area.

Click **Insert Parent Record** to open the workspace for the \{\{FRS\_VARS.Graphical\_Action\_Designer}}.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/Mk33Z4FLGe7eUImFm2anG_qa-insert-parent-record-workspace-gad.png" alt="An image of the workspace for the Insert Parent Record Quick Action block." position="flex-start" size="40" initialPath="../../../../Resources/Images/help-images/qa insert parent record workspace gad.png" githubPath="Content/Resources/Images/help-images/qa insert parent record workspace gad.png" showCaption="false"></Image>

- **Relationship**: Select the relationship from the dropdown menu.
- **Field Name**: Select the field name to update from the dropdown menu.
- **Value Expression**: Enter the value to insert or use the **Expression Editor** to create an expression for the value.
- Click **Add****+** to add a field to the list.
- Click **Delete**:inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/Bq5RUs4XUG4kziOcxWb8e_deleteicon1.png" alt caption size="10" width="15" height="18"} to remove a field from the list.

Click **Save&#x20;**&#x74;o commit your settings.
