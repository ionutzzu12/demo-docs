## About the Clone a Record Quick Action

The **Clone a Record** action creates a copy of the selected record and overwrites the specified fields, including copying from one-to-one relationships.

## Parameters

To access the parameters, select the **Action Block** to display them in the **Settings** pane.

### Action Block Settings

**Block Label**: A unique name for the block. This is useful to identify each action when you have more than one of the same action block added or cloned in the design area.

**Clone Editor**: Click **Clone Editor** to open the **Clone Record** workspace in the \{\{FRS\_VARS.Graphical\_Action\_Designer}}.

- **Field Name**: Select a field name from the dropdown menu.

<hint type="info" indent="2">
  When updating a field that uses counters (such as **IncidentNumber**), enable the **Unique** property (see [Creating a Field](../../../BusinessObjects/Creating-a-Field.mdx)) to avoid duplicate records.
</hint>

- **Value Expression**: Enter the desired replacement value for the **Field Name**. Use the **Expression Editor** to enter an expression for the value. See [Using the Expression Editor](<../../../../Reference/Expressions/Using Expression Editor.mdx>) for more information. To avoid copying values for any field (such as **ClosingEscLink**), leave this field empty.
- **Relationship Name**: Use the **Relationship Name** field to indicate the objects to associate and clone with the record.
