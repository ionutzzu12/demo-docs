## About the Search and Unlink Quick Action

The **Search and Unlink** action searches for and unlinks a business object record from another record. For example, you can unlink a configuration item record from an incident.

## Parameters

To access the parameters, select the **Action Block** to display them in the **Settings** pane.

### Action Block Settings

- **Block Label**: A unique name for the block. This is useful to identify each action when you have more than one of the same action block added or cloned in the design area.
- **Condition Editor**: Click **Condition Editor** to open the **Condition Editor** workspace in the \{\{FRS\_VARS.Graphical\_Action\_Designer}}.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/mkRYgiMoJPtzMxkaZZwG8_qa-condition-editor-gad.png" alt="Image of the Condition Editor in the Graphical Action Designer workspace" position="flex-start" size="40" indent="2" initialPath="../../../../Resources/Images/help-images/qa condition editor gad.png" githubPath="Content/Resources/Images/help-images/qa condition editor gad.png" showCaption="false"></Image>

<Paragraph listStyleType="disc" indent="2">
  **Relationship**: The relationship between the current business object and the business object from which you want to unlink. Select the desired relationship from the dropdown menu.
</Paragraph>

<hint type="info" indent="3">
  The relationship you choose will dictate how the resulting dropdown menus and options populate and interact.
</hint>

<Paragraph listStyleType="disc" indent="2">
  **Where**: When you select the relationship, the first of three dropdown menus populates with the master and related fields to evaluate and link.
</Paragraph>

<Paragraph listStyleType="disc" indent="3">
  **Find Where**: Where to search.
</Paragraph>

<Paragraph listStyleType="disc" indent="3">
  **Operator**: A comparison operator.
</Paragraph>

<Paragraph listStyleType="disc" listStart="2" indent="3">
  **String**: A string to use for comparison.
</Paragraph>

<Paragraph listStyleType="disc" indent="4">
  Enter a string manually, or click the **f(x)** icon to open the **Expression Editor** and enter an expression to use for comparison.
</Paragraph>

<Paragraph listStyleType="disc" listStart="3" indent="3">
  **Add**
</Paragraph>

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/Ug0yqnese1-CHfYDWwC8y_qa-search-and-link-add-button.png" alt="Image of the Add icon and drop-down menu for adding search parameters to the Search and Link Quick Action block" position="flex-start" size="40" indent="4" initialPath="../../../../Resources/Images/help-images/qa search and link add button.png" githubPath="Content/Resources/Images/help-images/qa search and link add button.png" showCaption="false"></Image>

<Paragraph indent="3">
  another parameter to the search. Use the dropdown menu to select an **AND** or an **OR** search parameter.
</Paragraph>

  **Delete** :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/Bq5RUs4XUG4kziOcxWb8e_deleteicon1.png" alt caption size="10" width="15" height="18"} the search parameter.
</Paragraph>
