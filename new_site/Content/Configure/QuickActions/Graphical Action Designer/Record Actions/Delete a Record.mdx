## About the Delete a Record Quick Action

The **Delete a Record&#x20;**&#x61;ction deletes the current context business object. Use this action within a search.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/9GOjLrPvQMXR8chyOpBku_qa-delete-record-block.png" alt="Image of the Delete a Record quick action block in the graphical action designer" position="flex-start" size="40" initialPath="../../../../Resources/Images/help-images/qa delete record block.png" githubPath="Content/Resources/Images/help-images/qa delete record block.png" showCaption="false"></Image>

<hint type="info">
  Use extreme caution if using this in a composite action because it could potentially delete the wrong value.
</hint>

## Parameters

To access the parameters, select the **Action Block** to display them in the **Settings** pane.

### Action Block Settings

**Block Label**: A unique name for the block. This is useful to identify each action when you have more than one of the same action block added or cloned in the design area.

There are no other specific parameters to configure for this action.
