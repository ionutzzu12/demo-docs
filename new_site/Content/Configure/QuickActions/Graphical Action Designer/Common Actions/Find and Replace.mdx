## About the Find and Replace Quick Action

The **Find and Replace** \{\{FRS\_VARS.Quick\_Action}} evaluates the selected records or all records for any instances of the specified field that contain the specified current value. When it finds that field and current value, it replaces the current value with the new value. You can replace values in an individual incident, multiple selected incidents, or all incidents simultaneously.

<hint type="info">
  The **Find and Replace** \{\{FRS\_VARS.Quick\_Action}} is not supported within composite actions, it can only be used as a stand-alone action. When creating a composite action, the **Find and Replace** \{\{FRS\_VARS.Quick\_Action}} template is disabled in the **New Actions** list on the Graphical Action Designer.
  The **Find and Replace** \{\{FRS\_VARS.Quick\_Action}} function is limited to a maximum of 100 records.
</hint>

## Suggestions for Use

Here are a few ways you can use the **Find and Replace** \{\{FRS\_VARS.Quick\_Action}}:

- **Replacing the&#x20;**\{\{FRS\_VARS.Incident}}**&#x20;owner's name**: For example: If a technician leaves the company, you can find all issues currently assigned to him or her as owner, and replace the value in owner field with a new technician's name.
- **Change the impact of incidents**: For example, you can make a change to a group of Incident records when Impact was previously Medium and it is now Low or High.

## Parameters

To access the parameters, select the **Action Block** to display them in the **Settings** pane.

### Action Block Settings

- **Block Label**: A unique name for the block. This is useful to identify each action when you have more than one of the same action block added or cloned in the design area.
- Click :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/3s_8tAFVBxXPx_U4PGM4l_qa-find-and-replace-add-field-button.png" alt caption size="10" width="89" height="14"} to add two fields:
  - **Field 1** where the user is prompted to enter a value to search for when running the **Find and Replace** \{\{FRS\_VARS.Quick\_Action}}.
  - **Field 2** where the user is prompted to enter a value to replace.
- To delete a field, click the **X** next to the field label.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/bc1-gSmpMcEBiz34edHnC_qa-find-and-replace-field-delete.png" position="flex-start" size="40" indent="2" initialPath="../../../../Resources/Images/help-images/qa find and replace field delete.png" githubPath="Content/Resources/Images/help-images/qa find and replace field delete.png" showCaption="false"></Image>

Click **Save** to commit your changes.
