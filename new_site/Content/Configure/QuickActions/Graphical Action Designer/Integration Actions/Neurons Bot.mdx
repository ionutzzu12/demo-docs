You can enable the \{\{FRS\_VARS.Neurons\_Bot}} for \{\{FRS\_VARS.Quick\_Actions}} from the \{\{FRS\_VARS.Neurons\_platform}} Integration console.

1. Log in to \{\{FRS\_VARS.ProductName}} as an \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
2. Open the Configuration console > **Extend** >**&#x20;Neurons Platform Integration**.
3. Toggle the \{\{FRS\_VARS.Neurons\_Bot}} section to enable it for the \{\{FRS\_VARS.Quick\_Actions}}.

## Parameters

To access the parameters, select the **Action Block** to display them in the **Settings** pane.

### Action Block Settings

<hint type="info">
  \{\{FRS\_VARS.Neurons\_Bot}} input definitions automatically synchronize once per hour. Any changes you have made within the last hour in \{\{FRS\_VARS.Company}} \{\{FRS\_VARS.Neurons}} to the Bot definition may not be immediately visible for selection. If you need to synchronize any changes you have made to the selected Bot, you can use the **Refresh** function to manually synchronize the \{\{FRS\_VARS.Neurons\_Bot}} \{\{FRS\_VARS.Quick\_Action}} without waiting.
</hint>

- **Block Label**: A unique name for the block. This is useful to identify each action when you have more than one of the same action block added or cloned in the design area.
- **Bot**: Select the \{\{FRS\_VARS.Neurons\_Bot}} instance from the drop-down list that you want the \{\{FRS\_VARS.Neurons\_Bot}} \{\{FRS\_VARS.Quick\_Action}} to trigger.
- **Refresh**: See the note above. Use this function to manually synchronize any changes you have made to the selected Bot definition in \{\{FRS\_VARS.Neurons}}. You may receive a **Warning** message if the refresh is unsuccessful, or has already been completed.
- **Inputs**: Inputs determine the action that the \{\{FRS\_VARS.Neurons\_Bot}} performs when invoked. Inputs vary depending on the \{\{FRS\_VARS.Neurons\_Bot}} instance selected. Some \{\{FRS\_VARS.Neurons\_Bot}} instances have no inputs. Inputs shown are defined in the \{\{FRS\_VARS.Neurons\_Bot}} configuration in \{\{FRS\_VARS.Company}} \{\{FRS\_VARS.Neurons}}. Select or enter parameter values and settings for input fields where required. For further information on inputs, refer to [Neurons Bots editor](https://help.ivanti.com/ht/help/en_US/CLOUD/vNow/neurons-bots-editor.htm). Field types are:
  - Text/String
  - Number
  - Boolean (True/False)
  - Date
  - Option - these are drop-down fields with multiple choice lists.
- Click **Save** to commit your settings.
