**Integration** \{\{FRS\_VARS.Quick\_Action}} blocks provide actions you can use with \{\{FRS\_VARS.ProductName}} integrations through the \{\{FRS\_VARS.Quick\_Action}} workspace.

## List of Integration Quick Actions

- [Decompress Discovery Message Quick Action](<Decompress Discovery Message.mdx>)
- [Mobile Push Notification Quick Action](<Mobile Push Notification.mdx>)
- [Neurons for Healing Quick Action](<Neurons for Healing.mdx>)
- [Start IPCM Voice Activity Quick Action](<Start IPCM Voice Activity.mdx>)
  \{\{FRS\_VARS.Classic\_Action\_Designer}}: [Start Ivanti Voice Activity Action](<../../Classical Action Designer/templates/Start IPCM Voice Activity.mdx>)
- [Start Remote Control Quick Action](<Start Remote Control.mdx>)
- [Sync Remote Control Session Quick Action](<Sync Remote Control Session.mdx>)

When you open the \{\{FRS\_VARS.Graphical\_Action\_Designer}} from a business object, a **New Actions** list is displayed containing action templates relevant to that context. See [Using the Graphical Action Designer (GAD)](../Using_the_Graphical_Action_Designer.mdx) for more information.

**To select and configure integration actions:**

1. Select the **New Actions** list, and then select **Integration Actions**.
2. Select the action you require, and drag it into the Design Board of the \{\{FRS\_VARS.Graphical\_Action\_Designer}}.
   When the **Action Block** is selected in the Design Board, its parameter fields and options are displayed in the **Settings** pane.
3. Configure the action block parameters.

<hint type="info">
  For information on the individual integration action parameter settings, see the action topics in this help section.
</hint>
