Use the **Sync Remote Control Session** action primarily with the **Run Program for the Remote Host** feature. Read more about [Using the Run Program for the Remote Host](../../../ImportExportData/Run-Program.mdx). Remote control settings are configured via [ITSM EPM Remote Control Integration](<../../../Endpoint Manager/ISM-EPM-Remote-Control-Integration.mdx>).

## About the Sync Remote Control Session Quick Action

The \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}} who configures the **Run Program for the Remote Host** feature can configure a \{\{FRS\_VARS.Quick\_Action}} using this action block. This action block:

1. Retrieves all on-going and planned remote control sessions from the table **Journal#RemoteControlSession**.
2. For each session, the \{\{FRS\_VARS.Quick\_Action}} checks for expiration and updates the start and end times of each session to the local time (if defined) in the table **Journal#RemoteControlSession**.

## Parameters

To access the parameters, select the **Action Block** to display them in the **Settings** pane.

### Action Block Settings

**Block Label**: A unique name for the block. This is useful to identify each action when you have more than one of the same action block added or cloned in the design area.

There are no other specific parameters to configure for this action.
