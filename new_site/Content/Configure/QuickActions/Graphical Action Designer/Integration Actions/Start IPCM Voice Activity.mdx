## About the Start IPCM Voice Activity Quick Action

The **Start IPCM Voice Activity** action configures and calls a web service for the \{\{FRS\_VARS.Company}} Voice application. Before configuring this action, you must set up the \{\{Primary.Company}} Voice integration to \{\{FRS\_VARS.ProductName}}. See [Configuring Ivanti Voice Integration](../../../Voice/Configure_Voice_Integrat.mdx).

## Parameters

To access the parameters, select the **Action Block** to display them in the **Settings** pane.

### Action Block Settings

When the action executes, the \{\{FRS\_VARS.ProductName}} server calls the \{\{FRS\_VARS.Company}} Voice web service, which in turn launches the chosen Voice application and sends any parameters specified as properties available to the Voice application.

<hint type="info">
  When a Voice application is started via the web service, there is no active phone call; it is simply an interaction in \{\{FRS\_VARS.Company}} Voice terms. If the \{\{FRS\_VARS.Quick\_Action}} needs to initiate a phone call, use the **Make Call** action block to initiate the call.
</hint>

- **Block Label**: A unique name for the block. This is useful to identify each action when you have more than one of the same action block added or cloned in the design area.

<hint type="info" indent="2">
  You can have multiple **Start IPCM Voice Activity** action blocks, each running a separate report, in one composite action; therefore, you should populate this field with a unique, descriptive title. See [Composite Action](<../../Classical Action Designer/templates/Composite Action.mdx>) for more information.
</hint>

- **Voice Application Names**: Choose the name of the Voice application to be started from the dropdown menu.

<hint type="info" indent="2">
  If the \{\{FRS\_VARS.Company}} Voice web service is not configured correctly, \{\{FRS\_VARS.ProductName}} cannot retrieve the list of available Voice applications and displays an error message.
</hint>

- **Property Name**: The property name to pass to the Voice application.
- **Property Value**: The property value to pass to the Voice application.
- **+ Add Field**: Adds a second Property Name and Property Value to pass to the Voice application.

Click **Save** to commit your settings.
