## About the Create RFC for Variance Quick Action

The **Create RFC for Variance** action creates a **Change** business object (request for change) for a variance. For example, when the RAM size changes from the baseline to another value, it varies from the baseline. This action creates a **Change** business object for that variance and links between the business objects as needed.

## Parameters

To access the parameters, select the **Action Block** to display them in the **Settings** pane.

### Action Block Settings

<hint type="info">
  Some parameter field names may differ slightly in the Classical Action Designer UI.
</hint>

- **Block Label**: A unique name for the block. This is useful to identify each action when you have more than one of the same action block added or cloned in the design area.
- **Field Name**: The name of the field. Select from the dropdown menu.
- **Field Value**: The value of the field.
- **+ Add Field**: Adds a field to the list.

Click **Save** to commit your settings.
