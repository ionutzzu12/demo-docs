By default, an action is created with one action. But sometimes you might want to create a more complex action that involves multiple actions.

## Adding Multiple Actions in the Classic Action Designer

1. From the Configuration console, click **Build > Automation Tools > Quick Actions** to open the **Actions Manager** workspace. The list of business objects that have actions appears.
2. Click **Add/Edit** to see the actions for the business object. The application displays the Action Designer selection popup.

<hint type="info">
  If your action(s) includes deprecated action(s), you will either be warned about limited editing functionality (CAD), or these actions won't be visible (GAD).
</hint>

3. Select **Classic Action Designer**
4. Select the **Your Actions** tab to view a list of actions.
5. Select an action from the list. The **Action Configuration** interface appears.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/CsX18tpoaAj4dAfDESv-i_quickactionedit.png" alt="Action configuration buttons" position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/quick_action_edit.png" githubPath="Content/Resources/Images/help-images/quick_action_edit.png" showCaption="false"></Image>

6. From the button area, click **Append action**, then choose the action type from the drop-down list. Fields associated with the action type appear in the configuration area. The **Add new action** drop-down list appears.
7. Enter data and configure in the new action fields as required.
8. To add more new actions, click the add icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/9SKkTHEVfabOeACXWkky6_addblue.png" alt caption size="10" width="16" height="12"} then enter data in the fields as required.

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/BdYoz_1ETYLgczFNI4GKY_quickactionremovelist.png" alt="Adding more actions to a composite action" position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/quick_action_remove_list.png" githubPath="Content/Resources/Images/help-images/quick_action_remove_list.png" showCaption="false"></Image>

9. After you have added all actions and fields, click **Save**.

<hint type="info">
  To remove an action from the list, click **Remove from List** on the right side of the action.
</hint>

## Adding Multiple Actions in the Graphical Action Designer

1. :Link[Open the Graphical Action Designer]{label="Open the Graphical Action Designer" overridedLabel="Open the Graphical Action Designer" spaceId docId="T_2hq2qRJzWxiHLCfgn-7" version="v2" docAnchorId="#To_open_the_Graphical_Action_Designer" loadingMethod="dynamic" newTab="false" githubPath="Graphical Action Designer/Using_the_Graphical_Action_Designer.htm#To_open_the_Graphical_Action_Designer" href="Graphical Action Designer/Using_the_Graphical_Action_Designer.mdx"}.
2. Select the \{\{FRS\_VARS.Quick\_Action}} from the **Saved Actions** list. The action block (or blocks if it is a composite action) are displayed on the Design Board.
3. Select an action from the **New Actions** list and drag it onto the Design Board.

<hint type="info">
  If your action(s) includes deprecated action(s), you will either be warned about limited editing functionality (CAD), or these actions won't be visible (GAD).
</hint>

4. Select the action block you want to link from, hover the mouse pointer over the **Link** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/-VPjBh3QwBIWCn6zIsTUy_qaguilinkactionblock.png" alt caption size="10" width="24" height="24"}, and then drag the link to the action block you want to link to. A link is displayed linking the two action blocks.

<hint type="info">
  Links are added in the desired order of workflow as shown by the arrow on the link. For example, in the diagram below the **Update a record** action runs, followed by the **Send an email** action.
</hint>

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/5Qch5sfLcWlPMqlDNC-3g_actiongraphicaldesignerlink.png" alt="Order that actions run in" position="flex-start" size="40" indent="1" initialPath="quick-actions-images/action_graphical_designer_link.png" githubPath="Content/Configure/QuickActions/quick-actions-images/action_graphical_designer_link.png" showCaption="false"></Image>

5. Enter data and configure the new action block fields as required.
6. Configure any existing action blocks if required as a result of adding the new action, including the **Start Block** fields if relevant.
7. To add more actions, select them one at a time from the **New Actions** list, and then drag and drop onto the Design Board.
8. After you have added, linked and configured all of the required action blocks and fields, clic&#x6B;**&#x20;Save**.

<hint type="info">
  To delete a link, select it and click **Remove Link** :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/Q-No1g8gET20kPcRZE1Al_qaguideletelink2actionblock.png" alt caption size="10" width="24" height="24"}.
  To remove an action from the Design Board, select it and click **Delete Action Block** :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/6cIM3pV2qzSgBSddt1myz_qaguideleteactionblock.png" alt caption size="10" width="24" height="24"}. This also deletes any links from/to the action block.
</hint>
