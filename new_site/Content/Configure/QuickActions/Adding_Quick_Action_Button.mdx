You can add an action button to the business object form toolbar for an action you create.

The default buttons (**New**, **Delete**, **Refresh** and **Action Menu**) are not action buttons. You cannot delete or select them while in action configuration mode (as indicated by the lock button).

## Add an Action button from the Classic Action Designer

1. From the Configuration console, click **Build > Automation Tools > Quick Actions** to open the **Actions Manager** workspace. The list of business objects that have actions appears.
2. Click **Add/Edit** to see the actions for the business object. The application displays the Action Designer selection popup.
3. Select **Classic Action Designer**.
4. Open an action from either the **Your Actions** list, or the **Create New Actions** list. The application displays the configuration interface.
5. Click **Edit**.
6. On the right side, under the **Action Settings** tab, select **Show as toolbar button**.

  *Show as toolbar button*
</Paragraph>

7. Click **Save**. The action is now a button on the toolbar.
8. Drag and drop the button along the toolbar to place it.
9. To change the position after you have placed it, drag and drop at the preferred position.

<hint type="info" indent="1">
  You can add/edit the actions from either the **Configuration** console or the **Service Desk** console (from any workspace > **More** > **Edit Actions** > **Edit**). You can change the position of the action button from where it was created. However, if you create the action from the **Configuration** console, then you can only change the position of the action from the **Configuration** console. Similarly, if the action was created from **Service Desk** console, you can only change the position from the **Service Desk** console.
</hint>

10. To remove the action button from the toolbar, drag it off the toolbar onto the main part of the page. The button is now gone, but the action still resides under the **Your Actions** tab.

## Add an Action button from the Graphical Action Designer

1. From the Configuration console, click **Build > Automation Tools > Quick Actions** to open the **Actions Manager** workspace. The list of business objects that have actions appears.
2. Click **Add/Edit** to see the actions for the business object. The application displays the Action Designer selection popup.
3. Select **Graphical Action Designer**.
4. Open an action from the **Your Actions** list, or drag an action from the **New Actions** list onto the Design Board.
5. On the right side, under **Settings**, select the **Common** tab, and then select **Show as toolbar button**.
6. Click **Save**. The action is now a button on the toolbar.
