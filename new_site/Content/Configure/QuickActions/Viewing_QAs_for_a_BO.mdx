<hint type="info">
  The administrator defines which roles can create, edit, or delete actions. See [Working with Roles](../../ServiceDesk/Roles/roles.mdx).
</hint>

Follow these steps to view the default actions for a business object:

1. From the Configuration console, click **Build > Automation Tools > Quick Actions** to open the **Actions Manager** workspace. The list of business objects that have actions appears.
2. Click **Add/Edit** for the business object. The application displays the Action Designer selection popup.
3. Select one of the following designers:
   - **Graphical Action Designer**: For more information on this designer, see [Using the Graphical Action Designer (GAD)](<Graphical Action Designer/Using_the_Graphical_Action_Designer.mdx>).
   - **Classic Action Designer**: For more information on this designer, see [Using the Classic Action Designer (CAD)](<Classical Action Designer/Using_the_Classic_Action_Designer.mdx>).

<Paragraph indent="1">
  The actions for that business object appear under the **Your Actions** tab.
</Paragraph>

4. To view the details of an action, click the action name.
