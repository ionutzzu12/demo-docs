createdAt: Tue Jun 30 2026 17:16:51 GMT+0300 (Eastern European Summer Time)

To add the Social Board to a role, do the following:

1. From the Configuration Console, click **Configure > Users and Permissions > Roles and Permissions** to open the **Roles and Permissions** workspace to view available roles.

<Paragraph indent="1">
  *Roles and Permissions Workspace*
</Paragraph>

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/4WREQr44rcdku2QTITYcH_socialactivate1.png" alt="Image of the Roles and Permissions workspace." position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/social.activate.1.png" githubPath="Content/Resources/Images/help-images/social.activate.1.png" showCaption="false"></Image>

2. Select a role to see the details for that role. This example uses the **SelfService** role.
3. Click **Top Level Tabs** to see the available tabs and workspaces for the **SelfService** role. You can optionally change its display order by dragging it up or down the workspace list.
4. Click **Add New Tab**.

<Paragraph indent="1">
  *Add User Interface*
</Paragraph>

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/JPJT9qX8cFP50UMy3FvXF_socialactivate2.png" alt="Image of the Add User interface." position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/social.activate.2.png" githubPath="Content/Resources/Images/help-images/social.activate.2.png" showCaption="false"></Image>

5. Select the **Social Board** workspace.

<Paragraph indent="1">
  *Social Board Workspace*
</Paragraph>

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/3SCbwBJErCeA5iI65a0pn_socialactivate3.png" alt="Selecting the Social Board workspace." position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/social.activate.3.png" githubPath="Content/Resources/Images/help-images/social.activate.3.png" showCaption="false"></Image>

6. For the **Social Board** tab, define the properties as necessary.

<Paragraph indent="1">
  *Social Board Properties*
</Paragraph>

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/-2GOgzHU4amSE41ErZ_zI_socialactivate4.png" alt="Defining the Social Board properties." position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/social.activate.4.png" githubPath="Content/Resources/Images/help-images/social.activate.4.png" showCaption="false"></Image>

7. Save your changes.

<Paragraph indent="1">
  The **Social Board** workspace is now visible for the **Self Service** role.
</Paragraph>

<Paragraph indent="1">
  *Social Board Workspace*
</Paragraph>
