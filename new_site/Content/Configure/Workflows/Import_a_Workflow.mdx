## Importing a Workflow

You can import a workflow to use as a basis for other workflows or to duplicate a workflow.

1. From the Configuration Console, click **Build > Workflow > Workflows** to open the **Workflows** workspace.
2. Click the workflow to import into. The version list appears.
3. Check the workflow version to import.
4. Click **Import Workflow** from the workspace toolbar. The navigation window appears.

*Importing a Workflow*

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/QA3o4vafIWDf8edvjhFZJ_workflow-import.png" position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/workflow import.png" githubPath="Content/Resources/Images/help-images/workflow import.png" showCaption="false"></Image>

5. Navigate to and select your workflow XML file, then click **Open**. The application imports the workflow and displays it as a new version on the version list.

## Exporting a Workflow

You can export a workflow to use in other workflows.

1. From the Configuration Console, click **Build > Workflow > Workflows** to open the **Workflows** workspace.
2. Click the workflow to export to. The version list appears.
3. Check the workflow version to export.

*Workflow Version List*

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/2UDn8lYdocvB4zFTmjYwt_workflow-export.png" position="flex-start" size="40" indent="1" initialPath="../../Resources/Images/help-images/workflow export.png" githubPath="Content/Resources/Images/help-images/workflow export.png" showCaption="false"></Image>

4. Choose **More Actions > Export Version**. The export confirmation window appears.
5. Check **Save File**, then click **OK**. The application saves the workflow XML file to your local drive, usually to your downloads folder.
