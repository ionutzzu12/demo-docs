The **Search and Link** workflow block searches and links a business object record to another record.

*Search and Link Workflow Block Properties*

## Exit Ports

| Exit Port | Function                                                                                             |
| --------- | ---------------------------------------------------------------------------------------------------- |
| OK        | Continue to the next block.                                                                          |
| Failed    | General exit port for a failed integration action. A logical next step is to trigger a notification. |
