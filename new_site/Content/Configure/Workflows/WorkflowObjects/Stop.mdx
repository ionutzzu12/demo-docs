The **Stop** workflow block ends the workflow. The **Stop** workflow block is required.

*Stop Workflow Block Properties*

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/QIyCQDXRNyNk2orfXGnNR_stopblock.png" position="flex-start" size="40" indent="1" initialPath="../../../Resources/Images/help-images/stop_block.png" githubPath="Content/Resources/Images/help-images/stop_block.png" showCaption="false"></Image>

## Exit Ports

None.

## Properties

| Property    | Description                            |
| ----------- | -------------------------------------- |
| Title       | A unique name for the workflow block.  |
| Description | A summary description of the workflow. |

## Actions

| Button | Action                 |
| ------ | ---------------------- |
| Save   | Commits your settings. |
| Cancel | Exits without saving.  |
