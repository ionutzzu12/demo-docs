## Exit Ports

This workflow block receives one of the following responses when it has finished executing:

| Exit Port           | Description                                                                                          |
| ------------------- | ---------------------------------------------------------------------------------------------------- |
| OK                  | Continue to the next block.                                                                          |
| Activation Failed   | The activation failed.                                                                               |
| Deactivation Failed | The deactivation failed.                                                                             |
| CI Unselected       | The configuration item was unselected.                                                               |
| CI Unidentified     | The configuration item was unidentified.                                                             |
| Unmanaged software  | The software was unmanaged.                                                                          |
| Assignment failed   | The assignment failed.                                                                               |
| Failed              | General exit port for a failed integration action. A logical next step is to trigger a notification. |

## Properties

| Property                               | Description                                                   |
| -------------------------------------- | ------------------------------------------------------------- |
| Title                                  | A unique name for the workflow block.                         |
| DSM Request Type Field                 | The HEAT DSM request type.                                    |
| Software Package Relationship          | The software package relationship.                            |
| Software Set Relationship              | The software set relationship.                                |
| Relationship to Software - Activation  | The relationship to software.                                 |
| CI Relationship                        | The configuration item relationship.                          |
| Ignore Maintenance Timeframe Field     | Specifies whether to ignore the maintenance time frame field. |
| Recalculation Installation Order Field | The recalculation installation order field.                   |
| Exception Message Field                | The exception message field.                                  |
