The **LDAP Sync** workflow block performs a synchronization of an existing LDAP configuration.

*LDAP Sync Workflow Block Properties*

## Exit Ports

| Exit Port | Description                                                                                          |
| --------- | ---------------------------------------------------------------------------------------------------- |
| Started   | Continue to the next block.                                                                          |
| Failed    | General exit port for a failed integration action. A logical next step is to trigger a notification. |

## Properties

| Property                | Description                                                                                                                                                                                     |
| ----------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Title                   | A unique name for the workflow block.                                                                                                                                                           |
| LDAP Configuration Name | The LDAP configuration name. Choose from the drop-down list. See [Configuring LDAP Settings](../../ImportExportData/LDAP_Settings.mdx).  Do this before activating and publishing the workflow. |

## Actions

| Button | Action                 |
| ------ | ---------------------- |
| Save   | Commits your settings. |
| Cancel | Exits without saving.  |
