Use the **Notification** workflow block to define settings for emails. You can create automated messaging within the body of the email message as well as designate the recipient individuals and teams.

## Exit Ports

| Exit Port | Description                 |
| --------- | --------------------------- |
| Sent      | Continue to the next block. |

## Properties

| Property       | Description                                                                                                                         |
| -------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| Title          | A unique name for the workflow block.                                                                                               |
| Log in Journal | Stores the text of the email in the journal.                                                                                        |
| From           | Who sends the email. Click **Add** to enter a valid sender email address or click **Address book**.                                 |
| To             | Who receives the email. Click **Add** to enter a valid recipient email destination address or click **Address book**.               |
| CC<br />BCC    | A recipient email address to copy (CC) or blind copy (BCC). Click **Add** to enter a valid email address or click **Address book**. |
| Plain Text     | Sends the email in plain text format (no HTML formatting).                                                                          |
| Subject        | The subject line.                                                                                                                   |
| Body           | The text of the automated message.                                                                                                  |

## Actions

| Button | Action                 |
| ------ | ---------------------- |
| Save   | Commits your settings. |
| Cancel | Exits without saving.  |
