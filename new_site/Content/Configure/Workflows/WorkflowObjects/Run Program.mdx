The **Run Program** workflow block runs a connection to a remote host to import data.

<hint type="info">
  This workflow block only supports the server mode quick action.
</hint>

## Exit Ports

| Exit Port | Description                                                                                          |
| --------- | ---------------------------------------------------------------------------------------------------- |
| OK        | Continue to the next block.                                                                          |
| Failed    | General exit port for a failed integration action. A logical next step is to trigger a notification. |

## Properties

| Property                | Description                                               |
| ----------------------- | --------------------------------------------------------- |
| Title                   | A unique name for the workflow block.                     |
| Run Program QuickAction | The defined quick action. Choose from the drop-down list. |

## Actions

| Button | Action                 |
| ------ | ---------------------- |
| Save   | Commits your settings. |
| Cancel | Exits without saving.  |
