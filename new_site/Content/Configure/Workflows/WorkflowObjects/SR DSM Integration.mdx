The **SR DSM Integration** workflow block performs the integration between the HEAT DSM server and \{\{FRS\_VARS.ProductName}}, such as provisioning and reinstalling computers and installing and uninstalling software.

*SR DSM Integration Workflow Block Properties*

## Exit Ports

| Exit Port | Description                                                                                          |
| --------- | ---------------------------------------------------------------------------------------------------- |
| OK        | Continue to the next block.                                                                          |
| Failed    | General exit port for a failed integration action. A logical next step is to trigger a notification. |
