The **Invoke for Child** workflow block invokes a workflow created for a child business object. The exit ports allow for paths contingent upon the workflow being started, completed, and failing.

*Invoke for Child Workflow Block Properties*

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/SmcNQHeH_ZAt6fWquKknm_invokeforchildblock.png" position="flex-start" size="40" indent="1" initialPath="../../../Resources/Images/help-images/invoke_for_child_block.png" githubPath="Content/Resources/Images/help-images/invoke_for_child_block.png" showCaption="false"></Image>

## Exit Ports

| Exit Port | Description                 |
| --------- | --------------------------- |
| OK        | Continue to the next block. |

## Properties

| Property                | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Title                   | A unique name for the workflow block.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |
| Relationship Name       | A valid relationship defined for the child object in a relationship. Choose from the drop-down list.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         |
| Workflow Name           | An existing workflow to invoke. Choose from the drop-down list.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| Child Filter Conditions |                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              |
| AND / OR                | A child filter condition.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| Field                   | An operand. Choose from the drop-down list.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |
| Operator                | An additional operator to apply. Choose from the drop-down list.<br />**Equal to**: The value in **Field** matches the value in **Criteria**.<br />**Not Equal to**: The value in **Field** does not match the value in **Criteria**.<br />**Begin with**: The value in **Field** starts with the value in **Criteria**.<br /><br />**Contains**: The value in **Field** contains the string value in **Criteria**.<br />**Does Not Contain**: The value in **Field** does not contain the string value in **Criteria**.<br />**Is Empty**: **Field** does not contain any values.<br />**Is Not Empty**: **Field** contains something (is not empty).<br /> |
| Value                   | A value. Select from the drop-down list.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
