After you configure the ticket classification feature in \{\{Primary.Product}}, the application may display the following error messages due to configuration or mapping issues:

| Error Message                                                                                                                 | Meaning                                                                                       |
| ----------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------- |
| Connection failed. URL links for Ticket Classification are not configured correctly. Please contact your System Administrator | Ticket classification Business Object endpoints are not configured or incorrectly configured. |
| Authentication failed. Please contact your System Administrator                                                               | Application fails to get token to access Ivanti Neurons tenant.                               |
| Ticket Classification Mappings are not configured. Please configure the mappings and try again                                | Mappings are not configured or Layout is not specified correctly.                             |
| Something Went Wrong. Please try again later                                                                                  | Any internal server error occurs.                                                             |
