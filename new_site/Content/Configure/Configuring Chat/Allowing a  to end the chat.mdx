---
title: Allowing Service Desk Analysts to end the chat
docTags: 
createdAt: Tue Jun 30 2026 17:16:51 GMT+0300 (Eastern European Summer Time)
---

A \{\{FRS\_VARS.Neurons\_ITSM\_-\_Service\_Desk\_Analyst-Service-Desk-Analyst}} can end the chat if there is no response from the end user after the first session expiry message has been sent.

However, this option is available to the \{\{FRS\_VARS.Neurons\_ITSM\_-\_Service\_Desk\_Analyst-Service-Desk-Analyst}} only if the \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}} has enabled the **EndChatSessionByAnalysts global** constant.

**To enable this feature:**

1. Log in to \{\{FRS\_VARS.ProductName}} as an \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
2. Open the Configuration console, select **Build** > **Global Constants**.
3. Set the **EndChatSessionByAnalysts** option as **True**.

<Paragraph indent="1">
  By default, this option is set as **False**.
</Paragraph>

4. Save the changes.
