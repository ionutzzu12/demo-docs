A \{\{FRS\_VARS.Neurons\_ITSM\_-\_Service\_Desk\_Analyst-Service-Desk-Analyst}} can create a security Incident from the chat session using the **Create Security Incident** Quick Action added to the **ivnt\_Chat** business object. This Quick Action can be customized as required.

**To customize the Create Incident Quick Action**

1. Log in to \{\{FRS\_VARS.ProductName}} as \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
2. Open the Configuration console.
3. Select **Build > Business Objects**.
4. In the **Find** field, enter **ivnt\_Chat** to search for the Chat business object, then select it in the list to open it.
5. Select the **Quick Actions** tab.
   The action designer selection dialog opens.
6. Select **Graphical Action Designer** or **Classic Action Designer**, depending on which designer you prefer to use.
   Refer to [Using the Graphical Action Designer (GAD)](<../QuickActions/Graphical Action Designer/Using_the_Graphical_Action_Designer.mdx>) or [Using the Classic Action Designer (CAD)](<../QuickActions/Classical Action Designer/Using_the_Classic_Action_Designer.mdx>) for more information.
7. Select **Saved Actions**.

<hint type="info">
  If you are using an older version of the **Classic Action Designer**, select **Your Actions**.
</hint>

8. Select **Create Security Incident**.
9. Edit the fields and parameters as required.
10. Click **Save**.
