## Access iPaaS configuration

Use \{\{FRS\_VARS.Product\_Name}} Integration to connect Ivanti Neurons to external automation tools and create workflows with available connectors. This section explains how to access \{\{FRS\_VARS.Product\_Name}}integration and create Action Designer using required interface.

To access \{\{FRS\_VARS.Product\_Name}} Integration in \{\{FRS\_VARS.ITSM}}, do the following:

1. Log in to Ivanti Neurons for \{\{FRS\_VARS.ITSM}} as an \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.
2. Open the Configuration console, scroll down and select **Extend** > \{\{FRS\_VARS.Product\_Name}}**&#x20;Integration**.

## Use the action designer

You can use either \{\{FRS\_VARS.Classic\_Action\_Designer}} (CAD) or the \{\{FRS\_VARS.Graphical\_Action\_Designer}} (GAD). Both offer a visual interface for designing, configuring automation workflows, creating, or modifying \{\{FRS\_VARS.Quick\_Actions}}.

To enable the CAD or GAD option:

1. In the Configuration console, scroll down and select **Build** > **Business Objects**.
