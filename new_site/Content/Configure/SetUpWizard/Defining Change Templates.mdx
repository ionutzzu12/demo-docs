Each step of the \{\{FRS\_VARS.Change\_Enablement}} setup process is carried out on through the **Application Setup -&#x20;**\{\{FRS\_VARS.Change\_Enablement}} page. To find the setup page:

1. Log in as an \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}} and open the \{\{FRS\_VARS.Configuration\_console}}.
2. From the **Settings** pane, select **Configure** > **Organizational Profile** > **Application Setup**.
3. In the **Application Setup** workspace, select **Change Management** to display the **Application Setup - Change Enablement** page.

## Defining the Standard Change Templates

Use the **Standard Change Templates** to support standard changes in your organization. The application provides eight Standard Change Templates by default, and it publishes each template to an assigned subset of roles.

All Standard Change Templates are published to the \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}, \{\{FRS\_VARS.Neurons\_ITSM\_-\_Change\_Manager}}, \{\{FRS\_VARS.Neurons\_ITSM\_-\_Release\_Manager}}, \{\{FRS\_VARS.Neurons\_ITSM\_-\_Service\_Desk\_Analyst-Service-Desk-Analyst}}, and \{\{FRS\_VARS.Neurons\_ITSM\_-\_Service\_Desk\_Manager}}. Additional roles assigned to each template are listed under the template name.

1. Click **Standard Change Templates** to display the **Application Setup - Standard Change Templates** page.
   - Create Mail Archive for MS Exchange Mail Folder:
     - Support Desk Analyst
     - Support Desk Manager
   - Install Approved Software Package:
     - \{\{FRS\_VARS.Neurons\_ITSM\_-\_Portfolio\_Manager}}
     - \{\{FRS\_VARS.Neurons\_ITSM\_-\_Project\_Manager}}
   - MS Security Patch Install for Windows Servers:
     - \{\{FRS\_VARS.Neurons\_ITSM\_-\_Portfolio\_Manager}}
     - \{\{FRS\_VARS.Neurons\_ITSM\_-\_Project\_Manager}}
   - Replace Printer Accessory:
     - \{\{FRS\_VARS.Neurons\_ITSM\_-\_Portfolio\_Manager}}
     - \{\{FRS\_VARS.Neurons\_ITSM\_-\_Project\_Manager}}
   - Server Reboot:
     - \{\{FRS\_VARS.Neurons\_ITSM\_-\_Portfolio\_Manager}}
     - \{\{FRS\_VARS.Neurons\_ITSM\_-\_Project\_Manager}}
   - Server Relocation:
     - Support Desk Analyst
     - Support Desk Manager
     - \{\{FRS\_VARS.Neurons\_ITSM\_-\_Portfolio\_Manager}}
     - \{\{FRS\_VARS.Neurons\_ITSM\_-\_Project\_Manager}}
   - Standard:
     - \{\{FRS\_VARS.Neurons\_ITSM\_-\_Portfolio\_Manager}}
     - \{\{FRS\_VARS.Neurons\_ITSM\_-\_Project\_Manager}}
   - Update Antivirus Files:
     - Support Desk Analyst
     - Support Desk Manager
     - \{\{FRS\_VARS.Neurons\_ITSM\_-\_Portfolio\_Manager}}
     - \{\{FRS\_VARS.Neurons\_ITSM\_-\_Project\_Manager}}
2. To enable a template, click the checkbox preceding its name.
3. To disable a template, clear the checkbox. You cannot delete templates, but you can disable them.
4. To rename a template:
   1. Click **Rename** next to the template you want to rename.
   2. Change the name.
   3. Click **Save**.
5. To edit a template:
   1. Click **Edit Templates** to open the **Actions Manager** workspace in a new tab.
   2. Select the template to edit.
   3. Edit the template as needed. Refer to [Using Quick Actions](../QuickActions/Using_Quick_Actions.mdx) for more information.
