Use the **Application Setup - Change Enablement** page to customize how the \{\{FRS\_VARS.Neurons\_ITSM\_-\_Service\_Desk\_Console}} handles changes.

1. Log in as an \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}} and open the \{\{FRS\_VARS.Configuration\_console}}.
2. From the **Settings** pane, select **Configure** > **Organizational Profile** > **Application Setup**.
3. In the **Application Setup** workspace, select **Change Management** to display the **Application Setup - Change Enablement** page.

<hint type="info">
  This is one step of the application setup process. Refer to [Setting Up the Application](<Setting Up The Application.mdx>) for an overview of the full setup process.
</hint>

The **Application Setup -&#x20;**\{\{FRS\_VARS.Change\_Enablement}} page walks you through each step of setting up \{\{FRS\_VARS.Change\_Enablement}}. \{\{FRS\_VARS.ProductName}} automatically ties change records to the **Change Calendar**. No set up is required. However, you can change the name of the workspace and configure access for different roles. Refer to [Configuring Calendars for Roles](<../Roles/Configuring Calendars for Roles.mdx>).
