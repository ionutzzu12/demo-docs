Each step of the \{\{FRS\_VARS.Change\_Enablement}} setup process is carried out on through the **Application Setup -&#x20;**\{\{FRS\_VARS.Change\_Enablement}} page. To find the setup page

1. Log in as an \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}} and open the \{\{FRS\_VARS.Configuration\_console}}.
2. From the **Settings** pane, select **Configure** > **Organizational Profile** > **Application Setup**.
3. In the **Application Setup** workspace, select **Change Management** to display the **Application Setup - Change Enablement** page.

## Configuring the Change Dashboard

Set up your Change Dashboard to display the Change process information you need to see in summary. Use the **Application Setup - Dashboard Manager** page to select the dashboard parts to display on the **Change** dashboard. The application offers six options for populating the Change Dashboard:

- Requested Changes
- Top 5 CIs with Active Changes
- Changes Pending Approval
- Variance Report
- Active Changes by Type
- Major Changes

### Adding the summaries you want on your Change Dashboard

1. Click **Manage Dashboard**. The application displays the **Application Setup - Dashboard Manager** page.
2. Click the checkbox beside a dashboard part name to display the information on the **Change** dashboard.
3. Click **Preview** to see a preview of what the dashboard will look like with this dashboard part. Click **Escape** to close the preview.
4. To rename a dashboard, do the following:
   1. Click **Rename**.
   2. Enter a new name.
   3. Click **Save**.
