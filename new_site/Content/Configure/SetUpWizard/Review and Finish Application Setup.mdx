The **Application Setup - Review & Finish** page is part of the **Application Setup** workspace. Use this page to review the information you entered in this workspace.

This is one step of the application setup process. For more information about this process, including how to access this page, refer to [Setting Up the Application](<Setting Up The Application.mdx>).

1. Click **Review & Finish**. The application displays the **Application Setup - Review & Finish** page.
2. To edit any areas, either click **Back** until you find the area to edit, or select an area from the left pane.
3. To save your changes and finish the application setup, click **Finish**.
