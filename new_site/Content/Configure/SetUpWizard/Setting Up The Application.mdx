Use the **Application Setup** workspace to initially set up your \{\{FRS\_VARS.ProductName}} application. After the initial setup, you can access various parts of the **Application Setup** workspace from the Configuration Console to modify the settings.

Before you initially set any application parameters, the application displays a **Setup Wizard** link on the \{\{FRS\_VARS.ProductName}} pages. After you run the setup wizards and save your settings, the **Setup Wizard** link no longer appears.

## Accessing the Application Setup Workspace

1. From the Configuration Console, click **Configure > Organization Profile > Application Setup** to open the **Application Setup** workspace. By default, it goes to the **Application Setup - Organization Information** page.
