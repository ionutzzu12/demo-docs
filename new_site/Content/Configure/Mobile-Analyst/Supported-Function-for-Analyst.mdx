The following functions are supported in the responsive Analyst UI:

- CurrentLoginId
- CurrentUserDisplayName
- CurrentUserEmail
- CurrentUserField
- CurrentUserIdentityType
- CurrentUserInTeam
- CurrentUserOrganizationalUnit
- CurrentUserOrganizationalUnitName
- CurrentUserRole
- CurrentUserTeam
- CurrentUserTeamNames
- For more information, see [Current User Functions](<../../Reference/Functions/_Current User Functions.mdx>).
