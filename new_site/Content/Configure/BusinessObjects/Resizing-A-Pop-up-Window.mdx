The HTML editor includes a resize pop-up option in addition to a resizer. If you select an email in the **Email** workspace, you can maximize the email window.

1. From the **Email** workspace, open a&#x6E;**&#x20;Email**.
2. Select the **Maximize** button on the top right corner to maximize the screen. The email message expands into a full-screen window.
3. Select **Close** to return to the email. The **Close** button updates all of the changes that you made in the HTML editor in the pop-up window.
