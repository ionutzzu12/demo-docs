The **Object Templates Manager** workspace is a convenient way to view the new record templates for your business objects. When users create new records, they have the option of choosing one of the defined quick action templates for that object, automatically entering data into certain fields.

## Adding or Modifying a Business Object Template

1. From the Configuration console, select **Build** > **Automation Tools** > **Object Templates** to open the **Object Templates** workspace.
2. Select **Add/Edit** under the **Action** column for the object to edit. The Action Designer appears.
3. Create or modify the quick action as described in [Using Quick Actions](../QuickActions/Using_Quick_Actions.mdx).

## Translating a Business Object Template

1. From the Configuration Console, select **Build** > **Automation Tools** > **Object Templates** to open the **Object Templates** workspace.
2. From the **Object Templates Manager** page, click **Translation Tool** under the **Launch** column for the object template to translate. The list of available languages appears.
3. Select a language to display the Translation Tool.
4. Enter the strings. Refer to [Using the Ivanti Translation Tool](<../Localization/Using the Translation Tool.mdx>).
