When a user logs out, the application sets the **LogoutType** field for the **frs\_ops\_logon\_history** business object to explicit.

<Paragraph indent="1">
  **Logout time** equals **the time when the session is created** minus **the session start time** plus **the session timeout**.
</Paragraph>

When a user times out, the application sets the **LogoutType** field to empty.
