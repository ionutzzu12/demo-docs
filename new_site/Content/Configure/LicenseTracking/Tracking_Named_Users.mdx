createdAt: Tue Jun 30 2026 17:16:49 GMT+0300 (Eastern European Summer Time)
---

Follow these steps to designate a user as a named user. If a user is not specified as a named user, the application assumes that the user is a concurrent user.

1. Log in to \{\{FRS\_VARS.ProductName}}.
2. Open the **Employee** workspace.
3. Open the employee record for the employee for whom to make a named user.
4. Check **Is Named User**.
5. Click **Save**.
6. Return to the list view and repeat steps 1 to 3 for each employee who has a named-user license.
