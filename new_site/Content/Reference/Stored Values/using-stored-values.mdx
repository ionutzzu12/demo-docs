A stored value is a named piece of data that stores commonly used information. Use Stored Values across \{\{FRS\_VARS.ProductName}} anywhere that expressions exist. Use them for everything from Stored Expressions and Quick Actions to searches. Access and use Stored Values from the Stored Values workspace.

In addition to using stored values in expressions and Quick Actions, you can retrieve and update stored values using the Rest API. For more information on using the Rest API, see [Rest API](../../Configure/API/RestAPI-Introduction.mdx)and [Working with the Ivanti Integration Web Service](<../../Configure/Develop/Web Service.mdx>).

<ExpandableHeading>
  ## Using Stored Values in Expressions

  Stored values can be used anywhere you can use an expression.

  **To call a stored value:**

  1. Open the **Expression Editor**.
  2. In the **Toolbox**, choose **Functions** > **System Functions** > GetStoredValue. You also can begin typing the function into the search bar to find it in the list.
  3. Use the rules in [Expression Grammar](<../Expressions/Expression Grammar.mdx>) to write your expression.
     Call your stored value with this expression: $(GetStoredValue("YourStoredValueNameHere")).
