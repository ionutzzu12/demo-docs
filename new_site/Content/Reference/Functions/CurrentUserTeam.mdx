Returns the primary team for the user associated with the current session, as defined in the **Team** field under the **Details** tab on the **Employee Details** form.

<hint type="info">
  Use this function to return the name of the primary team.

  Use the [CurrentUserTeamNames](CurrentUserTeamNames.mdx) function to return a list of all the team names.

  Use the [CurrentUserTeamIds](CurrentUserTeamIds.mdx) function to return a list of the team IDs for all the teams.
</hint>

## Syntax

CurrentUserTeam()
