Returns the local date and/or time formatted as specified.

## Syntax

FormatDate(date, format, timeZone)

## Parameters

| Parameter | Description                                                                                                                                                                                                                       |
| --------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| date      | The date and/or time to format.                                                                                                                                                                                                   |
| format    | The format for the date and/or time.  See [Standard date and time format strings](https://learn.microsoft.com/en-us/dotnet/standard/base-types/standard-date-and-time-format-strings) for information about the accepted formats. |
| timeZone  | (Optional) The time zone, in IANA format.                                                                                                                                                                                         |

## Return Value

<Paragraph indent="1">
  String representation of date time value.
</Paragraph>

## Example

$(FormatDate(ResolvedDateTime, "dd.MM.yyyy HH\:mm"))

<Paragraph indent="1">
  If the value of the **ResolvedDateTime** field is April 14, 2017 at 1:25pm, the above example returns 14.04.2017 13:25.
</Paragraph>
