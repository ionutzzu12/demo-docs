Returns the time zone associated with an hours of operation (HOP) schedule. See [Setting Up Hours of Operation](../../Configure/HoursOfOp/Hours_of_Operation.mdx).

## Syntax

HOPTimeZone(hopIdOrName)

## Parameters

| Parameter   | Description                                                         |
| ----------- | ------------------------------------------------------------------- |
| hopIdOrName | The hours of operation schedule name (for example, 24x7 or 9 to 5). |

## Return Value

<Paragraph indent="1">
  Number value.
</Paragraph>

## Example

$(HOPTimeZone("Weekly HOP"))

<Paragraph indent="1">
  The example above returns Africa/Cairo.
</Paragraph>
