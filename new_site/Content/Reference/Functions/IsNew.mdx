Identify whether the current record is being created and has not been saved yet. Use this when creating a new record to set default values or trigger actions.

## Syntax

IsNew()

## Parameters

None.

## Return Value

Boolean

- *True* if the record is new (being created and not yet saved).
- *False* if the record already exists in the system.

## Example

The following example shows how to use the *IsNew()* to set a default priority when creating a new Incident.

$(If(IsNew(), "Priority = 'Medium'", ""))

If the application returns *True* when a new Incident is being created, so the **Priority** is set to **Medium**.

If the application returns *False* the Incident already exists and the field value remains unchanged.
