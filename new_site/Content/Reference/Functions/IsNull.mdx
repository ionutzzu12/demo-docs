Checks to see if a value is null. See [About Null Values](<../Expressions/Null Value.mdx>) for more information about null values.

## Syntax

IsNull(value)

## Parameter

| Parameter | Description            |
| --------- | ---------------------- |
| value     | The input to evaluate. |

## Return Value

<Paragraph indent="1">
  Boolean value. Returns true if the **value** parameter is null; otherwise, returns false.
</Paragraph>

## Examples

IsNull("This is a test")

<Paragraph indent="1">
  In the above example, the application returns false, because the input parameter is not null.
</Paragraph>

IsNull(null)

<Paragraph indent="1">
  In the above example, the application returns true, because the input parameter is null.
</Paragraph>
