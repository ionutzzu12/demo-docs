Returns the absolute value of the original.

An example is:

- Original token: -42
- Modifier: Absolute Value
- Result: 42

## Syntax

AbsoluteValue(inputvalue)

## Parameters

| Parameter  | Description                                        |
| ---------- | -------------------------------------------------- |
| inputvalue | Input value to perform AbsoluteValue(Number value) |

## Return Value

Returns the absolute value or modulus of the original. Any minus sign is removed.

## Example

$(AbsoluteValue(-33))

Returns this value: 33
