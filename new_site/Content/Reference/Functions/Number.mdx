Converts the input parameter into a number value.

<hint type="info">
  This function can be used to replace the TextToNumber modifier as used in \{\{FRS\_VARS.CSM}}.
</hint>

## Syntax

Number(value)

## Parameter

| Parameter | Description            |
| --------- | ---------------------- |
| value     | The input to evaluate. |

## Return Value

<Paragraph indent="1">
  Number value. See also :Link[Conversion to Number Values]{label="Conversion to Number Values" overridedLabel="Conversion to Number Values" spaceId docId="eVZtF84cBSwgHH8QyCA6X" version="v2" docAnchorId="#number_Conversion" loadingMethod="dynamic" newTab="false" githubPath="../Expressions/Type Conversions.htm#number_Conversion" href="../Expressions/Type Conversions.mdx"}.
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  If the **value** parameter is a number, returns the number unchanged.
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  If the **value** parameter is a text value that contains a valid number image, returns the number in the text.
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  If the **value** parameter is a Boolean value and is true, returns 1. If the **value** parameter is a Boolean value and is false, returns 0 .
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  If the **value** parameter is a currency value, returns the amount.
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  If the **value** parameter is null, returns null.
</Paragraph>

## Example

$(Number("2.793"))

<Paragraph indent="1">
  The example above returns 2.793.
</Paragraph>

$(Number(1234.56HKD))

<Paragraph indent="1">
  The above example above returns 1234.56.
</Paragraph>

$(Number(true))

<Paragraph indent="1">
  The above example returns 1.
</Paragraph>
