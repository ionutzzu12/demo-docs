Returns result of a saved search for a particular field.

## Syntax

ExecuteSavedSearch(tableRef, SavedSearchName, fieldName)

## Parameters

| Parameter       | Description                                                                     |
| --------------- | ------------------------------------------------------------------------------- |
| tableRef        | The business object whose SavedSearch needs to be evaluated.                    |
| SavedSearchName | The name of the SavedSearch.                                                    |
| fieldName       | The name of the field, whose value has to be extracted from SavedSearch result. |

## Return Value

Array \[String].

<hint type="info">
  This function returns text (string) values only. It does not support numeric comparisons. If your saved search returns a number or integer, format it as a string or handle it using a separate logic path.
</hint>
