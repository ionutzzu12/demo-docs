Returns an XML element with a specific name.

## Syntax

FindElement(xmlString, tagName)

## Parameters

| Parameter | Description      |
| --------- | ---------------- |
| xmlString | XML input string |
| tagName   | XML element name |

## Return Value

Returns an XML element with a specific name.

## Example

$(FindElement('\<catalog>\<book>\<author>Johns, Jorge\</author>\<title>The Rest API\</title>\<price>3.50\</price>\</book>\<book>\<author>Boberson, Bob\</author>\<title>XML Modifiers and You\</title>\<price>\<priceUS>48.95\</priceUS>\<priceUK>48.96\</priceUK>\</price>\</book>\</catalog>', 'title'))

Returns this value: \<title>The Rest API\</title>
