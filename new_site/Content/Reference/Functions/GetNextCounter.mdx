Instructs the application to increment the specified counter and then returns the incremented value of a counter. For information on counters, see [Using Counters](../../Configure/Counters/Counters.mdx).

## Syntax

GetNextCounter(counterName)

## Parameter

| Parameter   | Description                           |
| ----------- | ------------------------------------- |
| counterName | The name of the counter to increment. |

## Return Value

<Paragraph indent="1">
  Text value.
</Paragraph>

## Example

$(GetNextCounter("AccountNumberCounter"))

<Paragraph indent="1">
  In the above example, if the value of the **AccountNumberCounter** counter is 1003 and the increment is 1, then the application updates the counter and returns 1004.
</Paragraph>
