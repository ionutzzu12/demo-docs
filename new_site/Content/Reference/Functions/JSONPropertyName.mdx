Returns the name of a property from a JSON object at a given index.

## Syntax

JSONPropertyName(jsonString, position)

## Parameters

| Parameter  | Description                                       |
| ---------- | ------------------------------------------------- |
| jsonString | Input JSON Object String                          |
| position   | Zero-based index of property whose name to return |

## Return Value

Returns the property name for given position from the input JSON string.

## Example

$(JSONPropertyName('\{"color": "red","price": 19.95,"used": "True","returnDate": "2012-04-23T18:25:43.5111Z"}', 2))

Returns this value: used
