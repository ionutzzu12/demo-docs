Returns the culture code associated with the current session. 

You can see the current browser culture by going to the Service Desk Console, clicking the down arrow next to your name, select **Change Locale**, and see the list of locales. Your current browser culture is marked. You can change the culture associated with a user from the **Employee** workspace.
