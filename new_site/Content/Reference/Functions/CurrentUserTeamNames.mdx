Returns the full set of team names for the user associated with the current session, as defined under the **Teams** tab on the **Employee Details** form.

<hint type="info">
  Use this function to return a list of all the team names.

  Use the [CurrentUserTeamIds](CurrentUserTeamIds.mdx) function to return a list of the team IDs for all the teams.

  Use the [CurrentUserTeam](CurrentUserTeam.mdx) function to return the name of the primary team.
</hint>

## Syntax

CurrentUserTeamNames()
