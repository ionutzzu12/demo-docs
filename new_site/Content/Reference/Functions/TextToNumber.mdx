Converts the value in a text token to a number. Numeric Modifiers can then be applied to the result. See [Numeric Modifiers](https://help.cherwell.com/bundle/csm_features_10_4_help_only/page/content/suite_features/modifiers/numeric_modifiers.html).

An example is:

- Original token: 12345
- Modifier: Text to Number
- Result: 12345 (now available for Numeric Modifiers)

## Syntax

TextToNumber(textValue)

## Parameters

| Parameter | Description                                         |
| --------- | --------------------------------------------------- |
| textValue | Input string to convert from text value to a number |

## Return Value

Returns a number for a text value.

## Example

$(TextToNumber('12345'))

Returns this value: 12345
