Returns a numerical value from a specified JSON element.

## Syntax

NumberFromElement(jsonString, nodeName, position, fieldName)

## Parameters

| Parameter  | Description       |
| ---------- | ----------------- |
| jsonString | Input JSON string |
| nodeName   | NodeName          |
| position   | Position          |
| fieldName  | FieldName         |

## Return Value

Returns a numerical value from a specified JSON element.

## Example

$(NumberFromElement('\{"book":\[\{"category":"reference","author":"Nigel Rees","title":"Sayings of the Century","price":8.95},\{"category":"fiction","author":"Evelyn Waugh","title":"Sword of Honour","price":12.99},\{"category":"fiction","author":"Herman Melville","title":"Moby Dick","isbn":"0-553-21311-3","price":8.99},\{"category":"fiction","author":"J. R. R. Tolkien","title":"The Lord of the Rings","isbn":"0-395-19395-8","price":22.99}]}', "book", 0, "price"))

Returns this value: 8.95
