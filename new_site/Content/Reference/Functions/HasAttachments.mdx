Identifies if the current record has any attachments linked. This is identified via the relationship to the Attachment business object.

<hint type="info">
  HasAttachments is only available, and can only be evaluated against, a business object that has a relationship to attachments.
</hint>

## Syntax

HasAttachments()

## Return Value

<Paragraph indent="1">
  Boolean value. Returns true if record has one or more attachments. Returns false if the record has no attachments.
