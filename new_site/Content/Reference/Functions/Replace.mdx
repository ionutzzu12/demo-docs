Replaces the specified text with some other specified text.

## Syntax

Replace(text,textToMatch,replacementText)

## Parameters

| Parameter       | Description                             |
| --------------- | --------------------------------------- |
| text            | The text in which to make replacements. |
| textToMatch     | The specific text to match.             |
| replacementText | The text with which to replace it with. |

## Return Value

<Paragraph indent="1">
  Text value.
</Paragraph>

## Example

$(Replace("This blue car is really blue", "blue", "yellow"))

<Paragraph indent="1">
  This example returns "This yellow car is really yellow."
</Paragraph>
