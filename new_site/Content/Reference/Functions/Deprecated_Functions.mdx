createdAt: Tue Jun 30 2026 17:16:58 GMT+0300 (Eastern European Summer Time)

Certain functions are deprecated. Expressions with these functions work; however, you cannot edit them and you cannot add new expressions with these functions. Those that give you a warning during use are marked with an asterisk \*.

The deprecated functions are:

- AsJSON
- AsXML
- AuditHistoryDescription
- CurrentGlobalCurrency - Use [DefaultCurrencySymbol](DefaultCurrencySymbol.mdx) instead.
- CurrentSecurityGroup
- CurrentUserInfo1
- CurrentUserName - Use [CurrentLoginId ](CurrentLoginID.mdx)or [CurrentUserDisplayName ](CurrentUserDisplayName.mdx).
- DateAdd\* - Use [AddYears](AddYears.mdx), [AddQuarters](AddQuarters.mdx), [AddMonths](AddMonths.mdx), [AddWeeks](AddWeeks.mdx), [AddDays](AddDays.mdx), [AddHours](AddHours.mdx), [AddMinutes](AddMinutes.mdx), or [AddSeconds](AddSeconds.mdx).
- DateDiff\* - Use [DateDiffSQL](DateDiffSQL.mdx).
- DateOrTimeFromElement - Use [DateTime](DateTime.mdx) with [JSONPathValue](JSONPathValue.mdx).
- DatePart\* - Use [LocalYear](LocalYear.mdx), [LocalQuarter](LocalQuarter.mdx), [LocalMonth](LocalMonth.mdx), [LocalWeek](LocalWeek.mdx), [LocalDay](LocalDay.mdx), [LocalHour](LocalHour.mdx), [LocalMinute](LocalMinute.mdx), or [LocalSecond](LocalSecond.mdx).
- EmptyString\* - Use "" instead.
- EndOfNextYear\* - Use [EndOfYear](EndOfYear.mdx)(1).
- EndOfThisMonth\* - Use [EndOfMonth](EndOfMonth.mdx)(0).
- EndOfThisYear\* - Use [EndOfYear](EndOfYear.mdx)(0).
- FindArray - Use [JSONPathArray](JSONPathArray.mdx).
- FindElement - Use [XPathElement](XPathElement.mdx).
- FindElementCollection - Use [XPathElement](XPathElement.mdx).
- FindElementwithAttribute - Use [XPathElement](XPathElement.mdx).
- FindObject - Use [JSONPathValue](JSONPathValue.mdx).
- FirstChild - Use [XPathElement](XPathElement.mdx).
- FormatDateTime - Use [FormatDate](FormatDate.mdx).
- GetActionVariable
- HasChildElements - Use [XPathElement](XPathElement.mdx).
- HashMACText - Replaced by [HashMACString](HashMACString.mdx).
- HashText - Replaced by [HashString](HashString.mdx).
- HashTrimmedText
- HOPCalcDuration\* - Use [DiffWorkingSeconds](DiffWorkingSeconds.mdx).
- HOPCalcTargetDate\* - Use [AddWorkingSeconds](AddWorkingSeconds.mdx).
- Last365Days\* - Use [BeginningOfDay](BeginningOfDay.mdx)(-365).
- Last3Months\* - Use [BeginningOfMonth](BeginningOfMonth.mdx)(-3).
- Last90Days\* - Use [BeginningOfDay](BeginningOfDay.mdx)(-90).
- LastChild - Use [XPathElement](XPathElement.mdx).
- LastDays\* - Use [BeginningOfDay](BeginningOfDay.mdx).
- LastSevenDays\* - Use [BeginningOfDay](BeginningOfDay.mdx)(-7).
- LastThirtyDays\* - Use [BeginningOfDay](BeginningOfDay.mdx)(-30).
- LastWeeks\* - Use [BeginningOfWeek](BeginningOfWeek.mdx).
- LastWord - Replaced by [FirstRegexMatch](FirstRegexMatch.mdx)
- LastYears\* - Use [BeginningOfYear](BeginningOfYear.mdx).
- Left - Replaced by [Substring0](Substring0.mdx)
- LogicalFromElement - Use [Boolean](Boolean.mdx) with [JSONPathValue](JSONPathValue.mdx).
- LogicalFromElementFromXML - Use [Boolean](Boolean.mdx) with [XPathValue](XPathValue.mdx)
- NextDays\* - Use [BeginningOfDay](BeginningOfDay.mdx).
- NumberFromElement - Use [Number](Number.mdx) with [JSONPathValue](JSONPathValue.mdx).
- NumberFromElementForXML - Use [Number](Number.mdx) with [XPathValue](XPathValue.mdx).
- NumericEmpty\* - Use null.
- RemoveCarriageReturns - Replaced by [ReplaceRegex](ReplaceRegex.mdx)
- RemoveLineBreaks - Replaced by [ReplaceRegex](ReplaceRegex.mdx)
- Right - Replaced by [Substring0](Substring0.mdx).
- SearchHierarchy - Use [FindAncestor](FindAncestor.mdx).
- SearchList
- StartOfNextYear\* - Use [BeginningOfYear](BeginningOfYear.mdx)(1).
- StartOfPreviousMonth\* - Use [BeginningOfMonth](BeginningOfMonth.mdx)(-1).
- StartOfThisMonth\* - Use [BeginningOfMonth](BeginningOfMonth.mdx)(0).
- StartOfThisYear\* - Use [BeginningOfYear](BeginningOfYear.mdx)(0).
- StringFromAttributeForXML - Use [XPathValue](XPathValue.mdx).
- StringFromElement - Use [JSONPathValue](JSONPathValue.mdx).
- StringFromElementForXML - Use [XPathValue](XPathValue.mdx).
- TextAfter - Use [StringAfter](StringAfter.mdx).
- TextBefore - Use [StringBefore](StringBefore.mdx).
- TextToLogical - Replaced by [Boolean](Boolean.mdx) function or logical comparison.
- TextToNumber - Replaced by [Number](Number.mdx).
- ToClientTime\* - Use [ToLocal](ToLocal.mdx).
- Today\* - Use [BeginningOfDay](BeginningOfDay.mdx).
- Tomorrow\* - Use [BeginningOfDay](BeginningOfDay.mdx)(1).
- TrimEnd - Replaced by [ReplaceRegex](ReplaceRegex.mdx).
- TrimStart - Replaced by [ReplaceRegex](ReplaceRegex.mdx).
- UserPresence
- Yesterday\* - Use [BeginningOfDay](BeginningOfDay.mdx)(-1).
