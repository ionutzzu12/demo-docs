Removes carriage returns to make the token fit on a single line.

## Syntax

RemoveCarriageReturn(str)

## Parameters

| Parameter | Description                                    |
| --------- | ---------------------------------------------- |
| str       | Input string to perform Remove Carriage Return |

## Return Value

Returns a single line string.

## Example

$(RemoveCarriageReturns("Now is the winter of our discontent\r,Made glorious summer by this son of York."))

Removes the carriage return and returns this value:

Now is the winter of our discontent,Made glorious summer by this son of York.
