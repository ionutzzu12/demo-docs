Returns a string value from a specified XML element.

## Syntax

StringFromElementForXML(xmlString, elementName, position, subElement)

## Parameters

| Parameter   | Description            |
| ----------- | ---------------------- |
| xmlString   | XML input string       |
| elementName | XML element name       |
| position    | Index of element array |
| subElement  | Sub element name       |

## Return Value

Returns a string value from a specified XML element.

## Examples

$(StringFromElementForXML('\<catalog>\<book>\<in\_stock>True\</in\_stock>\<title>The Rest API\</title>\<price>3.50\</price>\</book>\<book>\<in\_stock>False\</in\_stock>\<title>XML Modifiers and You\</title>\<price>\<priceUS>48.95\</priceUS>\<priceUK>48.96\</priceUK>\</price>\</book>\</catalog>', 'book', 1, 'title'))
