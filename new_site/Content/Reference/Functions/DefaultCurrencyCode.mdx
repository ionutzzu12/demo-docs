Returns the default currency code of the current session. The default currency code is set on the **Application Setup** page in the Configuration Console. See [Entering Account Information](<../../Configure/SetUpWizard/Entering Account Information.mdx>).

## Syntax

DefaultCurrencyCode()

## Parameters

None.

## Return Value

<Paragraph indent="1">
  Text value. Returns a currency code. See [http://www.xe.com/iso4217.php](http://www.xe.com/iso4217.php) for a list of ISO 4217 currency codes.
</Paragraph>

## Example

$(DefaultCurrencyCode())

<Paragraph indent="1">
  If the default currency code for the current session is USD, the above example returns "USD".
</Paragraph>
