Applies cryptographic hashing algorithms to text strings. This enables you to create encrypted strings you can use to create tokens for APIs and to detect data changes. By default, hashed strings are converted to hex format and use UTF-8 encoding.

## Syntax

HashString(textValue, hashingAlgorithm, isBase64Result)

## Return Value

String hash, either in hexidecimal format or Base64 encoded.

## Example

$(HashString("Test value", "MD5", false))

Returns this value:

"63e1ebd352652df46f8d00d8de6d177a"
