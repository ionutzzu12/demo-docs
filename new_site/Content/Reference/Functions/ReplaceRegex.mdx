Replaces all matches of a regular expression in the specified text with a specified pattern.

See [https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Regular\_Expressions](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Guide/Regular_Expressions) for information about regular expressions.

<hint type="info">
  This function can be used to replace the RemoveCarriageReturns, RemoveLineBreaks, Trim, TrimEnd, and TrimStart modifiers as used in \{\{FRS\_VARS.CSM}}.
</hint>

## Syntax

ReplaceRegex(text, exp, pattern)

## Parameters

| Parameter | Description                                                                                       |
| --------- | ------------------------------------------------------------------------------------------------- |
| text      | The text in which to match.                                                                       |
| exp       | The regular expression to look for in the text.                                                   |
| pattern   | The pattern with which to replace the regular expression.  Can have references to capture groups. |

## Return Value

<Paragraph indent="1">
  Text value.
</Paragraph>

## Example

$(ReplaceRegex("The large dog ran after the small cat.", /(The) (\[a-z ]\*) (dog .\*) (\[a-z]+) (cat)/,"$1 $4 $3 $2 $5"))

<Paragraph indent="1">
  This example returns "The small dog ran after the large cat."
</Paragraph>
