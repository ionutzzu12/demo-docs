Returns the currency code of a currency value. A currency value is a combination of the currency amount and the currency code, such as 3 USD.

<hint type="info">
  - Use this function to return the currency code portion of a currency value.
  - Use the [CurrencyValue](CurrencyValue.mdx) function to return the currency amount portion of a currency value.
</hint>

## Syntax

CurrencyCode(currencyValue)

### Parameter

| Parameter     | Description                                                                                                                             |
| ------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| currencyValue | The currency value (a combination of the currency amount and the currency code, such as 3 USD) from which to extract the currency code. |

## Return Value

<Paragraph indent="1">
  Text value. Returns the currency code of the currency value that was input. See [http://www.xe.com/iso4217.php](http://www.xe.com/iso4217.php) for a list of ISO 4217 currency codes.
</Paragraph>

<Paragraph indent="1">
  Returns null if the **currencyValue** parameter is null.
</Paragraph>

## Examples

$(CurrencyCode(44.2HKD)

<Paragraph indent="1">
  The example above returns "HKD", which is the currency code of the input.
</Paragraph>

$(CurrencyCode(Component\_Cost)

<Paragraph indent="1">
  If **Component\_Cost** is the name of a field that contains a currency value, this example returns the currency code of that field. If the **Component\_Cost** field contains "24 USD", this example returns USD.
</Paragraph>
