## Syntax

## Parameter

| Parameter | Description                                                         |
| --------- | ------------------------------------------------------------------- |
| input     | The input to evaluate. Can be text, number, currency, or a Boolean. |

## Return Value

<Paragraph indent="1">
  Boolean value. See :Link[Boolean Embedded Literals]{label="Boolean Embedded Literals" overridedLabel="Boolean Embedded Literals" spaceId docId="rPq_uRqKAng4wQvEUuDtC" version="v2" docAnchorId="#Boolean" loadingMethod="dynamic" newTab="false" githubPath="../Expressions/Literals.htm#Boolean" href="../Expressions/Literals.mdx"}. See also :Link[Conversion to Boolean Values]{label="Conversion to Boolean Values" overridedLabel="Conversion to Boolean Values" spaceId docId="eVZtF84cBSwgHH8QyCA6X" version="v2" docAnchorId="#Boolean_conversion" loadingMethod="dynamic" newTab="false" githubPath="../Expressions/Type Conversions.htm#Boolean_conversion" href="../Expressions/Type Conversions.mdx"}.
</Paragraph>

<Paragraph indent="1">
  If the **input** parameter is text:
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  If it contains "true" (case insensitive), returns **true**.
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  If it contains "false" (case insensitive), returns **false**.
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  If it contains anything else, returns an error.
</Paragraph>

<Paragraph indent="1">
  If the **input** parameter is a number or currency:
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  If it contains zero, returns **false**.
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  If it contains anything else, returns **true**.
</Paragraph>

<Paragraph indent="1">
  If the **input** parameter is a Boolean:
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  Returns the Boolean value.
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  Returns an error if the **input** parameter is another type, such as DateTime.
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  Returns null if the **input** parameter is null.
</Paragraph>

## Examples

$(Boolean("true"))

<Paragraph indent="1">
  In the example above, the application returns true.
</Paragraph>

$(Boolean("false"))

<Paragraph indent="1">
  In the example above, the application returns false.
</Paragraph>

$(Boolean("29 Apr 2014"))

<Paragraph indent="1">
  In the example above, the application returns an error.
</Paragraph>

$(Boolean(27.69))

<Paragraph indent="1">
  In the example above, the application returns true.
</Paragraph>

$(Boolean(0))

<Paragraph indent="1">
  In the example above, the application returns false.
</Paragraph>
