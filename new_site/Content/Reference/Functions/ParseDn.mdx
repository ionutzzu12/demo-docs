Parses a distinguished name. Only use this function in the LDAP expression field mappings.

## Syntax

ParseDn(dn,index)

## Parameters

| Parameter | Description                                                                                                                                                                                                                                                                          |
| --------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| dn        | Distinguished name of the form k1=v1, k2=v2, k3=v3, ..., kn=vn                                                                                                                                                                                                                       |
| index     | (Optional) The zero-based segment of the distinguished name to return.<br />**NOTE**: If you omit this parameter, if the parameter is less than zero, or if the parameter is greater than or equal to the number of segments, the application returns the entire distinguished name. |

## Return Value

<Paragraph indent="1">
  Unicode text value.
</Paragraph>

## Example

$(ParseDN("CN=Paul Stedman,OU=Disabled,OU=Users,OU=California,DC=na,DC=frs", 0))

<Paragraph indent="1">
  returns "Paul Stedman"
</Paragraph>

$(ParseDN(manager, 0))
