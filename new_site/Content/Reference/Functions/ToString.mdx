ToString function converts any type of input value to text form like a string.

## Syntax

ToString(text,precision)

## Parameters

| Parameter | Description                                                                                                                                                                                                                                                                                                                                                                                        |
| --------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| text      | The value to convert to a text value.                                                                                                                                                                                                                                                                                                                                                              |
| precision | (Optional) The number of decimal places returned if the **text** parameter is a number or a currency. The application ignores the **precision** parameter for values that are not a number or currency.<br /><br />The maximum precision for numbers is up to 15 digits in total (including digits before and after the decimal point). Values exceeding this limit result in an incorrect number. |

## Return Value

String - text value.

<hint type="info">
  If the **text** parameter is a text value, returns the text unchanged.
</hint>

## Example

$(ToString(CreatedDateTime))

This returns the text value of the **CreatedDateTime** field.

$(ToString(price, 2))

This returns a currency value to two decimal places. An example is 3.45 USD.
