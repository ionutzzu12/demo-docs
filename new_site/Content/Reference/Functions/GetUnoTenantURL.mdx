Returns the tenant-specific base URL of the Ivanti UNO instance. This function helps generate dynamic links in emails, notes, or system messages where hardcoding the URL would be impractical.

## Syntax

GetUnoTenantURL()

## Parameters

None

## Return Value

String

The base URL of the tenant instance like https\://\[company-name].ivanti.com

## Example

The following example shows how to use this function to build a dynamic link for an email template.

CONCAT($(GetUNOTenantURL()), "/HEAT/BusinessObject/Incident", $(RecID()))

This outputs a URL like:

https\://\[company-name].ivanti.com/HEAT/BusinessObject/Incident/\[incident-id]
