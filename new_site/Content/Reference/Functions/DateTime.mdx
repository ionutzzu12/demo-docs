## Syntax

DateTime(value)

## Parameter

| Parameter | Description                                                     |
| --------- | --------------------------------------------------------------- |
| value     | The input to evaluate. Can be a text value or a DateTime value. |

## Return Value

<Paragraph indent="1">
  DateTime value.  See :Link[DateTime Embedded Literals]{label="DateTime Embedded Literals" overridedLabel="DateTime Embedded Literals" spaceId docId="rPq_uRqKAng4wQvEUuDtC" version="v2" docAnchorId="#Date/Time_Literals" loadingMethod="dynamic" newTab="false" githubPath="../Expressions/Literals.htm#Date/Time_Literals" href="../Expressions/Literals.mdx"} and :Link[Conversion to DateTime Values]{label="Conversion to DateTime Values" overridedLabel="Conversion to DateTime Values" spaceId docId="eVZtF84cBSwgHH8QyCA6X" version="v2" docAnchorId="#Conversion_datetime" loadingMethod="dynamic" newTab="false" githubPath="../Expressions/Type Conversions.htm#Conversion_datetime" href="../Expressions/Type Conversions.mdx"}. Returns the date and time in the date and time format that is based on your culture code. See :Link[Available Languages and Cultures]{label="Available Languages and Cultures" overridedLabel="Available Languages and Cultures" spaceId docId="ZOdB7cSrDXVspZWLePaWn" version="v2" docAnchorId="#culture" loadingMethod="dynamic" newTab="false" githubPath="../../Configure/Localization/Setting Up Languages.htm#culture" href="../../Configure/Localization/Setting Up Languages.mdx"} for a list of the culture codes used by \{\{FRS\_VARS.ProductName}}.
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  Returns a date and time if the **value** parameter is a text value that contains a valid date and time literal.
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  Returns the same **value** parameter if the input is a DateTime value.
</Paragraph>

<Paragraph listStyleType="disc" indent="2">
  Returns null if the **value** parameter is null.
</Paragraph>

### Example

1. For **English (United Kingdom)** locale, accepted date format is *dd/mm/yy*.

<Paragraph indent="1">
  $(DateTime("31/01/24"))
</Paragraph>

<Paragraph indent="1">
  For the example above, returns 31/01/24 as date.
</Paragraph>

2. For **English (United states)** locale, accepted date format is *mm/dd/yy*.

<Paragraph indent="1">
  $(DateTime("01/31/24"))
</Paragraph>

<Paragraph indent="1">
  For the example above, returns 01/31/24 as date.
</Paragraph>

<hint type="info" indent="1">
  The return date value depends on the locale you're using. Check the date format based on your locale.
</hint>
