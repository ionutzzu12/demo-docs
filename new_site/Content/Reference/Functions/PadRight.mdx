Inserts a particular character(s) to the right of a value to make the value a particular length. One example is:

- Original token: ABC
- Modifier: Pad Right (0,7)
- Result: ABC0000

## Syntax

PadRight(textValue, padText, length)

## Parameters

| Parameter | Description                                                          |
| --------- | -------------------------------------------------------------------- |
| textValue | Input string to pad                                                  |
| padText   | Text to add at right hand side                                       |
| length    | Length amount that you want to extend the length of the input string |

## Return Value

Returns a string with the length equal to the given length including padText at the right of the string.

## Example

$(PadRight("1234", "$" , 10))

Returns this value:

1234$$$$$$
