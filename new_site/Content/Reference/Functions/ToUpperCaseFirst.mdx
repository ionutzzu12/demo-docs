Converts only the first character to uppercase. This is often useful when working with names and proper nouns.

## Syntax

ToUpperCaseFirst(inputText)

## Parameters

| Parameter | Description                                       |
| --------- | ------------------------------------------------- |
| inputText | Input string on which to perform ToUpperCaseFirst |

## Return Value

<Paragraph indent="1">
  Returns a string with the first character in uppercase.
</Paragraph>

## Example

<Paragraph indent="1">
  $(ToUpperCaseFirst('george'))
</Paragraph>

<Paragraph indent="1">
  Returns this value: George
</Paragraph>
