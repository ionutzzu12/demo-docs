  ## Incident Notifications

  | Condition for Notification                             | Email Sent To ...                                                            |
  | ------------------------------------------------------ | ---------------------------------------------------------------------------- |
  | Incident changed to active status                      | Incident owner                                                               |
  | New incident is created                                | Requester (customer)  <br />User reporting the incident (reported by)        |
  | Incident status changed to closed                      | Requester (customer)                                                         |
  | Incident created, with a priority of 1                 | Incident owner  <br />Incident owner team manager  <br />Incident owner team |
  | Incident status changed to resolved                    | Requester (customer)                                                         |
  | Incident owner is assigned                             | Incident owner                                                               |
  | Incident team is assigned                              | Incident owner team                                                          |
  | Incident priority has changed                          | Requester (customer)                                                         |
  | Incident response time has been breached               | Incident owner                                                               |
  | Survey response upon incident closure                  | Requester (customer)                                                         |
  | Incident requester updated to VIP requester (customer) | Incident owner  <br />Incident owner team manager  <br />Incident owner team |
  | Incident owner is reassigned                           | Requester (customer)  <br />Incident owner                                   |
</ExpandableHeading>

  ## Change Notifications

  | Condition for Notification   | Email Sent To ...                          |
  | ---------------------------- | ------------------------------------------ |
  | A logged change is requested | Change Manager                             |
  | A logged change is cancelled | Change Requester                           |
  | A requested change is denied | Change Coordinator  <br />Change Manager   |
  | Change is pending approval   | Change Requester  <br />Change Approvers   |
  | Change is scheduled          | Change Requester                           |
  | Change is implemented        | Change Requester                           |
  | Change is deferred           | Change Requester                           |
  | Change is closed             | Change Coordinator  <br />Change Requester |
  | Change tasks completed       | Change Manager  <br />Task owners          |
</ExpandableHeading>
