When you post a query in the interface, Agentic AI assigns it to the relevant agent, analyzes your question, and initiates a conversation by asking follow-up questions based on your query.

For example, consider a typical use case as follows:

1. Open the Agentic AI bot in Microsoft Teams, and the user will receive a welcome message.

<Paragraph indent="1">
  After the user submits a query, the AI bot interprets the intent as an Incident and assigns it to the Incident agent for further investigation.
</Paragraph>

<Paragraph indent="1">
  After analyzing the query, the agent asks follow‑up questions to perform device mining.
</Paragraph>

<Paragraph indent="1">
  The following image shows the device mining interaction.
</Paragraph>

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/lmml9e2BwFinL-VlUoBJ4_agentic-ai-example-1.png" position="flex-start" size="40" indent="1" initialPath="../Resources/Images/Agentic-AI-Images/agentic-ai-example-1.png" githubPath="Content/Resources/Images/Agentic-AI-Images/agentic-ai-example-1.png" showCaption="false"></Image>

2. Select the appropriate device from the given list.

<Paragraph indent="1">
  The agent then asks additional questions related to the issue the user is experiencing with the device.
</Paragraph>

<Paragraph indent="1">
  The following image shows the agent’s interaction with the user to clarify the device issue.
</Paragraph>

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/LwDTU0TUS4-trUCPpkA--_agentic-ai-example-2.png" position="flex-start" size="40" indent="1" initialPath="../Resources/Images/Agentic-AI-Images/agentic-ai-example-2.png" githubPath="Content/Resources/Images/Agentic-AI-Images/agentic-ai-example-2.png" showCaption="false"></Image>

<Paragraph indent="1">
  The agent provides a knowledge article with troubleshooting steps to resolve the issue.
</Paragraph>

<Paragraph indent="1">
  The following image shows the agent sharing a knowledge article with the user to troubleshoot the issue.
</Paragraph>

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/q2EG3fQ75P0Z3E660TbEJ_agentic-ai-example-3.png" position="flex-start" size="40" indent="1" initialPath="../Resources/Images/Agentic-AI-Images/agentic-ai-example-3.png" githubPath="Content/Resources/Images/Agentic-AI-Images/agentic-ai-example-3.png" showCaption="false"></Image>

3. Click **Yes&#x20;**&#x69;f the issue is resolved using the troubleshooting steps or click **No&#x20;**&#x69;f the issue is not resolved.

<Paragraph indent="1">
  If **No&#x20;**&#x69;s selected, the agent asks whether the user would like to create an incident or chat with a Live agent.
</Paragraph>

<Paragraph indent="1">
  The following image shows the agent offering further support to the user because the knowledge article did not resolve the issue.
</Paragraph>

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/RXwwUGnxyW_y2kQtnZKlE_agentic-ai-example-5.png" position="flex-start" size="40" indent="1" initialPath="../Resources/Images/Agentic-AI-Images/agentic-ai-example-5.png" githubPath="Content/Resources/Images/Agentic-AI-Images/agentic-ai-example-5.png" showCaption="false"></Image>

<hint type="info" indent="1">
  Click **Chat with Live Agent** to transfer the interaction to your Service Desk Human Agent in case the Agentic AI bot cannot solve the user's request.
</hint>

4. When the user clicks **Create Incident**, the agent proceeds to create a new incident record and presents the incident draft.

<Paragraph indent="1">
  The following image shows the agent creating an incident record and sharing the draft with the user.
</Paragraph>

<Image src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/7Ijnml8xgpQti5kUZ7nnp_agentic-ai-example-6.png" position="flex-start" size="40" indent="1" initialPath="../Resources/Images/Agentic-AI-Images/agentic-ai-example-6.png" githubPath="Content/Resources/Images/Agentic-AI-Images/agentic-ai-example-6.png" showCaption="false"></Image>

5. Click **Looks good**, and the agent creates an incident and shares a tracking link with the user.

<Paragraph indent="1">
  The following image shows the user accepting the Incident draft and the agent shares the Incident record link.
</Paragraph>
