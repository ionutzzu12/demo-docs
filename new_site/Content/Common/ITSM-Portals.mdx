\{\{Primary.Company}} \{\{FRS\_VARS.ProductName}} has different portals based on the role you are assigned.

## Configuration console

The Configuration console is available to Administrators who are responsible for the setup, assigning roles and permissions, and, other configuration settings of the application

**To access the Configuration console:**

1. Log in to \{\{FRS\_VARS.ProductName}} with the Administrator user role.
2. Click the **Settings** icon :inlineImage[]{src="https://archbee-image-uploads-qa.s3.amazonaws.com/yzXkkCTORWlcwRG_AVHKZ/bbCUwoxEIs0EtRX29NDLv_configureapplicationicon.png" alt caption size="10" width="12" height="12"}. The Configuration console is displayed in a new window.

<Paragraph indent="1">
  Application settings are setup from the Configuration console by users with Administrator role. For more information, refer to [Configuring Ivanti Neurons for ITSM](../Configure/CONFIGURE.mdx).
</Paragraph>

## Service Desk console

The Service Desk console displays all workspaces to user roles other than \{\{FRS\_VARS.Self\_Service\_Portal-Self-Service}} and \{\{FRS\_VARS.Self\_Service\_Mobile\_portal}} users. The workspaces that are available to you depend on the role and access your \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}} has provided to you.

For example, an Incident Manager may not have access to the Project Portfolio Management workspace.

**To access the Service Desk console:**

- Log in to \{\{FRS\_VARS.ProductName}} with user role other than Self Service, Self Service Mobile, and Mobile Analyst.

<Paragraph indent="1">
  Service Desk console is accessed by various user roles to perform different tasks. For more information, refer to [Using the Service Desk console](<../ServiceDesk/Using_Service Desk Console.mdx>).
</Paragraph>

## Self Service Portal

The Self Service portal is available to all \{\{FRS\_VARS.Self\_Service\_Portal-Self-Service}} users who can raise service tickets and view them. This is a classic view and not the modern view.

**To access the Self Service Portal:**

- Log in to \{\{FRS\_VARS.ProductName}} with the Self Service user role.

<Paragraph indent="1">
  Self Service users access the Self Service portal to report and track Incidents, submit Service Requests, and so on.
</Paragraph>

## Self Service Mobile Portal

The Self Service Mobile portal is available to all \{\{FRS\_VARS.Self\_Service\_Mobile\_portal}} users who can raise service tickets and view them. This is the modern view that is compatible with mobile devices.

- Log in to \{\{FRS\_VARS.ProductName}} with the Self Service Mobile user role.

<Paragraph indent="1">
  The Self Service Mobile portal is displayed in responsive user interface suitable for accessing \{\{FRS\_VARS.ProductName}} on mobile devices.
</Paragraph>
