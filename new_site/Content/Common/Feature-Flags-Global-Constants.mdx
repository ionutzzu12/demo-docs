**Role**: \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}}.

**Minimum Version**: \{\{FRS\_VARS.ProductName}} 2024.4.

The following feature flags and global constants that are used to control the availability of a certain \{\{FRS\_VARS.unified\_product\_experience}} features and capabilities to different types of users.

- Feature flags are set up in ConfigDB tenant > **Feature Management**.

<hint type="info" indent="1">
  Only on-premises users have access to ConfigDB tenant. Cloud users can raise a service ticket to Ivanti Operations team for support.
</hint>

- Global Constants are set up in the Configuration console > **Build** > **Global Constants**.

| Name                           | Type         | Default Value      | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| ------------------------------ | ------------ | ------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| EnableOneNavUI                 | Feature Flag | Disable            | When this option is set to **Unconditionally ON**, existing cloud and on-premises users can avail the new \{\{FRS\_VARS.unified\_product\_experience}} navigation.<br />This is not applicable to FedRamp.                                                                                                                                                                                                                                                                                                                                 |
| ForceOneNav                    | Feature Flag | Disable            | When this is set to **Unconditionally ON**, the new \{\{FRS\_VARS.unified\_product\_experience}} navigation is the default view for all existing cloud and on-premises users. This overrides the **EnableOneNavU**I settings. This forces the users to use the new navigation and the option to switch between classic and new navigation is prohibited.<br />This is not applicable to FedRamp.                                                                                                                                           |
| EnableGSAuthentication         | Feature Flag | Disable            | When this is set to **Unconditionally ON**, the **Authentication** feature under **Global Settings** in the new \{\{FRS\_VARS.unified\_product\_experience}} navigation is available to cloud customers.<br />This allows customers to setup common authentication for all the applications. For more information, refer to [Authentication](https://help.ivanti.com/ht/help/en_US/CLOUD/vNow/authentication.htm).<br />This is not applicable to on-premises and FedRamp.                                                                 |
| EnableGSAccessControl          | Feature Flag | Disable            | When this is set to **Unconditionally ON**, the **Access Control** feature under **Global Settings** in the new \{\{FRS\_VARS.unified\_product\_experience}} navigation is available to cloud customers.<br />This allows the \{\{FRS\_VARS.Neurons\_ITSM\_-\_Administrator-Administrator.}} to setup access control to their users. For more information, refer to [Authentication](https://help.ivanti.com/ht/help/en_US/CLOUD/vNow/authentication.htm).<br />This is applicable only to cloud users and not to on-premises and FedRamp. |
| EnableWhatsNew                 | Feature Flag | Disable            | When this is set to **Unconditionally ON**, the What's New icon on the header bar in the new \{\{FRS\_VARS.unified\_product\_experience}} navigation is available which displays the product's new features and announcements.<br />This is applicable only to cloud users and not to on-premises.<br />                                                                                                                                                                                                                                   |
| EnableGlobalSettings           | Feature Flag | Disable            | When this is set to **Unconditionally ON**, the Global Settings option is available to users. This is used in setting up access control, branding, and authentication settings.                                                                                                                                                                                                                                                                                                                                                            |
| EnableGSBranding               | Feature Flag | Disable            | When this is set to **Unconditionally ON**, the **Branding** option under Global Settings is available to users. This is used to configure and customize page settings.                                                                                                                                                                                                                                                                                                                                                                    |
| ContainInfiniteLoopForWorkflow | Feature Flag | Unconditionally ON | When this is set to **Unconditionally ON**, it enables a feature-level guardrail in the tenant's configuration database to prevent infinite loops in workflows.                                                                                                                                                                                                                                                                                                                                                                            |
| EnableOIL                      | Feature Flag | Disable            | When this is set to **Unconditionally ON**, it enables the Unified Ivanti Login feature to users.                                                                                                                                                                                                                                                                                                                                                                                                                                          |
| EnableOIDCClientSecretEncode   | Feature Flag | Disable            | This feature flag is related to OpenID Connect (OIDC) authentication. When this is set to **Unconditionally ON**, it enables the Unified Ivanti Login feature to users.                                                                                                                                                                                                                                                                                                                                                                    |
| EnableNeuronsPLP               | Feature Flag | Disable            | When this is set to **Unconditionally ON**, it enables the \{\{FRS\_VARS.Portfolio\_Landing\_Page}} feature to users.                                                                                                                                                                                                                                                                                                                                                                                                                      |
