For Administrators or user roles with access to the Configuration console, find information about setting up and configuring the application, and managing how other users access and use the application in [Configure and Administer](../Configure/CONFIGURE.mdx).

For IT Staff roles that manage Service Requests, Incidents, Problems, Changes, etc., find information about using the different workspaces in Ivanti Neurons for ITSM in [Using the Service Desk Console.](<../ServiceDesk/Using_Service Desk Console.mdx>)

For employees or external customers looking for information on how to use the Self Service portal, please see [Getting Started with Self Service portal](../../../self/Content/Self-Service/LandingPage-Self.htm).
